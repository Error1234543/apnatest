Pro Notes

ICBN

📖 ICBN (ઇન્ટરનેશનલ કોડ ઓફ બોટેનિકલ નોમેનક્લેચર)

ICBN ની પરિભાષા અને મહત્વ

ICBN એ વનસ્પતિ શાસ્ત્રીય નામકરણ માટેની આંતરરાષ્ટ્રીય સંહિતા છે, જે વનસ્પતિઓના વૈજ્ઞાનિક નામોને એકરૂપ અને સુસંગત બનાવે છે. તે વનસ્પતિશાસ્ત્રીઓ દ્વારા વપરાતા નિયમો અને સૂચનો પ્રદાન કરે છે. NEET મા આનો ઉપયોગ વનસ્પતિના સંપૂર્ણ વૈજ્ઞાનિક નામ લખવા અને ઓળખવા માટે થાય છે.

  • ICBN નું પૂરું નામ: International Code of Botanical Nomenclature
  • વનસ્પતિઓના વૈજ્ઞાનિક નામો માટેની માન્ય સંહિતા
  • પ્રથમ વખત ૧૯૦૫ મા પ્રકાશિત થઈ
  • હાલમાં ICN (International Code of Nomenclature for algae, fungi, and plants) તરીકે ઓળખાય છે
🧠 Trick: ICBN = International Code for Botanical Name (વનસ્પતિ નામકરણની આંતરરાષ્ટ્રીય કોડ)

વૈજ્ઞાનિક નામકરણના મૂળભૂત નિયમો

વૈજ્ઞાનિક નામ બે શબ્દોનું બનેલું હોય છે: જાતિ (Genus) અને પ્રજાતિ (Species). જાતિનું નામ મોટા અક્ષરે અને પ્રજાતિનું નામ નાના અક્ષરે લખાય છે. બંને શબ્દો ઇટાલિક અથવા અન્ડરલાઇનમાં લખાય છે. ઉદાહરણ: *Mangifera indica* (આંબો).

  • વૈજ્ઞાનિક નામ = જાતિ (Genus) + પ્રજાતિ (Species)
  • જાતિનું નામ મોટા અક્ષરે (ઉદા. *Mangifera*)
  • પ્રજાતિનું નામ નાના અક્ષરે (ઉદા. *indica*)
  • બંને શબ્દો ઇટાલિક અથવા અન્ડરલાઇનમાં લખાય
  • લેટિન ભાષા અથવા લેટિનીકૃત શબ્દો વાપરવા
🧠 Trick: મોટું જાતિ, નાનું પ્રજાતિ (Genus = મોટા, Species = નાના)

નામકરણમાં વપરાતા પૂરક શબ્દો

વૈજ્ઞાનિક નામ પછી, કેટલીકવાર ઉપજાતિ (Subspecies), વર્ગ (Variety), અથવા રૂપ (Form) જેવા પૂરક શબ્દો ઉમેરાય છે. ઉદાહરણ: *Homo sapiens sapiens* (માનવની ઉપજાતિ). આ શબ્દો પણ ઇટાલિકમાં લખાય છે અને પ્રજાતિના નામ પછી લખાય છે.

  • ઉપજાતિ (Subspecies) - ssp. અથવા subsp.
  • વર્ગ (Variety) - var.
  • રૂપ (Form) - f.
  • બધા પૂરક શબ્દો ઇટાલિકમાં લખાય
  • પ્રજાતિના નામ પછી લખાય છે

નામકરણમાં લેખકનું નામ

વૈજ્ઞાનિક નામ પછી, વનસ્પતિની શોધ કરનાર વૈજ્ઞાનિકનું નામ લખી શકાય છે, પરંતુ તે ઇટાલિકમાં નહીં લખાય. ઉદાહરણ: *Mangifera indica* Linn. (Linn. = Linnaeus). આ નામને ઓથર સિટેશન (Author Citation) કહેવાય છે.

  • લેખકનું નામ વૈજ્ઞાનિક નામ પછી લખાય
  • લેખકનું નામ ઇટાલિકમાં નહીં લખાય
  • સંક્ષિપ્ત રૂપમાં લખાય (ઉદા. Linn., Hook.)
  • NEET મા આ નિયમ ઓળખવો જરૂરી છે

નામકરણમાં પ્રાથમિકતા અને વરિષ્ઠતા

ICBN મુજબ, વનસ્પતિનું પ્રથમ પ્રકાશિત થયેલું વૈજ્ઞાનિક નામ જ માન્ય ગણાય છે. જો એક જ વનસ્પતિ માટે બે અલગ નામો પ્રકાશિત થયા હોય, તો પ્રથમ પ્રકાશિત થયેલું નામ વાપરવામાં આવે છે. આને નામકરણની વરિષ્ઠતા (Priority) કહેવાય છે.

  • પ્રથમ પ્રકાશિત થયેલું નામ જ માન્ય
  • વરિષ્ઠતા (Priority) ના નિયમ મુજબ
  • જો બે નામો હોય તો જૂનું નામ વાપરવું
  • NEET મા આ નિયમ પર પ્રશ્નો પૂછાય છે
🧠 Trick: પહેલું નામ = માન્ય નામ (First = Valid)

નામકરણમાં ભાષા અને ઉચ્ચાર

વૈજ્ઞાનિક નામો લેટિન ભાષામાં અથવા લેટિનીકૃત શબ્દોમાં લખાય છે. આ નામોનો ઉચ્ચાર સ્થાનિક ભાષા મુજબ થઈ શકે છે, પરંતુ લિખિત રૂપમાં તે લેટિનમાં જ રહે છે. ઉદાહરણ: *Solanum tuberosum* (બટાટા).

  • નામો લેટિન ભાષામાં લખાય
  • લેટિનીકૃત શબ્દો પણ વાપરી શકાય
  • ઉચ્ચાર સ્થાનિક ભાષા મુજબ થઈ શકે
  • લિખિત રૂપ લેટિનમાં જ રહે

ICBN ના અપવાદો અને ખાસ કેસ

કેટલીક વનસ્પતિઓ માટે, જૂના નામો અથવા સ્થાનિક નામો પણ વાપરવામાં આવે છે, પરંતુ ICBN મુજબ, વૈજ્ઞાનિક નામ જ માન્ય ગણાય છે. ઉદાહરણ: *Azadirachta indica* (લીમડો) ને કેટલીકવાર *Melia azadirachta* તરીકે પણ ઓળખવામાં આવે છે, પરંતુ ICBN મુજબ *Azadirachta indica* જ માન્ય છે.

  • વૈજ્ઞાનિક નામ જ માન્ય ગણાય
  • સ્થાનિક નામો વાપરી શકાય પરંતુ માન્ય નહીં
  • જૂના નામો અપવાદરૂપે વાપરાય
  • NEET મા માન્ય નામ ઓળખવું જરૂરી છે

⚡ Quick Revision

  • ICBN એ વનસ્પતિઓના વૈજ્ઞાનિક નામકરણ માટેની આંતરરાષ્ટ્રીય સંહિતા છે.
  • વૈજ્ઞાનિક નામ = જાતિ (Genus) + પ્રજાતિ (Species).
  • જાતિનું નામ મોટા અક્ષરે, પ્રજાતિનું નામ નાના અક્ષરે લખાય છે.
  • બંને શબ્દો ઇટાલિક અથવા અન્ડરલાઇનમાં લખાય છે.
  • લેખકનું નામ વૈજ્ઞાનિક નામ પછી લખાય પરંતુ ઇટાલિકમાં નહીં.
  • પ્રથમ પ્રકાશિત થયેલું નામ જ માન્ય ગણાય છે (વરિષ્ઠતા નિયમ).
  • નામો લેટિન ભાષામાં અથવા લેટિનીકૃત શબ્દોમાં લખાય છે.
  • ઉપજાતિ, વર્ગ, રૂપ જેવા પૂરક શબ્દો ઇટાલિકમાં લખાય છે.
  • NEET મા વૈજ્ઞાનિક નામ અને લેખકનું નામ ઓળખવા પર પ્રશ્નો પૂછાય છે.
🎯 Exam Tip: NEET મા ICBN પરના પ્રશ્નો માટે, વૈજ્ઞાનિક નામનું ફોર્મેટ (Genus + Species) અને લેખકનું નામ યાદ રાખો. પ્રથમ પ્રકાશિત થયેલું નામ જ માન્ય છે એ નિયમ પર ધ્યાન આપો.
