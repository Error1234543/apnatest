ICZN

📖 ICZN (ઇન્ટરનેશનલ કોડ ઓફ ઝૂલોજિકલ નોમેનક્લેચર)

ICZN ની પરિભાષા અને ઉદ્દેશ

ICZN એ પ્રાણીશાસ્ત્રમાં પ્રાણીઓના વૈજ્ઞાનિક નામોને નિયંત્રિત કરવા માટેની આંતરરાષ્ટ્રીય સંસ્થા છે. તેનો મુખ્ય ઉદ્દેશ્ય પ્રાણીઓના નામકરણમાં એકસમાનતા, સ્પષ્ટતા અને સ્થિરતા લાવવી છે. NEETમાં પ્રાણીઓના વૈજ્ઞાનિક નામો અને તેના નિયમો પર પ્રશ્નો પૂછાય છે.

  • ICZN = International Code of Zoological Nomenclature
  • પ્રાણીઓના વૈજ્ઞાનિક નામો માટેના નિયમો નક્કી કરે છે
  • લેટિન ભાષામાં નામકરણ થાય છે
  • દ્વિનામ પદ્ધતિ (Binomial Nomenclature)નો ઉપયોગ થાય છે

દ્વિનામ પદ્ધતિ (Binomial Nomenclature)

કેરોલસ લિનિયસ દ્વારા શોધાયેલી આ પદ્ધતિમાં પ્રાણીના નામના બે ભાગ હોય છે: જાતિ (Genus) અને પ્રજાતિ (Species). ઉદા. Homo sapiens. જાતિનું નામ મોટા અક્ષરે અને પ્રજાતિનું નામ નાના અક્ષરે લખાય છે.

  • જાતિ (Genus) + પ્રજાતિ (Species) = વૈજ્ઞાનિક નામ
  • જાતિનું નામ મોટા અક્ષરે (ઉદા. Panthera)
  • પ્રજાતિનું નામ નાના અક્ષરે (ઉદા. leo)
  • ઇટાલિક અથવા અંડરલાઇનમાં લખાય છે
🧠 Trick: મોટું જાતિ, નાનું પ્રજાતિ

ICZN ના મુખ્ય નિયમો

ICZN અનુસાર, પ્રથમ વખત પ્રકાશિત થયેલ નામને વરિષ્ઠતા (Priority) મળે છે. નામ સ્પષ્ટ, સંક્ષિપ્ત અને લેટિન અથવા લેટિનીકરણ કરેલ હોવું જોઈએ. નામમાં રાજકીય અથવા વ્યક્તિગત ભાવનાઓ ન હોવી જોઈએ.

  • પ્રથમ પ્રકાશિત નામને વરિષ્ઠતા (Priority Rule)
  • નામ લેટિન અથવા લેટિનીકરણ કરેલ હોવું જોઈએ
  • નામ સ્પષ્ટ અને સંક્ષિપ્ત હોવું જોઈએ
  • રાજકીય અથવા વ્યક્તિગત ભાવનાઓ વર્જિત

નામકરણમાં સુધારાઓ અને સંકેટો

જો કોઈ નામ ICZN ના નિયમોનો ભંગ કરતું હોય તો તેને બદલી શકાય છે. નામમાં સુધારો કરવા માટે ICZN ની મંજૂરી જરૂરી છે. નામકરણમાં વપરાતા સંકેટો જેવા કે sp. (Species), spp. (Multiple Species), subsp. (Subspecies)નો ઉપયોગ થાય છે.

  • sp. = અજ્ઞાત પ્રજાતિ (Unknown Species)
  • spp. = બહુવિધ પ્રજાતિઓ (Multiple Species)
  • subsp. = ઉપપ્રજાતિ (Subspecies)
  • ICZN ની મંજૂરી વિના નામ બદલી શકાતું નથી

NEET માટે મહત્વપૂર્ણ ઉદાહરણો

NEETમાં પ્રાણીઓના વૈજ્ઞાનિક નામો પર પ્રશ્નો પૂછાય છે. ઉદા. મનુષ્ય: Homo sapiens, વાઘ: Panthera tigris, શેર: Panthera leo. આ નામો યાદ રાખવા અને તેના નિયમો સમજવા મહત્વપૂર્ણ છે.

  • Homo sapiens - મનુષ્ય
  • Panthera tigris - વાઘ
  • Panthera leo - શેર
  • Rattus rattus - કાળો ઉંદર
🧠 Trick: હોમો સેપિયન્સ = હું સૌથી હોશિયાર

ICZN અને આંતરરાષ્ટ્રીય સ્વીકૃતિ

ICZN ના નિયમો આંતરરાષ્ટ્રીય સ્તરે સ્વીકૃત છે અને તમામ પ્રાણીશાસ્ત્રીઓએ તેનું પાલન કરવું જરૂરી છે. આ નિયમો પ્રાણી વિજ્ઞાનમાં સંશોધન અને અભ્યાસ માટે મહત્વપૂર્ણ છે.

  • ICZN નિયમો આંતરરાષ્ટ્રીય સ્તરે લાગુ પડે છે
  • તમામ પ્રાણીશાસ્ત્રીઓએ પાલન કરવું જરૂરી છે
  • સંશોધન અને અભ્યાસ માટે મહત્વપૂર્ણ
  • NEETમાં પ્રશ્નો પૂછાય છે

  • sp. = અજ્ઞાત પ્રજાતિ, spp. = બહુવિધ પ્રજાતિઓ
