Key

📖 કી (Key) - જૈવિક, ભૌતિક અને રસાયણિક પરિપ્રેક્ષ્યમાં

જીવવિજ્ઞાન: જનીન કી (Genetic Key)

જનીન કી એ DNA અથવા RNA ની ક્રમિક રચના છે જે પ્રોટીન સંશ્લેષણ માટે આનુવંશિક માહિતી એન્કોડ કરે છે. NCERT મુજબ, જનીન કી એ કોડોનના રૂપમાં હોય છે જે mRNA પર અમીનો એસિડની ઓળખ કરે છે. પ્રોટીન સંશ્લેષણમાં tRNA તેની એન્ટીકોડોન દ્વારા mRNA ના કોડોન સાથે જોડાય છે.

  • જનીન કી = કોડોન (mRNA પર 3 ન્યુક્લિયોટાઇડનો સમૂહ)
  • AUG એ પ્રારંભ કોડોન (મિથાયોનિન માટે)
  • UAA, UAG, UGA એ અંત કોડોન (ટર્મિનેશન)
  • tRNA એ એન્ટીકોડોન દ્વારા કોડોનને ઓળખે છે
  • 64 કોડોનમાંથી 61 અમીનો એસિડ માટે કોડ કરે છે
🧠 Trick: AUG કહે 'GO', UAA/UAG/UGA કહે 'STOP'

ભૌતિકવિજ્ઞાન: કી એનર્જી અને ક્વોન્ટમ મેકેનિક્સ

ભૌતિકવિજ્ઞાનમાં, કી એનર્જી એ કણની ગતિ એનર્જીનો એક ભાગ છે જે તાપમાન સાથે સંકળાયેલ છે. ક્વોન્ટમ મેકેનિક્સમાં, કી એનર્જી એ કણની સરેરાશ ગતિ એનર્જી છે જે તાપમાન પર આધારિત છે (kT, જ્યાં k બોલ્ટઝમેનનો અચળાંક છે).

  • કી એનર્જી = (3/2)kT (એક-પરમાણ્વિક ગેસ માટે)
  • k = બોલ્ટઝમેન અચળાંક (1.38 × 10^-23 J/K)
  • તાપમાન વધે ત્યારે કી એનર્જી વધે
  • ક્વોન્ટમ મેકેનિક્સમાં, kT એ ઉર્જાના ક્વોન્ટા સાથે સંકળાયેલ છે
  • માઈક્રોસ્કોપિક કણો માટે કી એનર્જી મહત્વની છે

રસાયણવિજ્ઞાન: કી રાસાયણિક પ્રક્રિયાઓ અને ઉત્પ્રેરકો

રસાયણવિજ્ઞાનમાં, કી એ ઉત્પ્રેરક (કેટાલિસ્ટ) દ્વારા પ્રક્રિયાની ગતિ નક્કી કરતો પરિબળ છે. ઉત્સેચક (એન્ઝાઇમ) એ જૈવિક ઉત્પ્રેરક છે જે પ્રક્રિયાની સક્રિયણ ઉર્જા ઘટાડે છે. NCERT મુજબ, ઉત્સેચક કી એ સબસ્ટ્રેટ સાથે જોડાય છે અને પ્રક્રિયાને વેગ આપે છે.

  • ઉત્સેચક = જૈવિક ઉત્પ્રેરક (પ્રોટીન અથવા RNA)
  • કી એ સબસ્ટ્રેટ સાથે જોડાય છે (લોક-એન્ડ-કી મોડેલ)
  • પ્રક્રિયાની ગતિ = k[સબસ્ટ્રેટ]^n (k = ગતિ અચળાંક)
  • તાપમાન અને pH ઉત્સેચકની ક્રિયાશીલતાને અસર કરે છે
  • ઉત્સેચક કી એ પ્રક્રિયાની વિશિષ્ટતા નક્કી કરે છે
🧠 Trick: લોક-એન્ડ-કી: ઉત્સેચક = લોક, સબસ્ટ્રેટ = કી

જીવવિજ્ઞાન: કી પેશીઓ અને અંગો

કી પેશીઓ એવી પેશીઓ છે જે અંગોની રચના અને કાર્યમાં મહત્વની ભૂમિકા ભજવે છે. ઉદાહરણ તરીકે, કોષકેન્દ્ર (ન્યુક્લિયસ) એ કોષની કી પેશી છે જે અનુવંશિક માહિતીનું નિયંત્રણ કરે છે. NCERT મુજબ, પેશી સ્તરે, કી પેશીઓ એ અંગોની કાર્યક્ષમતાને નક્કી કરે છે.

  • કોષકેન્દ્ર = કોષની કી પેશી (DNA ધરાવે છે)
  • પેશી સ્તરે કી પેશીઓ એ અંગોની રચના કરે છે
  • હૃદયની પેશી = હૃદયની કી પેશી (રુધિર પરિવહન)
  • મજ્જા પેશી = હાડકાંની કી પેશી (રક્તકણો ઉત્પન્ન કરે છે)
  • તંત્રિકા પેશી = ચેતાતંત્રની કી પેશી (સંકેત પરિવહન)

રસાયણવિજ્ઞાન: કી કાર્બોદિત અને પ્રોટીન

કાર્બોદિત અને પ્રોટીન એ જૈવિક અણુઓ છે જે કી ભૂમિકા ભજવે છે. કાર્બોદિત એ ઉર્જાનો મુખ્ય સ્રોત છે, જ્યારે પ્રોટીન એ કોષની રચના અને કાર્ય માટે આવશ્યક છે. NCERT મુજબ, પ્રોટીન એ અમીનો એસિડના પોલિમર છે જે પેપ્ટાઇડ બંધ દ્વારા જોડાયેલા હોય છે.

  • કાર્બોદિત = ઉર્જાનો મુખ્ય સ્રોત (ગ્લુકોઝ, સ્ટાર્ચ)
  • પ્રોટીન = અમીનો એસિડના પોલિમર (પેપ્ટાઇડ બંધ)
  • કી પ્રોટીન: ઉત્સેચક, હોર્મોન, એન્ટિબોડી
  • કાર્બોદિતનું શ્વસન = ATP ઉત્પન્ન કરે છે
  • પ્રોટીનનું પાચન = અમીનો એસિડ મુક્ત કરે છે
🧠 Trick: CARBS = Quick Energy, PROTEIN = Building Blocks

જીવવિજ્ઞાન: કી પ્રક્રિયાઓ - પ્રકાશસંશ્લેષણ અને શ્વસન

પ્રકાશસંશ્લેષણ અને શ્વસન એ કી જૈવિક પ્રક્રિયાઓ છે જે ઉર્જાનું પરિવહન કરે છે. પ્રકાશસંશ્લેષણમાં, સૂર્યપ્રકાશની ઉર્જાનો ઉપયોગ કરીને કાર્બોદિત બનાવવામાં આવે છે. શ્વસનમાં, કાર્બોદિતનું ઓક્સિડેશન થઈને ATP ઉત્પન્ન થાય છે.

  • પ્રકાશસંશ્લેષણ = 6CO2 + 6H2O → C6H12O6 + 6O2 (ક્લોરોફિલની હાજરીમાં)
  • શ્વસન = C6H12O6 + 6O2 → 6CO2 + 6H2O + ATP
  • પ્રકાશસંશ્લેષણમાં ક્લોરોપ્લાસ્ટ કી ભૂમિકા ભજવે છે
  • શ્વસનમાં માઇટોકોન્ડ્રિયા કી ભૂમિકા ભજવે છે
  • ATP = કોષની ઉર્જા ચલણ
🧠 Trick: પ્રકાશસંશ્લેષણ = સૂરજ + પાણી + CO2 = ગ્લુકોઝ, શ્વસન = ગ્લુકોઝ + O2 = ATP

⚡ Quick Revision

  • જનીન કી = કોડોન (mRNA પર 3 ન્યુક્લિયોટાઇડ)
  • કી એનર્જી = (3/2)kT (ભૌતિકવિજ્ઞાન)
  • ઉત્સેચક કી = લોક-એન્ડ-કી મોડેલ (રસાયણવિજ્ઞાન)
  • કોષકેન્દ્ર = કોષની કી પેશી (જીવવિજ્ઞાન)
  • કાર્બોદિત = ઉર્જાનો મુખ્ય સ્રોત
  • પ્રકાશસંશ્લેષણ = ક્લોરોપ્લાસ્ટમાં થાય છે
  • શ્વસન = માઇટોકોન્ડ્રિયામાં થાય છે
  • પ્રોટીન = અમીનો એસિડના પોલિમર
  • કી પેશી = અંગોની રચના અને કાર્ય
  • UAA/UAG/UGA = અંત કોડોન (ટર્મિનેશન)
🎯 Exam Tip: NEETમાં 'કી' શબ્દ સાથે સંકળાયેલા પ્રશ્નો માટે જનીન કી (કોડોન), કી એનર્જી (kT), અને ઉત્સેચક કી (લોક-એન્ડ-કી) પર ધ્યાન કેન્દ્રિત કરો.
