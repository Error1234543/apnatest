Prions

📖 પ્રિયોન્સ (Prions)

પરિભાષા અને સ્વરૂપ

પ્રિયોન્સ એ અતિ સૂક્ષ્મ, સજીવ નહીં હોય તેવા રોગકારક કણો છે જે માત્ર પ્રોટીનથી બનેલા હોય છે. તેઓમાં DNA અથવા RNA નથી હોતું. તેઓ મગજની પેશીઓમાં અસાધારણ રીતે ગુંચવાયેલા પ્રોટીનના કણો તરીકે જોવા મળે છે.

  • પ્રિયોન્સ = પ્રોટીન-ઇન્ફેક્શિયસ પાર્ટિકલ્સ (Proteinaceous Infectious Particles)
  • કોઈ જનીન સામગ્રી (DNA/RNA) ધરાવતા નથી
  • માત્ર અસાધારણ રચનાવાળા પ્રોટીન (PrP^Sc) ધરાવે છે
  • મગજની પેશીઓને અસર કરે છે
🧠 Trick: પ્રિયોન્સ = પી (પ્રોટીન) + રિયો (રોગ) + ન્સ (નવો)

રચના અને ક્રિયાવિધિ

પ્રિયોન્સમાં પ્રાયોન પ્રોટીન (PrP) નું અસાધારણ રૂપ (PrP^Sc) હોય છે, જે સામાન્ય પ્રોટીન (PrP^C) ની રચનાને બદલીને તેને રોગકારક બનાવે છે. PrP^Sc બીટા-શીટ સ્ટ્રક્ચર ધરાવે છે, જે કોષના પાચનતંત્ર દ્વારા નાશ પામતું નથી.

  • PrP^C = સામાન્ય પ્રોટીન (અલ્ફા-હેલિક્સ સ્ટ્રક્ચર)
  • PrP^Sc = રોગકારક પ્રોટીન (બીટા-શીટ સ્ટ્રક્ચર)
  • PrP^Sc, PrP^C ને PrP^Sc માં રૂપાંતરિત કરે છે
  • કોષના ઉત્સેચકો દ્વારા નાશ પામતું નથી
🧠 Trick: PrP^C → PrP^Sc: C = સામાન્ય, Sc = સ્ક્રેપી (રોગ)

રોગો અને લક્ષણો

પ્રિયોન્સ દ્વારા થતા રોગોને ટ્રાન્સમિસિબલ સ્પોંગિફોર્મ એન્સેફેલોપેથી (TSE) કહેવાય છે. આ રોગોમાં મગજની પેશીઓમાં છિદ્રો પડી જાય છે, જેના કારણે સ્નાયુઓનો નિયંત્રણ ગુમાવી, મૃત્યુ થાય છે. ઉદા. મડ ડિઝીઝ (કુરુ), ક્રેજી કાઉ ડિઝીઝ (BSE), સ્ક્રેપી.

  • TSE = ટ્રાન્સમિસિબલ સ્પોંગિફોર્મ એન્સેફેલોપેથી
  • મગજમાં સ્પોંગી રચના (છિદ્રો) થાય છે
  • લક્ષણો: સ્નાયુઓનો કંટ્રોલ ગુમાવવો, મેમરી લોસ, મૃત્યુ
  • ઉદા. કુરુ, BSE (ગાયનો રોગ), સ્ક્રેપી (ભેંસનો રોગ)
🧠 Trick: TSE = ટ્રાન્સમિસિબલ + સ્પોંગી + એન્સેફેલોપેથી

સંક્રમણ અને ફેલાવો

પ્રિયોન્સનો ફેલાવો રોગી પ્રાણીના મગજ અથવા અંગોના સેવન દ્વારા, રક્તાધાન દ્વારા અથવા વારસાગત રીતે થઈ શકે છે. તેઓ ઉચ્ચ તાપમાન, રસાયણો અને કિરણોત્સર્ગ પ્રત્યે પ્રતિરોધક હોય છે.

  • સંક્રમણના માધ્યમ: રોગી મગજ/અંગોનું સેવન, રક્તાધાન, વારસાગત
  • ઉચ્ચ તાપમાન (100°C) અને રસાયણો પ્રત્યે પ્રતિરોધક
  • કિરણોત્સર્ગ (UV, X-ray) પ્રત્યે પણ પ્રતિરોધક
  • મનુષ્યમાં કુરુ રોગ વારસાગત અથવા રસ્મી રીતે ફેલાય છે
🧠 Trick: પ્રિયોન્સ = પકાવવા, રસાયણો, કિરણોત્સર્ગ પ્રત્યે પ્રતિરોધક

નિદાન અને બચાવ

પ્રિયોન્સ દ્વારા થતા રોગોનું નિદાન મગજની બાયોપ્સી અથવા MRI દ્વારા થાય છે. હાલમાં આ રોગો માટે કોઈ સારવાર નથી, પરંતુ સંક્રમણ ટાળવા માટે સાવચેતી રાખવી જરૂરી છે.

  • નિદાન: મગજની બાયોપ્સી, MRI, ઇલેક્ટ્રોન માઇક્રોસ્કોપી
  • સારવાર: કોઈ સારવાર નથી (ફક્ત લક્ષણોનું ઉપચાર)
  • બચાવ: રોગી પ્રાણીના મગજ/અંગોનું સેવન ટાળવું
  • રક્તદાનમાં સાવચેતી રાખવી

NEET માટે મહત્વની બાબતો

પ્રિયોન્સ એ અતિ સૂક્ષ્મ રોગકારક કણો છે જે માત્ર પ્રોટીનથી બનેલા હોય છે. તેઓમાં જનીન સામગ્રી હોતી નથી અને મગજની પેશીઓને અસર કરે છે. NEETમાં પ્રિયોન્સની રચના, ક્રિયાવિધિ, રોગો અને સંક્રમણ પધ્ધતિ પર પ્રશ્નો પૂછાઈ શકે છે.

  • પ્રિયોન્સ = માત્ર પ્રોટીનથી બનેલા રોગકારક કણો
  • કોઈ DNA/RNA નથી
  • TSE રોગો (કુરુ, BSE, સ્ક્રેપી) સાથે સંકળાયેલા
  • સંક્રમણ: રોગી અંગોનું સેવન, રક્તાધાન, વારસાગત

⚡ Quick Revision

  • પ્રિયોન્સ = પ્રોટીન-ઇન્ફેક્શિયસ પાર્ટિકલ્સ (કોઈ જનીન સામગ્રી નથી)
  • PrP^C (સામાન્ય) → PrP^Sc (રોગકારક) રૂપાંતરણ
  • TSE રોગો: કુરુ, BSE, સ્ક્રેપી (મગજમાં સ્પોંગી રચના)
  • સંક્રમણ: રોગી અંગોનું સેવન, રક્તાધાન, વારસાગત
  • ઉચ્ચ તાપમાન, રસાયણો, કિરણોત્સર્ગ પ્રત્યે પ્રતિરોધક
  • નિદાન: બાયોપ્સી, MRI, ઇલેક્ટ્રોન માઇક્રોસ્કોપી
  • સારવાર: કોઈ નથી (ફક્ત લક્ષણોનું ઉપચાર)
  • બચાવ: રોગી અંગોનું સેવન ટાળવું
  • NEETમાં રચના, ક્રિયાવિધિ, રોગો અને સંક્રમણ પર પ્રશ્નો પૂછાઈ શકે
  • પ્રિયોન્સ = પી (પ્રોટીન) + રિયો (રોગ) + ન્સ (નવો)
🎯 Exam Tip: NEETમાં પ્રિયોન્સ પર પ્રશ્નો પૂછતા વખતે, તેમની અસાધારણ રચના (માત્ર પ્રોટીન) અને TSE રોગો સાથેના સંકળાયેલા લક્ષણો (સ્પોંગી મગજ) પર ધ્યાન કેન્દ્રિત કરો. પ્રશ્નોમાં રોગોના નામો અને સંક્રમણના માધ્યમો પણ મહત્વના હોય છે.
