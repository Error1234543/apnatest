📖 ફ્લોએમ: સીવ નળી અને સહચર કોષો

ફ્લોએમનું પરિચય

ફ્લોએમ પાઉદરોમાં ખોરાકનું પરિવહન કરતું જટિલ પેશી છે. તે સીવ નળી, સહચર કોષો, ફ્લોએમ પેરેંકાઇમાની રચના અને ફ્લોએમ ફાઇબર ધરાવે છે. ખોરાકનું પરિવહન સક્રિય પ્રક્રિયા છે જેમાં ATPની જરૂર પડે છે.

  • ફ્લોએમ = ખોરાક પરિવહન પેશી
  • સક્રિય પરિવહન (ATP-આધારિત)
  • બહુકોષીય રચના
  • જાઇલમ સાથે મળીને વાસ્ક્યુલર બંડલ બનાવે

સીવ નળી (Sieve Tubes)

સીવ નળી લાંબી, નળાકાર કોષોની શ્રેણી છે જેમાં કોષકેન્દ્ર અનુપસ્થિત હોય છે. તેઓના છેડે સીવ પ્લેટ (સીવ છિદ્રો) હોય છે જે પરિવહનમાં મદદ કરે છે. કોષરસમાં P-પ્રોટીન અને કોષદીવાલ પાતળી હોય છે.

  • કોષકેન્દ્ર રહિત (Enucleate)
  • સીવ પ્લેટ = પરિવહન માટે છિદ્રો
  • P-પ્રોટીન = સીવ નળીમાં રક્ષણાત્મક
  • પાતળી કોષદીવાલ = કાર્યક્ષમ પરિવહન
🧠 Trick: સીવ = સુગર ટ્યુબ (Sugar Tube)

સહચર કોષો (Companion Cells)

સહચર કોષો સીવ નળી સાથે નજીકથી જોડાયેલા હોય છે અને તેમનું કોષકેન્દ્ર સક્રિય હોય છે. તેઓ સીવ નળીને ઉર્જા (ATP) અને ઉત્સેચક પુરા પાડે છે. સહચર કોષો અને સીવ નળી વચ્ચે પ્લાઝમોડેસમેટા હોય છે.

  • કોષકેન્દ્ર ધરાવે છે
  • ATP અને ઉત્સેચક પુરા પાડે
  • પ્લાઝમોડેસમેટા દ્વારા જોડાયેલા
  • સીવ નળીની કામગીરી જાળવે
🧠 Trick: સહચર = સીવ નળીના 'મિત્ર' (Companion = Friend)

સીવ નળી અને સહચર કોષોનું કાર્ય

સીવ નળી ખોરાક (મુખ્યત્વે સુક્રોઝ)નું પરિવહન કરે છે, જ્યારે સહચર કોષો તેને ઉર્જા અને ચયાપચય સહાય પુરા પાડે છે. પરિવહન માટે પ્રેશર ફ્લો થિયરી (Pressure Flow Theory) મહત્વની છે.

  • સુક્રોઝ = મુખ્ય પરિવહન પદાર્થ
  • પ્રેશર ફ્લો થિયરી = પરિવહનનો આધાર
  • સહચર કોષો = ચયાપચય સહાય
  • સક્રિય લોડિંગ (Active Loading) = સુક્રોઝને સીવ નળીમાં પ્રવેશ કરાવે

પ્રેશર ફ્લો થિયરી (Pressure Flow Theory)

આ થિયરી મુજબ, સુક્રોઝનો સંગ્રહ સીવ નળીમાં થાય છે, જેના કારણે ત્યાં દબાણ વધે છે. આ દબાણના કારણે સુક્રોઝનું પરિવહન થાય છે. સુક્રોઝનું લોડિંગ સહચર કોષો દ્વારા થાય છે.

  • સુક્રોઝનો સંગ્રહ = દબાણ વધારે
  • દબાણ ગ્રેડિયન્ટ = પરિવહન ચાલક શક્તિ
  • સક્રિય લોડિંગ = સહચર કોષો દ્વારા
  • અનલોડિંગ = ગંતવ્ય સ્થાને સુક્રોઝ બહાર કાઢવું
🧠 Trick: દબાણ = પરિવહનનો 'પંપ' (Pressure = Pump)

ફ્લોએમની અન્ય રચનાઓ

ફ્લોએમ પેરેંકાઇમાની રચના સંગ્રહ અને પરિવહનમાં મદદ કરે છે, જ્યારે ફ્લોએમ ફાઇબર યાંત્રિક ટેકો પુરો પાડે છે. ફ્લોએમમાં કેલોઝ પણ હોય છે, જે રક્ષણાત્મક કાર્ય કરે છે.

  • ફ્લોએમ પેરેંકાઇમા = સંગ્રહ
  • ફ્લોએમ ફાઇબર = યાંત્રિક ટેકો
  • કેલોઝ = રક્ષણાત્મક કોષો
  • બહુકોષીય રચના = કાર્યક્ષમતા વધારે

NEET માટે મહત્વના બિંદુઓ

સીવ નળી અને સહચર કોષો વચ્ચેના સંબંધને સમજવો અગત્યનો છે. પ્રેશર ફ્લો થિયરી અને સક્રિય લોડિંગ NEETમાં વારંવાર પૂછાય છે. ફ્લોએમની રચના અને કાર્યને યાદ રાખવું જરૂરી છે.

  • સીવ નળી + સહચર કોષો = એકકીટ (Unit)
  • પ્રેશર ફ્લો થિયરી = પરિવહનનો આધાર
  • સક્રિય લોડિંગ = ATP-આધારિત
  • ફ્લોએમ = ખોરાક પરિવહન પેશી

⚡ Quick Revision

  • ફ્લોએમ = ખોરાક પરિવહન પેશી
  • સીવ નળી = કોષકેન્દ્ર રહિત, સીવ પ્લેટ ધરાવે
  • સહચર કોષો = કોષકેન્દ્ર ધરાવે, ATP પુરા પાડે
  • પ્રેશર ફ્લો થિયરી = સુક્રોઝ પરિવહનનો આધાર
  • સક્રિય લોડિંગ = સુક્રોઝને સીવ નળીમાં પ્રવેશ કરાવે
  • પ્લાઝમોડેસમેટા = સીવ નળી અને સહચર કોષ વચ્ચેનો જોડાણ
  • ફ્લોએમ પેરેંકાઇમા = સંગ્રહ કાર્ય
  • ફ્લોએમ ફાઇબર = યાંત્રિક ટેકો
  • કેલોઝ = રક્ષણાત્મક કોષો
  • જાઇલમ + ફ્લોએમ = વાસ્ક્યુલર બંડલ
🎯 Exam Tip: NEETમાં ફ્લોએમ સંબંધિત પ્રશ્નો માટે પ્રેશર ફ્લો થિયરી અને સીવ નળી-સહચર કોષના સંબંધને યાદ રાખો. ડાયાગ્રામ આધારિત પ્રશ્નો માટે સીવ પ્લેટ અને પ્લાઝમોડેસમેટાની રચના સમજો.
