ગિબ્સ ઊર્જા

📖 ગિબ્સ ઊર્જા (Gibbs Energy)

પરિભાષા અને મૂળભૂત સંકલ્પના

ગિબ્સ મુક્ત ઊર્જા (G) એ થર્મોડાયનેમિક્સની એક સ્થિતિ ફંક્શન છે જે રાસાયણિક પ્રક્રિયાઓની સ્વયંસ્ફૂર્તિતા નક્કી કરે છે. તે એનથાલ્પી (H) અને એન્ટ્રોપી (S) ના સંયોજન પર આધારિત છે. સૂત્ર: ΔG = ΔH - TΔS, જ્યાં T એ તાપમાન (Kમાં) છે.

  • ΔG < 0: સ્વયંસ્ફૂર્ત પ્રક્રિયા (Exergonic)
  • ΔG > 0: અસ્વયંસ્ફૂર્ત પ્રક્રિયા (Endergonic)
  • ΔG = 0: સંતુલન અવસ્થા
  • G એ સ્થિતિ ફંક્શન છે, માર્ગ પર નિર્ભર નથી
  • તાપમાન (T) એ K (કેલ્વિન)માં હોવું જરૂરી છે
🧠 Trick: G = H - TS → 'Ghar Hoga To Sab Saste' (G < 0 = સ્વયંસ્ફૂર્ત)

ગિબ્સ ઊર્જા અને રાસાયણિક સંતુલન

ગિબ્સ ઊર્જામાં ફેરફાર (ΔG) પ્રક્રિયાની દિશા અને સંતુલનની સ્થિતિ બતાવે છે. જ્યારે ΔG = 0, ત્યારે પ્રક્રિયા સંતુલનમાં હોય છે. સંતુલન અચળાંક (K_eq) સાથેનો સંબંધ: ΔG° = -RT ln(K_eq).

  • ΔG° એ માપદંડ અવસ્થામાં ગિબ્સ ઊર્જા ફેરફાર છે
  • K_eq > 1 → ΔG° < 0 (પ્રક્રિયા આગળ વધે)
  • K_eq < 1 → ΔG° > 0 (પ્રક્રિયા પાછળ જાય)
  • R = 8.314 J/mol·K (ગેસ અચળાંક)
  • ln(K_eq) એ નેચરલ લોગારિધમ છે

જૈવિક પ્રક્રિયાઓમાં ગિબ્સ ઊર્જાનું મહત્વ

જીવવિજ્ઞાનમાં, ગિબ્સ ઊર્જા જૈવિક પ્રક્રિયાઓની સ્વયંસ્ફૂર્તિતા નક્કી કરે છે. ઉદા. શ્વસન (ΔG < 0) અને પ્રકાશસંશ્લેષણ (ΔG > 0, પરંતુ સૂર્યપ્રકાશથી ઊર્જા મળે છે). ATPનું જળવિભાજન (ΔG = -30.5 kJ/mol) એ એક ઉદાહરણ છે.

  • શ્વસન: ગ્લુકોઝનું ઓક્સિડેશન (ΔG = -2880 kJ/mol)
  • પ્રકાશસંશ્લેષણ: CO2 અને H2O થી ગ્લુકોઝ (ΔG > 0)
  • ATP → ADP + Pi (ΔG = -30.5 kJ/mol)
  • જૈવિક પ્રણાલીઓમાં એન્ડરગોનિક પ્રક્રિયાઓ એક્ઝર્ગોનિક પ્રક્રિયાઓ સાથે જોડાયેલી હોય છે
  • ઉત્સેચક પ્રક્રિયાઓ ΔG ને બદલતા નથી, પરંતુ સક્રિયણ ઊર્જા ઘટાડે છે
🧠 Trick: ATP: 'Adenosine Tri Phosphate - Energy Currency'

તાપમાનનો પ્રભાવ

તાપમાન (T) એ ΔG પર પ્રભાવ પાડે છે કારણ કે ΔG = ΔH - TΔS. ઉચ્ચ T એ ΔS ના મૂલ્યને વધારે મહત્વ આપે છે. ઉદા. બરફનું પીગળવું (ΔS > 0) ઉચ્ચ T એ સ્વયંસ્ફૂર્ત બનાવે છે.

  • ΔS > 0 અને ΔH < 0 → હમેશા સ્વયંસ્ફૂર્ત (ΔG < 0)
  • ΔS < 0 અને ΔH > 0 → હમેશા અસ્વયંસ્ફૂર્ત (ΔG > 0)
  • ΔS > 0 અને ΔH > 0 → ઉચ્ચ T એ ΔG < 0 બનાવે
  • ΔS < 0 અને ΔH < 0 → નીચા T એ ΔG < 0 બનાવે
  • તાપમાનનો પ્રભાવ ΔS પર નિર્ભર કરે છે

ગિબ્સ ઊર્જા અને રાસાયણિક કોષો

વિદ્યુતરાસાયણિક કોષોમાં, ગિબ્સ ઊર્જા અને વિદ્યુતપોટેન્શિયલ (E) વચ્ચે સંબંધ છે: ΔG° = -nFE°. જ્યાં n = ઇલેક્ટ્રોનની સંખ્યા, F = ફેરાડે અચળાંક (96500 C/mol), E° = માપદંડ વિદ્યુતપોટેન્શિયલ.

  • ΔG° < 0 → E° > 0 (સ્વયંસ્ફૂર્ત કોષ)
  • ΔG° > 0 → E° < 0 (અસ્વયંસ્ફૂર્ત કોષ)
  • F = 96500 C/mol (ફેરાડે અચળાંક)
  • n = ઇલેક્ટ્રોનની સંખ્યા પ્રક્રિયામાં
  • Nernst સમીકરણ: E = E° - (RT/nF) ln(Q)
🧠 Trick: ΔG° = -nFE° → 'Ghar Nahin Faraar Electron' (G < 0 = સ્વયંસ્ફૂર્ત)

ગિબ્સ ઊર્જા અને જૈવરાસાયણિક પ્રક્રિયાઓ

જૈવરાસાયણિક પ્રક્રિયાઓમાં, ગિબ્સ ઊર્જા પ્રક્રિયાની દિશા અને કાર્યક્ષમતા નક્કી કરે છે. ઉદા. ગ્લાયકોલિસિસ, ક્રેબ્સ ચક્ર, અને ઓક્સિડેટિવ ફોસ્ફોરાઇલેશનમાં ΔG નું મહત્વ છે.

  • ગ્લાયકોલિસિસ: ગ્લુકોઝ → પાયરુવેટ (ΔG = -146 kJ/mol)
  • ક્રેબ્સ ચક્ર: એસિટાઇલ-CoA → CO2 (ΔG = -40 kJ/mol per turn)
  • ઓક્સિડેટિવ ફોસ્ફોરાઇલેશન: NAD+ અને FAD નું રિડક્શન
  • ATP સિંથેસ: ΔG = +30.5 kJ/mol (એન્ડરગોનિક)
  • ઉત્સેચકો પ્રક્રિયાની ગતિ વધારે છે પરંતુ ΔG બદલતા નથી

⚡ Quick Revision

  • ગિબ્સ ઊર્જા (G) એ થર્મોડાયનેમિક્સની સ્થિતિ ફંક્શન છે
  • ΔG = ΔH - TΔS (સૂત્ર)
  • ΔG < 0 → સ્વયંસ્ફૂર્ત પ્રક્રિયા
  • ΔG > 0 → અસ્વયંસ્ફૂર્ત પ્રક્રિયા
  • ΔG = 0 → સંતુલન અવસ્થા
  • ΔG° = -RT ln(K_eq) (સંતુલન અચળાંક)
  • ΔG° = -nFE° (વિદ્યુતરાસાયણિક કોષો)
  • ATP → ADP + Pi (ΔG = -30.5 kJ/mol)
  • તાપમાન (T) એ ΔG પર પ્રભાવ પાડે છે
  • ઉત્સેચકો ΔG ને બદલતા નથી
🎯 Exam Tip: NEETમાં ગિબ્સ ઊર્જા સંબંધિત પ્રશ્નો માટે ΔG = ΔH - TΔS અને ΔG° = -nFE° સૂત્રો યાદ રાખો. પ્રશ્નોમાં ΔG ની ગણતરી કરવી પડે તે માટે T એ હમેશા Kમાં લેવું.
