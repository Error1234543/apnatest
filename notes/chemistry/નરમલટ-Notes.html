નોર્માલિટી

📖 નોર્માલિટી (Normality)

નોર્માલિટીની વ્યાખ્યા અને સૂત્ર

નોર્માલિટી એ દ્રાવણની સાંદ્રતાનું માપ છે, જે દ્રાવણના પ્રતિ લિટર ગ્રામ સમતુલ્યાંક (gram equivalent) દર્શાવે છે. તેને 'N' વડે દર્શાવાય છે. સૂત્ર: N = (ગ્રામમાં દ્રાવ્યનો જથ્થો) / (દ્રાવ્યનો સમતુલ્યાંક × દ્રાવણનું કદ (લિટરમાં)). નોર્માલિટી એસિડ-બેઇઝ ટાઇટ્રેશન અને રેડોક્સ પ્રક્રિયાઓમાં ખૂબ ઉપયોગી છે.

  • નોર્માલિટી = ગ્રામ સમતુલ્યાંક / લિટર
  • સમતુલ્યાંક = આણ્વિક ભાર / n-ફેક્ટર (n = H⁺, OH⁻ અથવા ઇલેક્ટ્રોનની સંખ્યા)
  • નોર્માલિટી મોલારિટી કરતાં અલગ છે (N = n × M)
🧠 Trick: નોર્મલ માણસને 'N' ગ્રામ સમતુલ્યાંક જોઈએ, લિટરમાં!

n-ફેક્ટરની ગણતરી

n-ફેક્ટર એ એસિડમાં H⁺ આયન, બેઇઝમાં OH⁻ આયન અથવા રેડોક્સ પ્રક્રિયામાં સ્થાનાંતરિત થતા ઇલેક્ટ્રોનની સંખ્યા છે. ઉદાહરણ: HCl માટે n=1, H₂SO₄ માટે n=2, NaOH માટે n=1, KMnO₄ (ઍસિડિક માધ્યમ) માટે n=5.

  • એસિડ: n = H⁺ આયનની સંખ્યા
  • બેઇઝ: n = OH⁻ આયનની સંખ્યા
  • રેડોક્સ: n = ઇલેક્ટ્રોનની સંખ્યા (ઑક્સિડેશન/રિડક્શનમાં)
🧠 Trick: એસિડ-બેઇઝ: 'H⁺ કે OH⁻ ગણો', રેડોક્સ: 'e⁻ ગણો'!

નોર્માલિટી અને મોલારિટી વચ્ચેનો સંબંધ

નોર્માલિટી (N) અને મોલારિટી (M) વચ્ચેનો સંબંધ: N = n × M, જ્યાં n = n-ફેક્ટર. ઉદાહરણ: 1 M H₂SO₄ = 2 N H₂SO₄ (n=2). આ સંબંધ ટાઇટ્રેશન ગણતરીઓમાં મહત્વપૂર્ણ છે.

  • N = n × M
  • જો n=1 હોય તો N = M (ઉદા. HCl, NaOH)
  • રેડોક્સ પ્રક્રિયામાં n ફેરફાર થાય છે (ઉદા. KMnO₄: ઍસિડિકમાં n=5, બેઝિકમાં n=3)
🧠 Trick: નોર્મલ = n × મોલર!

નોર્માલિટીના ઉપયોગો

નોર્માલિટીનો મુખ્ય ઉપયોગ એસિડ-બેઇઝ ટાઇટ્રેશન અને રેડોક્સ ટાઇટ્રેશનમાં થાય છે. તે દ્રાવણની પ્રક્રિયાત્મક ક્ષમતા દર્શાવે છે. ઉદાહરણ: Na₂CO₃નું ટાઇટ્રેશન HCl સાથે (n=2), KMnO₄નું ટાઇટ્રેશન Fe²⁺ સાથે (n=5).

  • ટાઇટ્રેશન ગણતરીઓમાં ઉપયોગી
  • રેડોક્સ પ્રક્રિયાઓમાં ઇલેક્ટ્રોન સ્થાનાંતરણ માપે
  • ઔદ્યોગિક પ્રક્રિયાઓમાં સાંદ્રતા નિયંત્રણ
🧠 Trick: ટાઇટ્રેશનમાં 'N' વગર નહીં ચાલે!

નોર્માલિટીની ગણતરીના ઉદાહરણો

1. 4.9 ગ્રામ H₂SO₄ (આણ્વિક ભાર=98) 500 મિલી દ્રાવણમાં ઓગાળવામાં આવે તો નોર્માલિટી શોધો. ઉકેલ: સમતુલ્યાંક = 98/2 = 49, N = (4.9/49) / 0.5 = 0.2 N. 2. 5.3 ગ્રામ Na₂CO₃ (આણ્વિક ભાર=106) 200 મિલી દ્રાવણમાં ઓગાળવામાં આવે તો નોર્માલિટી શોધો. ઉકેલ: સમતુલ્યાંક = 106/2 = 53, N = (5.3/53) / 0.2 = 0.5 N.

  • સમતુલ્યાંક = આણ્વિક ભાર / n
  • નોર્માલિટી = (ગ્રામ / સમતુલ્યાંક) / કદ (લિટર)
  • કદ હંમેશા લિટરમાં લો
🧠 Trick: ગ્રામ ભાગ્યા સમતુલ્યાંક, ભાગ્યા લિટર — નોર્મલ!

નોર્માલિટીના મર્યાદાઓ અને વૈકલ્પિક એકમો

નોર્માલિટીનો ઉપયોગ મર્યાદિત છે કારણ કે n-ફેક્ટર પ્રક્રિયા પર આધારિત બદલાય છે (ઉદા. KMnO₄). વૈકલ્પિક એકમો: મોલારિટી (M), મોલાલિટી (m), અને ફોર્માલિટી (F). જૈવિક પ્રણાલીઓમાં મોલારિટી વધુ ઉપયોગી છે કારણ કે તે પ્રક્રિયાથી સ્વતંત્ર છે.

  • n-ફેક્ટર પ્રક્રિયા આધારિત બદલાય છે
  • મોલારિટી વધુ સાર્વત્રિક એકમ છે
  • જૈવરસાયણમાં મોલારિટી પ્રાધાન્યક્ષમ
🧠 Trick: નોર્મલ બદલાય, મોલર સ્થિર!

⚡ Quick Revision

  • નોર્માલિટી = ગ્રામ સમતુલ્યાંક / લિટર (N)
  • સમતુલ્યાંક = આણ્વિક ભાર / n-ફેક્ટર
  • n-ફેક્ટર: એસિડમાં H⁺, બેઇઝમાં OH⁻, રેડોક્સમાં e⁻
  • N = n × M (મોલારિટી)
  • ટાઇટ્રેશન ગણતરીઓમાં નોર્માલિટી જરૂરી
  • KMnO₄: ઍસિડિકમાં n=5, બેઝિકમાં n=3
  • નોર્માલિટી પ્રક્રિયા આધારિત બદલાય છે
  • જૈવિક પ્રણાલીઓમાં મોલારિટી વધુ ઉપયોગી
  • ગણતરી: ગ્રામ → સમતુલ્યાંક → નોર્માલિટી
  • કદ હંમેશા લિટરમાં લો
🎯 Exam Tip: નોર્માલિટીના પ્રશ્નોમાં n-ફેક્ટરની ગણતરી પર ખાસ ધ્યાન આપો — એસિડ-બેઇઝ માટે H⁺/OH⁻ ગણો, રેડોક્સ માટે ઇલેક્ટ્રોન ગણો. ટાઇટ્રેશન પ્રશ્નોમાં N₁V₁ = N₂V₂ સૂત્ર યાદ રાખો અને એકમો લિટરમાં રૂપાંતરિત કરો.
