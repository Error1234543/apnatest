પ્રકાશીય સમઘટન

📖 પ્રકાશીય સમઘટન

પરિભાષા અને મૂળભૂત સિદ્ધાંત

પ્રકાશીય સમઘટન એ પ્રકાશની મદદથી કાર્બનિક સંયોજનોના વિઘટનની પ્રક્રિયા છે. તે પ્રકાશસંશ્લેષણનો ઉલટ પ્રક્રિયા ગણી શકાય, જ્યાં પ્રકાશ ઉર્જા વડે CO₂ અને H₂O બનાવવા કાર્બોદિતો તૂટે છે. આ પ્રક્રિયા મુખ્યત્વે હરિતદ્રવ્ય ધરાવતા કોષોમાં થાય છે.

  • પ્રકાશીય સમઘટન = પ્રકાશ + ઓક્સિડેટિવ વિઘટન
  • હરિતદ્રવ્ય (ક્લોરોફિલ) ઉત્સેચક તરીકે કામ કરે છે
  • CO₂ + H₂O + પ્રકાશ → કાર્બોદિતોનું વિઘટન
  • ઓક્સિજન મુક્ત થાય છે
🧠 Trick: પ્રકાશ + હરિત = વિઘટન (પ્રહવિ)

પ્રકાશીય સમઘટનની પ્રક્રિયા

પ્રકાશીય સમઘટન બે તબક્કામાં થાય છે: પ્રકાશ-આશ્રિત પ્રક્રિયા (લાઇટ ડિપેન્ડન્ટ) અને પ્રકાશ-સ્વતંત્ર પ્રક્રિયા (લાઇટ ઇન્ડિપેન્ડન્ટ). પ્રથમ તબક્કામાં પ્રકાશ ઉર્જા શોષાઈ ATP અને NADPH બનાવે છે, જ્યારે બીજા તબક્કામાં CO₂ ફિક્સેશન થાય છે.

  • પ્રકાશ-આશ્રિત: પ્રકાશ + H₂O → O₂ + ATP + NADPH
  • પ્રકાશ-સ્વતંત્ર: CO₂ + ATP + NADPH → કાર્બોદિતો
  • કેલ્વિન ચક્ર (C₃ ચક્ર) મુખ્ય પથ છે
  • RUBP + CO₂ → 2x 3-PGA (પ્રથમ સ્થિર સંયોજન)
🧠 Trick: પ્રકાશ-આશ્રિત = ATP-NADPH, પ્રકાશ-સ્વતંત્ર = CO₂ ફિક્સ

હરિતદ્રવ્યની ભૂમિકા

હરિતદ્રવ્ય (ક્લોરોફિલ a અને b) પ્રકાશ શોષી પ્રકાશીય સમઘટનની શરૂઆત કરે છે. તે પ્રકાશ ઉર્જાને રાસાયણિક ઉર્જામાં રૂપાંતરિત કરે છે. ક્લોરોફિલ a મુખ્ય પ્રકાશ-શોષક છે, જ્યારે ક્લોરોફિલ b સહાયક ભૂમિકા ભજવે છે.

  • ક્લોરોફિલ a: પ્રાથમિક પ્રકાશ-શોષક
  • ક્લોરોફિલ b: સહાયક, નીલ-લીલો પ્રકાશ શોષે છે
  • હરિતદ્રવ્ય પાસે Mg²⁺ કેંદ્ર છે
  • થાયલેકોઈડમાં સ્થિત છે
🧠 Trick: ક્લોરોફિલ = પ્રકાશ + Mg (કલપ્રમ)

પ્રકાશ-આશ્રિત પ્રક્રિયા (લાઇટ ડિપેન્ડન્ટ)

આ પ્રક્રિયા થાયલેકોઈડના ગ્રેના અને સ્ટ્રોમા થાયલેકોઈડમાં થાય છે. પ્રકાશ શોષણથી ઇલેક્ટ્રોન ઉત્તેજિત થાય છે, જે ઇલેક્ટ્રોન પરિવહન શૃંખલા (ETC) મારફતે વહન પામે છે. આ દરમિયાન H₂Oનું ફોટોલિસિસ થાય છે, જેમાં O₂, પ્રોટોન (H⁺) અને ઇલેક્ટ્રોન મુક્ત થાય છે.

  • ફોટોલિસિસ: 2H₂O → 4H⁺ + 4e⁻ + O₂
  • PSII → PSI → NADP⁺ → NADPH
  • પ્રોટોન ગ્રેડિયન્ટથી ATP સિંથેઝ થાય છે
  • ઓક્સિજન બાયપ્રોડક્ટ છે
🧠 Trick: ફોટોલિસિસ = H₂O → O₂ + H⁺ + e⁻ (ફોહો)

કેલ્વિન ચક્ર (C₃ ચક્ર)

પ્રકાશ-સ્વતંત્ર પ્રક્રિયામાં CO₂ ફિક્સેશન કેલ્વિન ચક્ર દ્વારા થાય છે. આ ચક્ર સ્ટ્રોમામાં થાય છે અને તેને RUBISCO ઉત્સેચકની જરૂર પડે છે. CO₂નું ફિક્સેશન RUBP (રિબ્યુલોઝ-1,5-બાઈસફોસ્ફેટ) સાથે થાય છે, જેથી 3-PGA બને છે.

  • RUBISCO: દુનિયાનો સૌથી વપરાતો ઉત્સેચક
  • 3-PGA → 1,3-બાઈસફોસ્ફોગ્લિસરેટ → G3P
  • G3P વડે ગ્લુકોઝ બને છે
  • રિજનરેશન: RUBP પુનઃબનાવવું
🧠 Trick: RUBISCO + CO₂ = 3-PGA (રુકો)

C₄ અને CAM પથ

C₄ પથ (હેચ-સ્લેક પથ) અને CAM પથ (ક્રેસ્યુલેશિયન એસિડ મેટાબોલિઝમ) ઉચ્ચ તાપમાન અને શુષ્ક પરિસ્થિતિઓમાં અસરકારક છે. C₄ પથમાં CO₂નું પ્રથમ ફિક્સેશન PEP કાર્બોક્સિલેઝ વડે થાય છે, જ્યારે CAM પથ રાત્રે CO₂ ફિક્સ કરે છે.

  • C₄ પથ: PEP + CO₂ → ઓક્સાલોએસિટેટ (4-C)
  • CAM પથ: રાત્રે CO₂ ફિક્સ, દિવસે કેલ્વિન ચક્ર
  • C₄ છોડ: મકાઈ, કણસ, શેરડી
  • CAM છોડ: કેક્ટસ, પાઇનએપલ
🧠 Trick: C₄ = દિવસ, CAM = રાત (સીરાત)

પ્રકાશીય સમઘટનનું મહત્વ

પ્રકાશીય સમઘટન વનસ્પતિઓ માટે ઉર્જા અને કાર્બનિક સંયોજનોનું સ્રોત છે. તે વાતાવરણમાં O₂નું સંતુલન જાળવે છે અને ખોરાક શૃંખલાનું આધાર છે. આ પ્રક્રિયા વડે વનસ્પતિઓ તેમના વૃદ્ધિ અને વિકાસ માટે જરૂરી ઉર્જા મેળવે છે.

  • O₂નું ઉત્પાદન: વાતાવરણીય સંતુલન
  • કાર્બોદિતોનું સિંથેઝ: ઉર્જા સંગ્રહ
  • ખોરાક શૃંખલાનું આધાર
  • વનસ્પતિ વૃદ્ધિ અને વિકાસ

⚡ Quick Revision

  • પ્રકાશીય સમઘટન = પ્રકાશ + CO₂ + H₂O → કાર્બોદિતો + O₂
  • હરિતદ્રવ્ય (ક્લોરોફિલ) પ્રકાશ શોષે છે
  • પ્રકાશ-આશ્રિત: ATP + NADPH બને છે
  • પ્રકાશ-સ્વતંત્ર: CO₂ ફિક્સેશન (કેલ્વિન ચક્ર)
  • ફોટોલિસિસ: H₂O → O₂ + H⁺ + e⁻
  • RUBISCO: CO₂ + RUBP → 3-PGA
  • C₄ પથ: PEP + CO₂ → ઓક્સાલોએસિટેટ
  • CAM પથ: રાત્રે CO₂ ફિક્સ, દિવસે કેલ્વિન ચક્ર
  • O₂ અને કાર્બોદિતો પ્રકાશીય સમઘટનના મુખ્ય ઉત્પાદનો છે
  • વનસ્પતિઓમાં ઉર્જા અને કાર્બનિક સંયોજનોનું સ્રોત
🎯 Exam Tip: પ્રકાશીય સમઘટન અને પ્રકાશસંશ્લેષણ વચ્ચેનો તફાવત સ્પષ્ટ કરો: પ્રકાશસંશ્લેષણમાં CO₂ અને H₂O વડે ગ્લુકોઝ બને છે, જ્યારે પ્રકાશીય સમઘટનમાં ગ્લુકોઝ તૂટી CO₂ અને H₂O બને છે.
