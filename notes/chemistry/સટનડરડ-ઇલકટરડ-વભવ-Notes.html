સ્ટાન્ડર્ડ ઇલેક્ટ્રોડ વિભવ

📖 સ્ટાન્ડર્ડ ઇલેક્ટ્રોડ વિભવ

પરિભાષા અને મૂળભૂત સંકલ્પના

સ્ટાન્ડર્ડ ઇલેક્ટ્રોડ વિભવ (E°) એ ઇલેક્ટ્રોડ અને તેની આયનની સાંદ્રતા 1 M હોવાની સ્થિતિમાં માપવામાં આવતો વિદ્યુત વિભવ છે. તે ઇલેક્ટ્રોડની વિદ્યુત રાસાયણિક શ્રેણીમાં સ્થાન નક્કી કરે છે. હાઇડ્રોજન ઇલેક્ટ્રોડને સ્ટાન્ડર્ડ તરીકે લેવામાં આવે છે, જેનું E° = 0 V ગણવામાં આવે છે.

  • E° એ સ્ટાન્ડર્ડ સ્થિતિમાં ઇલેક્ટ્રોડનો વિભવ છે.
  • હાઇડ્રોજન ઇલેક્ટ્રોડનો E° = 0 V છે.
  • તાપમાન 298 K અને દબાણ 1 atm હોવું જરૂરી છે.
  • સાંદ્રતા 1 M હોવી જોઈએ.
🧠 Trick: H₂ → 2H⁺ + 2e⁻; E° = 0 (હાઇડ્રોજન મૂળભૂત સ્ટાન્ડર્ડ)

નર્નસ્ટ સમીકરણ

નર્નસ્ટ સમીકરણ ઇલેક્ટ્રોડ વિભવને સાંદ્રતા અને તાપમાન સાથે જોડે છે: E = E° - (RT/nF) ln(Q). જ્યાં Q એ પ્રતિક્રિયા ભાગફલ છે, R એ ગેસ અચળાંક, T એ તાપમાન, n એ ઇલેક્ટ્રોનની સંખ્યા અને F એ ફેરેડે અચળાંક છે.

  • E = E° - (0.0591/n) log(Q) (25°C પર)
  • Q = [ઑક્સીડાઇઝડ]/[રિડ્યુસ્ડ] (પ્રતિક્રિયા ભાગફલ)
  • n = ઇલેક્ટ્રોનની સંખ્યા
  • F = 96500 C/mol (ફેરેડે અચળાંક)
🧠 Trick: Nernst = Normal - (RT/nF) ln(Q)

વિદ્યુત રાસાયણિક શ્રેણી

વિદ્યુત રાસાયણિક શ્રેણી ઇલેક્ટ્રોડને તેમના સ્ટાન્ડર્ડ વિભવના આધારે ઓર્ડર કરે છે. જેમ જેમ E° વધે તેમ ઇલેક્ટ્રોડ ઓક્સિડેશન તરફ વધુ પ્રબળ બને છે. ઉચ્ચ E° વાળા ઇલેક્ટ્રોડ રિડક્શન તરફ પ્રબળ હોય છે.

  • E° વધુ હોવો = રિડક્શન પ્રબળ
  • E° ઓછો હોવો = ઓક્સિડેશન પ્રબળ
  • Li (E° = -3.04 V) સૌથી ઓક્સિડેશન પ્રબળ
  • F₂ (E° = +2.87 V) સૌથી રિડક્શન પ્રબળ
🧠 Trick: લિથિયમ લેફ્ટ, ફ્લોરિન રાઇટ (ઓક્સિડેશનથી રિડક્શન)

ગિબ્સ મુક્ત ઉર્જા અને વિભવનો સંબંધ

ગિબ્સ મુક્ત ઉર્જા (ΔG) અને સેલ વિભવ (E°_cell) વચ્ચેનો સંબંધ ΔG = -nFE°_cell છે. જો E°_cell ધન હોય તો પ્રતિક્રિયા સ્વત:સ્ફૂર્તિ છે. જો E°_cell ઋણ હોય તો પ્રતિક્રિયા અસ્વત:સ્ફૂર્તિ છે.

  • ΔG = -nFE°_cell
  • E°_cell > 0 → સ્વત:સ્ફૂર્તિ પ્રતિક્રિયા
  • E°_cell < 0 → અસ્વત:સ્ફૂર્તિ પ્રતિક્રિયા
  • nF એ કુલ વહન થતાં ચાર્જની માત્રા છે.
🧠 Trick: ધન E° = સ્વત: ચાલે, ઋણ E° = ન ચાલે

સેલ વિભવની ગણતરી

સેલ વિભવ (E°_cell) એ કેથોડ અને એનોડના સ્ટાન્ડર્ડ વિભવનો તફાવત છે: E°_cell = E°_cathode - E°_anode. કેથોડ પર રિડક્શન અને એનોડ પર ઓક્સિડેશન થાય છે.

  • E°_cell = E°_રિડક્શન - E°_ઓક્સિડેશન
  • કેથોડ = રિડક્શન (ઉચ્ચ E°)
  • એનોડ = ઓક્સિડેશન (નીચો E°)
  • સેલ વિભવ હંમેશા ધન હોવો જોઈએ સ્વત:સ્ફૂર્તિ માટે.
🧠 Trick: કેથોડ - એનોડ = સેલનો વિભવ

અનુપ્રયોગિક મહત્વ

સ્ટાન્ડર્ડ ઇલેક્ટ્રોડ વિભવનો ઉપયોગ બેટરી, કોરોઝન અને ઇલેક્ટ્રોલાઇસિસની સમજણમાં થાય છે. તે ધાતુઓની પ્રતિક્રિયાશીલતા અને વિદ્યુત રાસાયણિક સેલની ડિઝાઇનમાં મદદ કરે છે.

  • બેટરીની ડિઝાઇનમાં ઉપયોગી
  • કોરોઝનની આગાહી કરે છે
  • ઇલેક્ટ્રોલાઇસિસમાં વિભવ નક્કી કરે છે
  • ધાતુની શુદ્ધિકરણ પ્રક્રિયામાં મદદરૂપ

પરીક્ષામાં સાવચેતી

સ્ટાન્ડર્ડ ઇલેક્ટ્રોડ વિભવની ગણતરીમાં હંમેશા હાઇડ્રોજન ઇલેક્ટ્રોડને સ્ટાન્ડર્ડ તરીકે લો. નર્નસ્ટ સમીકરણમાં Qની ગણતરી યોગ્ય રીતે કરો, ખાસ કરીને ગેસ અને ઘન પદાર્થો માટે.

  • હાઇડ્રોજન ઇલેક્ટ્રોડને સ્ટાન્ડર્ડ ગણો
  • Qમાં ઘન અને શુદ્ધ દ્રાવણોની સાંદ્રતા 1 લો
  • તાપમાન 298 K પર 0.0591નો ઉપયોગ કરો
  • ઇલેક્ટ્રોડ લખતી વખતે ઓક્સિડેશન અને રિડક્શન હાફ-રિએક્શન યોગ્ય રીતે લખો.

⚡ Quick Revision

  • E° એ સ્ટાન્ડર્ડ ઇલેક્ટ્રોડ વિભવ છે, હાઇડ્રોજન ઇલેક્ટ્રોડનો E° = 0 V
  • નર્નસ્ટ સમીકરણ: E = E° - (0.0591/n) log(Q) (25°C)
  • વિદ્યુત રાસાયણિક શ્રેણી: Li સૌથી ઓક્સિડેશન પ્રબળ, F₂ સૌથી રિડક્શન પ્રબળ
  • ΔG = -nFE°_cell; ધન E°_cell = સ્વત:સ્ફૂર્તિ પ્રતિક્રિયા
  • E°_cell = E°_કેથોડ - E°_એનોડ
  • કેથોડ પર રિડક્શન, એનોડ પર ઓક્સિડેશન
  • બેટરી, કોરોઝન અને ઇલેક્ટ્રોલાઇસિસમાં ઉપયોગી
  • હાઇડ્રોજન ઇલેક્ટ્રોડ સ્ટાન્ડર્ડ છે
  • Qમાં ઘન અને શુદ્ધ પદાર્થોની સાંદ્રતા 1 લો
  • તાપમાન 298 K પર 0.0591નો અચળાંક વાપરો
🎯 Exam Tip: સેલ વિભવની ગણતરીમાં કેથોડ અને એનોડની ઓળખ સ્પષ્ટ કરો, અને હાફ-રિએક્શન યોગ્ય રીતે લખો. નર્નસ્ટ સમીકરણમાં Qની ગણતરી ભૂલશો નહીં.
