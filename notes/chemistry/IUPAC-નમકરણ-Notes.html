IUPAC નામકરણ

📖 IUPAC નામકરણ (આંતરરાષ્ટ્રીય શુદ્ધ અને અપ્લાઈડ રસાયણવિજ્ઞાન સંગઠન નામકરણ)

મૂળભૂત નિયમો

IUPAC નામકરણમાં કાર્બનિક સંયોજનોના નામ લાંબી સાંકળના આધારે નક્કી થાય છે. સૌથી લાંબી સતત કાર્બન સાંકળને મુખ્ય સાંકળ કહેવાય છે, જેના આધારે મૂળ નામ નક્કી કરવામાં આવે છે.

  • લાંબી સાંકળને પ્રાથમિકતા આપો
  • સાંકળની લંબાઈ અનુસાર પ્રીફિક્સ (મેથ-, ઇથ-, પ્રોપ-, બ્યુટ- વગેરે) નક્કી કરો
  • સાંકળમાં ડબલ બોન્ડ (ઇન) અથવા ટ્રિપલ બોન્ડ (આયન) હાજર હોય તો તેને સૂચવો
  • હેલોજન, હાઇડ્રોક્સિલ વગેરે પ્રતિચયોને સબસ્ટિટ્યુટ તરીકે ગણો
🧠 Trick: લાંબી સાંકળ → મુખ્ય નામ, બોન્ડ → ઇન/આયન, પ્રતિચય → સબસ્ટિટ્યુટ

મુખ્ય સાંકળની પસંદગી

જ્યારે એકથી વધુ સાંકળો શક્ય હોય ત્યારે સૌથી લાંબી સાંકળ, સૌથી વધુ સબસ્ટિટ્યુટ્સ ધરાવતી સાંકળ અથવા સૌથી ઓછી ક્રમાંકિત સબસ્ટિટ્યુટ્સ ધરાવતી સાંકળ પસંદ કરો.

  • લાંબી સાંકળને પ્રાથમિકતા
  • સરખી લંબાઈ હોય તો વધુ સબસ્ટિટ્યુટ્સ વાળી સાંકળ
  • સરખા સબસ્ટિટ્યુટ્સ હોય તો ઓછા ક્રમાંકિત સબસ્ટિટ્યુટ્સ વાળી સાંકળ
  • ડબલ/ટ્રિપલ બોન્ડ ધરાવતી સાંકળને પ્રાથમિકતા
🧠 Trick: લાંબી → વધુ સબસ્ટિટ્યુટ્સ → ઓછા ક્રમાંક

સબસ્ટિટ્યુટ્સ અને તેમના સ્થાન

સબસ્ટિટ્યુટ્સને તેમના સ્થાન અનુસાર ક્રમાંકિત કરો. ક્રમાંકન સાંકળના એવા છેડેથી શરૂ કરો જેથી સબસ્ટિટ્યુટ્સને નીચા ક્રમાંક મળે. સબસ્ટિટ્યુટ્સના નામને મુખ્ય નામ પહેલા લખો.

  • સબસ્ટિટ્યુટ્સને નીચા ક્રમાંકથી શરૂ કરો
  • એકથી વધુ સબસ્ટિટ્યુટ્સ હોય તો અલ્ફાબેટિકલ ઓર્ડરમાં લખો
  • સરખા સબસ્ટિટ્યુટ્સ હોય તો ડાય-, ટ્રાય- વગેરે પ્રીફિક્સ વાપરો
  • હેલોજન: ફ્લોરો-, ક્લોરો-, બ્રોમો-, આયોડો-
🧠 Trick: નીચા ક્રમાંક → અલ્ફાબેટિકલ → ડાય-, ટ્રાય-

ક્રિયાત્મક સમૂહો (Functional Groups)

ક્રિયાત્મક સમૂહોને પ્રાથમિકતા અનુસાર નામ આપવામાં આવે છે. કેટલાક સમૂહો જેમ કે -COOH, -SO3H, -CHO વગેરેને કર્બોક્સિલિક એસિડ, સલ્ફોનિક એસિડ, એલ્ડિહાઇડ વગેરે કહેવાય છે.

  • -COOH → કર્બોક્સિલિક એસિડ (ઓલિક એસિડ)
  • -OH → હાઇડ્રોક્સિલ (ઓલ)
  • -CHO → એલ્ડિહાઇડ (એલ)
  • -C=O → કિટોન (ઓન)
  • -NH2 → એમિન (એમિન)
🧠 Trick: COOH → ઓલિક, OH → ઓલ, CHO → એલ, C=O → ઓન

બહુવિધ બોન્ડ અને સબસ્ટિટ્યુટ્સ

જ્યારે એકથી વધુ ડબલ અથવા ટ્રિપલ બોન્ડ હોય ત્યારે તેમના સ્થાનને ક્રમાંકિત કરો અને 'ડાયઇન', 'ટ્રાયઇન' વગેરે વાપરો. સબસ્ટિટ્યુટ્સ અને બોન્ડ બંનેના ક્રમાંકને નીચા ક્રમાંકથી શરૂ કરો.

  • ડબલ બોન્ડ → -ઇન, ટ્રિપલ બોન્ડ → -આયન
  • બહુવિધ બોન્ડ → ડાય-, ટ્રાય- વગેરે
  • બોન્ડ અને સબસ્ટિટ્યુટ્સ બંનેના ક્રમાંક નીચા થી શરૂ કરો
  • બોન્ડને ક્રમાંક આપતી વખતે ડબલ બોન્ડને પ્રાથમિકતા
🧠 Trick: ડબલ → ઇન, ટ્રિપલ → આયન, બહુવિધ → ડાય-, ટ્રાય-

ચક્રીય સંયોજનો (Cyclic Compounds)

ચક્રીય સંયોજનો માટે 'સાયક્લો-' પ્રીફિક્સ વાપરો. ચક્રમાં સબસ્ટિટ્યુટ્સના સ્થાનને ક્રમાંકિત કરો અને સૌથી નીચા ક્રમાંકથી શરૂ કરો. જો ચક્રમાં ડબલ બોન્ડ હોય તો તેને 'સાયક્લોઅલ્કીન' કહેવાય.

  • ચક્રીય સાંકળ → સાયક્લો-
  • સબસ્ટિટ્યુટ્સના ક્રમાંક નીચા થી શરૂ કરો
  • ડબલ બોન્ડ → સાયક્લોઅલ્કીન
  • બહુવિધ સબસ્ટિટ્યુટ્સ → અલ્ફાબેટિકલ ઓર્ડર
🧠 Trick: ચક્ર → સાયક્લો-, ડબલ બોન્ડ → સાયક્લોઅલ્કીન

ઓપ્ટિકલ આઇસોમરિઝમ (R-S નામકરણ)

જ્યારે કાર્બન પર ચાર ભિન્ન સમૂહો હોય ત્યારે તે કિરાલ કહેવાય છે. CIP નિયમો અનુસાર, પ્રાથમિકતા અનુસાર R અથવા S કોન્ફિગરેશન નક્કી કરવામાં આવે છે.

  • કિરાલ કાર્બન → ચાર ભિન્ન સમૂહો
  • CIP નિયમો: ઉચ્ચ અણુ ક્રમાંક → ઉચ્ચ પ્રાથમિકતા
  • R → ક્લોકવાઇઝ, S → એન્ટી-ક્લોકવાઇઝ
  • R/S નામકરણ માટે કિરાલ સેન્ટરને ઓળખો
🧠 Trick: R → રાઇટ (ક્લોકવાઇઝ), S → લેફ્ટ (એન્ટી-ક્લોકવાઇઝ)

  • IUPAC નામકરણમાં લાંબી સાંકળને મુખ્ય સાંકળ કહેવાય છે
  • સબસ્ટિટ્યુટ્સને નીચા ક્રમાંકથી શરૂ કરો અને અલ્ફાબેટિકલ ઓર્ડરમાં લખો
  • ડબલ બોન્ડ → -ઇન, ટ્રિપલ બોન્ડ → -આયન
  • ક્રિયાત્મક સમૂહોને પ્રાથમિકતા અનુસાર નામ આપો (ઉદા. -COOH → ઓલિક એસિડ)
  • ચક્રીય સંયોજનો માટે 'સાયક્લો-' પ્રીફિક્સ વાપરો
  • કિરાલ કાર્બન માટે R/S નામકરણ કરો
  • બહુવિધ સબસ્ટિટ્યુટ્સ → ડાય-, ટ્રાય- વગેરે પ્રીફિક્સ વાપરો
  • હેલોજન સબસ્ટિટ્યુટ્સ: ફ્લોરો-, ક્લોરો-, બ્રોમો-, આયોડો-
  • ક્રિયાત્મક સમૂહો જેમ કે -OH, -CHO, -C=O માટે ઓલ, એલ, ઓન વાપરો
  • બોન્ડ અને સબસ્ટિટ્યુટ્સ બંનેના ક્રમાંક નીચા થી શરૂ કરો
🎯 Exam Tip: NEETમાં IUPAC નામકરણના પ્રશ્નો માટે સાંકળની લંબાઈ, સબસ્ટિટ્યુટ્સના સ્થાન અને ક્રિયાત્મક સમૂહોની પ્રાથમિકતા પર ધ્યાન આપો. સૌથી સામાન્ય ભૂલો સબસ્ટિટ્યુટ્સના ક્રમાંક અને અલ્ફાબેટિકલ ઓર્ડરમાં થાય છે.
