ફેલિંગ પરીક્ષણ

📖 ફેલિંગ પરીક્ષણ

પરિચય અને ઉદ્દેશ્ય

ફેલિંગ પરીક્ષણ એ કાર્બોદિતો (ખાસ કરીને શર્કરા)ની હાજરી શોધવા માટેનો રાસાયણિક પરીક્ષણ છે. આ પરીક્ષણ ફેલિંગ દ્રાવણ (કોપર સલ્ફેટ + સોડિયમ પોટેશિયમ ટાર્ટરેટ + સોડિયમ હાઇડ્રોક્સાઇડ)નો ઉપયોગ કરે છે, જે શર્કરા સાથે પ્રક્રિયા કરીને લાલ ક્યુપ્રસ ઓક્સાઇડનું અવક્ષેપન આપે છે.

  • ફેલિંગ દ્રાવણ A + B નું મિશ્રણ છે (A: કોપર સલ્ફેટ, B: રોકેફેલર દ્રાવણ).
  • શર્કરા (ગ્લુકોઝ, ફ્રુક્ટોઝ, લેક્ટોઝ) સાથે પ્રક્રિયા થાય છે.
  • ગરમ કરવાથી લાલ અવક્ષેપ મળે છે.
  • નન-રિડ્યુસિંગ શર્કરા (સુક્રોઝ) માટે નકારાત્મક પરિણામ.
🧠 Trick: ફેલિંગ = ફેંકી દો લાલ રંગ (ગ્લુકોઝ માટે)

ફેલિંગ દ્રાવણની રચના

ફેલિંગ દ્રાવણ બે ભાગોમાં તૈયાર કરવામાં આવે છે: દ્રાવણ A (કોપર સલ્ફેટ) અને દ્રાવણ B (સોડિયમ પોટેશિયમ ટાર્ટરેટ + સોડિયમ હાઇડ્રોક્સાઇડ). બંનેને સમાન પ્રમાણમાં મિશ્ર કરવામાં આવે છે, જે કોપર (II) ટાર્ટરેટ કોમ્પ્લેક્સ બનાવે છે.

  • દ્રાવણ A: 3.5% CuSO₄·5H₂O
  • દ્રાવણ B: સોડિયમ પોટેશિયમ ટાર્ટરેટ (રોકેફેલર ક્ષાર) + NaOH
  • મિશ્રણ સમયે નીલો રંગ બદલાઈ જાય છે.
  • કોપર (II) આયનો સ્થિર રહે છે.
🧠 Trick: A = CuSO₄ (નીલો), B = ક્ષાર + ક્ષારક (રંગહીન)

પ્રક્રિયા યંત્રણ (Mechanism)

ફેલિંગ દ્રાવણમાં કોપર (II) આયનો શર્કરા દ્વારા રિડક્શન પામે છે. શર્કરા (અલ્ડિહાઇડ કે કિટોન સમૂહ ધરાવતી) કોપર (II) ને કોપર (I) માં રિડ્યુસ કરે છે, જે લાલ ક્યુપ્રસ ઓક્સાઇડ (Cu₂O) તરીકે અવક્ષેપિત થાય છે.

  • રિડ્યુસિંગ શર્કરા (ગ્લુકોઝ, લેક્ટોઝ) પ્રક્રિયા કરે છે.
  • Cu²⁺ → Cu⁺ (રિડક્શન) + Cu₂O (લાલ અવક્ષેપ).
  • નન-રિડ્યુસિંગ શર્કરા (સુક્રોઝ) પ્રક્રિયા કરતી નથી.
  • પ્રક્રિયા માટે ગરમી જરૂરી છે.
🧠 Trick: રિડ્યુસ = લાલ રંગ (Cu₂O)

પરીક્ષણની પદ્ધતિ

નમૂનાને ફેલિંગ દ્રાવણ સાથે મિશ્ર કરો અને ગરમ કરો. લાલ અવક્ષેપની હાજરી રિડ્યુસિંગ શર્કરાની હાજરી સૂચવે છે. જો કોઈ અવક્ષેપ ન મળે તો નમૂનામાં રિડ્યુસિંગ શર્કરા નથી.

  • નમૂના + ફેલિંગ દ્રાવણ → ગરમ કરો.
  • લાલ અવક્ષેપ = સક્રિય પરિણામ.
  • નીલો રંગ રહે = નકારાત્મક પરિણામ.
  • સુક્રોઝ માટે પહેલા હાઇડ્રોલિસિસ જરૂરી છે.
🧠 Trick: ગરમ કરો, લાલ જુઓ = શર્કરા છે

અનુપ્રયોગો અને મર્યાદાઓ

ફેલિંગ પરીક્ષણ ખાંડ, મધ, ફળોના રસ જેવા નમૂનાઓમાં શર્કરાની હાજરી શોધવા માટે ઉપયોગી છે. પરંતુ તે નન-રિડ્યુસિંગ શર્કરા (સુક્રોઝ) માટે કામ કરતું નથી. તેના માટે પહેલા એસિડ હાઇડ્રોલિસિસ કરવું પડે છે.

  • રિડ્યુસિંગ શર્કરા માટે સંવેદનશીલ.
  • સુક્રોઝ માટે નકારાત્મક.
  • હાઇડ્રોલિસિસ પછી સુક્રોઝ પણ ડિટેક્ટ થાય.
  • કેટલીક અશુદ્ધિઓ (એસિડ, ક્લોરાઇડ) પરિણામને અસર કરે છે.
🧠 Trick: સુક્રોઝ = સુધારો કરો (હાઇડ્રોલિસિસ)

NEET માટે મહત્વપૂર્ણ બાબતો

ફેલિંગ પરીક્ષણ NEETમાં કાર્બોહાઇડ્રેટ્સના અભ્યાસમાં મહત્વપૂર્ણ છે. ખાસ કરીને શર્કરાના પ્રકારો અને તેમની રાસાયણિક પ્રક્રિયાઓ સમજવા માટે. પરીક્ષામાં પ્રાયોગિક પાસાં અને સૈદ્ધાંતિક પાસાં બંને પૂછી શકાય છે.

  • રિડ્યુસિંગ vs નન-રિડ્યુસિંગ શર્કરા વચ્ચેનો તફાવત.
  • Cu₂O અવક્ષેપની ઓળખ.
  • ફેલિંગ દ્રાવણની રચના અને ભૂમિકા.
  • પરીક્ષણની મર્યાદાઓ અને ઉપયોગિતા.
🧠 Trick: NEET = નીરખો એનઝાઇમ અને ટેસ્ટ (પરીક્ષણ)

⚡ Quick Revision

  • ફેલિંગ પરીક્ષણ શર્કરા શોધવા માટે છે.
  • ફેલિંગ દ્રાવણ = CuSO₄ + રોકેફેલર ક્ષાર + NaOH.
  • રિડ્યુસિંગ શર્કરા (ગ્લુકોઝ) સાથે લાલ અવક્ષેપ મળે.
  • નન-રિડ્યુસિંગ શર્કરા (સુક્રોઝ) માટે નકારાત્મક.
  • ગરમી વિના પ્રક્રિયા થતી નથી.
  • Cu²⁺ → Cu⁺ (રિડક્શન) + Cu₂O (લાલ).
  • સુક્રોઝ માટે પહેલા હાઇડ્રોલિસિસ જરૂરી.
  • નમૂના + ફેલિંગ → ગરમ કરો → લાલ રંગ = સક્રિય.
  • અશુદ્ધિઓ પરિણામને અસર કરે છે.
  • NEETમાં કાર્બોહાઇડ્રેટ્સના અભ્યાસમાં મહત્વપૂર્ણ.
🎯 Exam Tip: પરીક્ષામાં ફેલિંગ પરીક્ષણના પ્રશ્નો માટે રિડ્યુસિંગ અને નન-રિડ્યુસિંગ શર્કરાનો તફાવત અને Cu₂O અવક્ષેપની ઓળખ યાદ રાખો. પ્રાયોગિક પાસાં સાથે સૈદ્ધાંતિક પાસાં પણ તૈયાર કરો.
