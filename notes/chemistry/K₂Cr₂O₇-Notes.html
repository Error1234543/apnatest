Pro Notes

K₂Cr₂O₇

📖 K₂Cr₂O₇ (પોટેશિયમ ડાયક્રોમેટ)

રાસાયણિક સૂત્ર અને બંધારણ

K₂Cr₂O₇ એ પોટેશિયમ ડાયક્રોમેટ છે, જેમાં ક્રોમિયમની ઓક્સીડેશન અવસ્થા +6 છે. તેના બંધારણમાં બે CrO₄ ટેટ્રાહેડ્રા શેર કરેલા ઓક્સિજન પરમાણુ દ્વારા જોડાયેલા હોય છે. આયનિક યૌગિક છે અને પાણીમાં દ્રાવ્ય છે.

  • K₂Cr₂O₇ નું મોલેક્યુલર વજન = 294.18 g/mol
  • ક્રોમિયમની ઓક્સીડેશન અવસ્થા = +6
  • રંગ: નારંગી-લાલ (Orange-red)
  • પાણીમાં દ્રાવ્ય, આલ્કોહોલમાં અદ્રાવ્ય
  • સ્ફટિક સ્વરૂપ: ત્રિકોણીય (Trigonal)
🧠 Trick: ક્રોમિયમ +6, પોટેશિયમ 2, ઓક્સિજન 7 – ‘કપો’ (K₂Cr₂O₇)

ભૌતિક ગુણધર્મો

K₂Cr₂O₇ એ સ્ફટિકમય ઘન પદાર્થ છે જેનો ગલનબિંદુ 398°C અને ઉત્કલનબિંદુ 500°C જેટલો છે. તે પાણીમાં દ્રાવ્ય છે અને દ્રાવણ નારંગી રંગનું હોય છે. તે પ્રબળ ઓક્સિડાઈઝિંગ એજન્ટ છે.

  • ગલનબિંદુ: 398°C
  • ઉત્કલનબિંદુ: 500°C
  • દ્રાવ્યતા: પાણીમાં ઉચ્ચ (10% દ્રાવણ સ્થિર)
  • રંગ: નારંગી-લાલ
  • સ્વાદ: કડવો અને લવણીય

રાસાયણિક ગુણધર્મો

K₂Cr₂O₇ એ પ્રબળ ઓક્સિડાઈઝિંગ એજન્ટ છે. એસિડિક મધ્યમમાં તે Cr³⁺માં રિડક્શન પામે છે. તે આલ્કોહોલ, SO₂, H₂S, અને I⁻ જેવા રિડ્યુસિંગ એજન્ટ્સ સાથે પ્રક્રિયા કરે છે. તે એસિડ સાથે પ્રક્રિયા કરીને ક્રોમિક એસિડ (H₂CrO₄) બનાવે છે.

  • એસિડિક મધ્યમમાં: Cr₂O₇²⁻ + 14H⁺ + 6e⁻ → 2Cr³⁺ + 7H₂O
  • પ્રબળ ઓક્સિડાઈઝિંગ એજન્ટ (E° = +1.33 V)
  • આલ્કોહોલ સાથે: K₂Cr₂O₇ + 3C₂H₅OH + 4H₂SO₄ → Cr₂(SO₄)₃ + 3CH₃COOH + K₂SO₄ + 7H₂O
  • SO₂ સાથે: K₂Cr₂O₇ + 3SO₂ + H₂SO₄ → Cr₂(SO₄)₃ + K₂SO₄ + H₂O
  • H₂S સાથે: K₂Cr₂O₇ + 3H₂S + 4H₂SO₄ → Cr₂(SO₄)₃ + K₂SO₄ + 7H₂O + 3S
🧠 Trick: ક્રોમેટ ઓક્સિડાઈઝ કરે, રિડ્યુસ કરે Cr³⁺

તૈયારી (પ્રયોગશાળા પદ્ધતિ)

K₂Cr₂O₇ ની તૈયારી ક્રોમાઇટ અયસ્ક (FeCr₂O₄) માંથી થાય છે. અયસ્કને સોડિયમ કાર્બોનેટ અને ચૂનાના પાઉડર સાથે ગરમ કરીને સોડિયમ ક્રોમેટ (Na₂CrO₄) મેળવવામાં આવે છે, જેને પછી પોટેશિયમ ક્લોરાઇડ સાથે પ્રક્રિયા કરવામાં આવે છે.

  • 4FeCr₂O₄ + 8Na₂CO₃ + 7O₂ → 8Na₂CrO₄ + 2Fe₂O₃ + 8CO₂
  • Na₂CrO₄ + 2KCl → K₂CrO₄ + 2NaCl
  • 2K₂CrO₄ + H₂SO₄ → K₂Cr₂O₇ + K₂SO₄ + H₂O
🧠 Trick: સોડા + ક્રોમાઇટ → ક્રોમેટ → ડાયક્રોમેટ

ઉપયોગો

K₂Cr₂O₇ નો ઉપયોગ લેદર ટેનિંગ, પેઈન્ટ્સ અને વર્ણકો, કાચ ઉદ્યોગ, અને રાસાયણિક વિશ્લેષણમાં થાય છે. તે ક્રોમિયમ પ્લેટિંગમાં અને કોઈપણ ઓર્ગેનિક યૌગિકોના શુદ્ધિકરણમાં પણ વપરાય છે.

  • લેદર ટેનિંગ (ક્રોમ ટેનિંગ)
  • પેઈન્ટ્સ અને વર્ણકો (ક્રોમ યેલો, ક્રોમ રેડ)
  • કાચ ઉદ્યોગ (હરિયો રંગ)
  • રાસાયણિક વિશ્લેષણ (ઓક્સિડાઈઝિંગ એજન્ટ)
  • ક્રોમિયમ પ્લેટિંગ
🧠 Trick: ક્રોમેટ = કલર, લેદર, કેમિકલ

સાવધાનીઓ અને પર્યાવરણીય અસરો

K₂Cr₂O₇ એ વિષાક્ત અને કર્કરોગજનક છે. તે ચામડી અને શ્વસન તંત્રને નુકસાન પહોંચાડે છે. તે પાણી અને મૃદામાં પ્રદૂષણ પહોંચાડે છે, જેથી તેનો નિકાલ સાવધાનીપૂર્વક કરવો જોઈએ.

  • વિષાક્ત (LD₅₀ = 190 mg/kg)
  • કર્કરોગજનક (IARC ગ્રુપ 1)
  • ચામડી અને આંખોને જળવે છે
  • પર્યાવરણીય પ્રદૂષક (ભારે ધાતુ)
  • નિકાલ: એસિડ અથવા આલ્કલી સાથે મિશ્રિત ન કરવો
🧠 Trick: ક્રોમેટ = ખતરનાક, કેંસર, કચરો સાવધ

NEET માટે મહત્વના પ્રશ્નો

NEETમાં K₂Cr₂O₇ પર પ્રશ્નો ઓક્સિડેશન-રિડક્શન પ્રક્રિયાઓ, દ્રાવ્યતા, રંગ, અને ઉપયોગો પર આધારિત હોય છે. એસિડિક મધ્યમમાં તેનું વર્તન અને ક્રોમિયમની ઓક્સીડેશન અવસ્થા પર ખાસ ધ્યાન આપવું.

  • K₂Cr₂O₇ એસિડિક મધ્યમમાં ઓક્સિડાઈઝિંગ એજન્ટ તરીકે કેવી રીતે કામ કરે છે?
  • K₂Cr₂O₇ ની તૈયારીની પ્રક્રિયા
  • K₂Cr₂O₇ ના ઉપયોગો અને સાવધાનીઓ
  • Cr₂O₇²⁻ આયનનું રંગ અને આકાર
  • K₂Cr₂O₇ ની દ્રાવ્યતા અને ભૌતિક ગુણધર્મો
🧠 Trick: NEET = ઓક્સિડેશન, ક્રોમેટ, ઉપયોગ
