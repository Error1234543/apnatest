આણ્વિકતા

📖 આણ્વિકતા (Mole Concept)

આણ્વિકતા ની વ્યાખ્યા

આણ્વિકતા એટલે પદાર્થના એક મોલમાં રહેલા કણો (પરમાણુ, અણુ, આયન)ની સંખ્યા. 1 મોલ = 6.022 × 10²³ કણો (અવોગેડ્રો સંખ્યા). NEETમાં આ મૂળભૂત સંકલ્પ છે જેમાં દ્રવ્યમાન, કણો અને મોલ વચ્ચેનો સંબંધ સમજાવવામાં આવે છે.

  • 1 મોલ = અવોગેડ્રો સંખ્યા (N₀) = 6.022 × 10²³ કણો
  • 1 મોલ ગેસ STP પર 22.4 L જગ્યા ઘેરે છે
  • મોલ = દ્રવ્યમાન (g) / આણ્વિક દ્રવ્યમાન (g/mol)
🧠 Trick: મોલ = મોટી સંખ્યા (6.022 × 10²³) ઈશારો: 'મો' (મોલ) + 'લ' (લાખો કણો)

અવોગેડ્રો નિયમ

સમાન તાપમાન અને દબાણે, સમાન કદના કોઈપણ ગેસમાં કણોની સંખ્યા સમાન હોય છે. આ નિયમ ગેસોના આણ્વિક દ્રવ્યમાન નક્કી કરવા ઉપયોગી છે. NEETમાં આ નિયમ વાયુ અવસ્થા સંકર મિશ્રણો માટે મહત્વનો છે.

  • સમાન STP પર, સમાન કદ = સમાન કણો
  • V ∝ n (કદ ∝ મોલની સંખ્યા)
  • ગેસના આણ્વિક દ્રવ્યમાન = 2 × વાયુ ઘનતા (g/L) STP પર
🧠 Trick: સમાન કદ, સમાન કણો – 'સમ' (સમાન) યાદ રાખો

આણ્વિક દ્રવ્યમાન અને મોલર દ્રવ્યમાન

આણ્વિક દ્રવ્યમાન એ અણુનું દ્રવ્યમાન છે જ્યારે મોલર દ્રવ્યમાન એ 1 મોલ પદાર્થનું દ્રવ્યમાન છે. NEETમાં આ બંને સંકલ્પો વચ્ચેનો તફાવત સમજવો જરૂરી છે. મોલર દ્રવ્યમાન = આણ્વિક દ્રવ્યમાન (g/mol).

  • આણ્વિક દ્રવ્યમાન = અણુનું દ્રવ્યમાન (a.m.u.)
  • મોલર દ્રવ્યમાન = આણ્વિક દ્રવ્યમાન (g/mol)
  • H₂O નું મોલર દ્રવ્યમાન = 18 g/mol
🧠 Trick: મોલર = મોટું (g/mol), આણ્વિક = અણુનું (a.m.u.)

સ્ટોઇકીયોમેટ્રી (Stoichiometry)

રાસાયણિક સમીકરણોમાં પદાર્થોના પ્રમાણનો અભ્યાસ. NEETમાં પ્રતિક્રિયાઓમાં વપરાતા અને ઉત્પન્ન થતા પદાર્થોના મોલનો ગણિતી હિસાબ કરવો પડે છે. આ માટે સમતોલિત સમીકરણ જરૂરી છે.

  • સમતોલિત સમીકરણમાં કોફીસન્ટ = મોલનું પ્રમાણ
  • પ્રતિક્રિયકો અને ઉત્પાદનોના મોલ સમાન રહે
  • ગણિત: મોલ = દ્રવ્યમાન / મોલર દ્રવ્યમાન
🧠 Trick: સમીકરણ સમતોલી, મોલ ગણી

દ્રાવણમાં આણ્વિકતા

દ્રાવણની સાંદ્રતા મોલરિટી (M) અથવા મોલલિટી (m) વડે વ્યક્ત કરવામાં આવે છે. NEETમાં દ્રાવણની સાંદ્રતા અને પ્રતિક્રિયા દર વચ્ચેનો સંબંધ મહત્વનો છે.

  • મોલરિટી (M) = મોલ / દ્રાવણનો લીટર (L)
  • મોલલિટી (m) = મોલ / દ્રાવકનો કિલો (kg)
  • N₁V₁ = N₂V₂ (તનુકરણ નિયમ)
🧠 Trick: M = મોલ/લીટર, m = મોલ/કિલો

ગેસો માટે આણ્વિકતા

ગેસોના આણ્વિક દ્રવ્યમાનની ગણતરી અવોગેડ્રો નિયમ અને આદર્શ ગેસ સમીકરણ (PV = nRT) વડે કરવામાં આવે છે. NEETમાં ગેસોના મિશ્રણ અને વાયુ ઘનતાની ગણતરી મહત્વની છે.

  • PV = nRT (R = 0.0821 L·atm·K⁻¹·mol⁻¹)
  • ગેસની ઘનતા (d) = PM / RT
  • સરેરાશ આણ્વિક દ્રવ્યમાન = (n₁M₁ + n₂M₂) / (n₁ + n₂)
🧠 Trick: PV = nRT → 'પીવી એન આર ટી' યાદ રાખો

ઉદાહરણો અને એપ્લિકેશન

NEETમાં આણ્વિકતાનો ઉપયોગ રાસાયણિક પ્રતિક્રિયાઓ, ગેસના નિયમો, અને દ્રાવણ સાંદ્રતા જેવા વિષયોમાં થાય છે. ઉદાહરણ: 2g H₂માં કેટલા મોલ? (H₂ નું મોલર દ્રવ્યમાન = 2g/mol → 1 મોલ).

  • 2g H₂ = 1 મોલ H₂
  • 11.2 L CO₂ STP પર = 0.5 મોલ CO₂
  • 36g H₂O = 2 મોલ H₂O

⚡ Quick Revision

  • 1 મોલ = 6.022 × 10²³ કણો (અવોગેડ્રો સંખ્યા)
  • 1 મોલ ગેસ STP પર 22.4 L જગ્યા ઘેરે
  • મોલર દ્રવ્યમાન = આણ્વિક દ્રવ્યમાન (g/mol)
  • મોલરિટી (M) = મોલ / લીટર
  • મોલલિટી (m) = મોલ / કિલો
  • PV = nRT (આદર્શ ગેસ સમીકરણ)
  • સમાન STP પર, સમાન કદ = સમાન કણો
  • ગેસની ઘનતા = PM / RT
  • સમતોલિત સમીકરણમાં કોફીસન્ટ = મોલનું પ્રમાણ
  • દ્રવ્યમાન (g) = મોલ × મોલર દ્રવ્યમાન
🎯 Exam Tip: NEETમાં આણ્વિકતા પરના પ્રશ્નો માટે હંમેશા સમતોલિત સમીકરણ લખો અને મોલ, દ્રવ્યમાન, અને કણો વચ્ચેના સંબંધને સ્પષ્ટ રાખો. ગણતરી કરતી વખતે એકમો (g, L, mol) પર ધ્યાન આપો.
