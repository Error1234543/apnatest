ઓક્સિડેશન અવસ્થા

📖 ઓક્સિડેશન અવસ્થા

પરિભાષા અને મૂળભૂત સંકલ્પના

ઓક્સિડેશન અવસ્થા એ ટ્રૅકમાં રહેલા પરમાણુ પરનો કાલ્પનિક વીજભાર છે, જ્યારે આયનિક યોગિકોમાં તે આયન પરનો વાસ્તવિક વીજભાર હોય છે. તે પરમાણુની ઇલેક્ટ્રોન ગુમાવવાની અથવા મેળવવાની ક્ષમતા દર્શાવે છે.

  • ઓક્સિડેશન અવસ્થા = પરમાણુ પરનો કાલ્પનિક વીજભાર
  • મૂળતત્વોમાં ઓક્સિડેશન અવસ્થા શૂન્ય હોય છે
  • સંકીર્ણ આયનોમાં વીજભારની ગણતરી કરવા માટે ઉપયોગી
  • ઇલેક્ટ્રોન ગુમાવવા = ઓક્સિડેશન અવસ્થા વધે, મેળવવા = ઓક્સિડેશન અવસ્થા ઘટે
🧠 Trick: OIL RIG: Oxidation Is Loss, Reduction Is Gain

નિયમો (Rules for Assigning Oxidation States)

મૂળતત્વની ઓક્સિડેશન અવસ્થા શૂન્ય, એક-પરમાણુક આયનની ઓક્સિડેશન અવસ્થા તેના વીજભાર જેટલી હોય છે. હાઇડ્રોજનની ઓક્સિડેશન અવસ્થા +1 (મેટલ હાઇડ્રાઇડમાં -1), ઓક્સિજનની -2 (પરોક્સાઇડમાં -1, ફ્લોરિન સાથે +2) હોય છે.

  • મૂળતત્વ: 0 (ઉદા. O₂, N₂, Na)
  • એક-પરમાણુક આયન: વીજભાર = ઓક્સિડેશન અવસ્થા (ઉદા. Na⁺ = +1, Cl⁻ = -1)
  • H = +1 (મેટલ હાઇડ્રાઇડમાં -1, ઉદા. NaH)
  • O = -2 (પરોક્સાઇડમાં -1, ઉદા. H₂O₂; OF₂માં +2)
  • હેલોજન: સામાન્ય રીતે -1 (ફ્લોરિન હંમેશા -1)
  • યોગિકમાં તટસ્થ પરમાણુનો સરવાળો શૂન્ય હોય છે
🧠 Trick: H is +1, O is -2, F is -1, Metals are positive, Non-metals can vary

ઓક્સિડેશન અવસ્થાની ગણતરી

યોગિકમાં ઓક્સિડેશન અવસ્થાની ગણતરી માટે તટસ્થતા અથવા વીજભારના સિદ્ધાંતનો ઉપયોગ કરો. ઉદા. H₂SO₄માં Sની ઓક્સિડેશન અવસ્થા: 2*(+1) + x + 4*(-2) = 0 → x = +6.

  • તટસ્થ યોગિકમાં ઓક્સિડેશન અવસ્થાનો સરવાળ = 0
  • બહુ-પરમાણુક આયનમાં સરવાળ = આયનનો વીજભાર
  • ઉદા. KMnO₄માં Mn = +7, Fe₃O₄માં Fe = +8/3
  • ગણતરી માટે અજ્ઞાત ઓક્સિડેશન અવસ્થા ધરાવતા પરમાણુને x ગણો

ઓક્સિડેશન અવસ્થામાં ફેરફાર (Redox Reactions)

જ્યારે ઓક્સિડેશન અવસ્થા વધે ત્યારે ઓક્સિડેશન થાય છે (ઇલેક્ટ્રોન ગુમાવવા), જ્યારે ઓક્સિડેશન અવસ્થા ઘટે ત્યારે રિડક્શન થાય છે (ઇલેક્ટ્રોન મેળવવા). રેડોક્સ પ્રક્રિયામાં બંને એકસાથે થાય છે.

  • ઓક્સિડેશન: ઓક્સિડેશન અવસ્થા વધે (ઇલેક્ટ્રોન ગુમાવવા)
  • રિડક્શન: ઓક્સિડેશન અવસ્થા ઘટે (ઇલેક્ટ્રોન મેળવવા)
  • રેડોક્સ પ્રક્રિયા = ઓક્સિડેશન + રિડક્શન
  • ઉદા. Zn + Cu²⁺ → Zn²⁺ + Cu (Zn ઓક્સિડાઇઝ થાય, Cu²⁺ રિડ્યુસ થાય)
🧠 Trick: LEO the lion says GER: Lose Electrons Oxidation, Gain Electrons Reduction

અપવાદ અને વિશેષ કેસ

કેટલાક કિસ્સાઓમાં ઓક્સિડેશન અવસ્થા સામાન્ય નિયમોને અનુસરતી નથી. ઉદા. હાઇડ્રોજન પરોક્સાઇડ (H₂O₂)માં Oની ઓક્સિડેશન અવસ્થા -1 હોય છે, જ્યારે સુપરઑક્સાઇડ (KO₂)માં Oની ઓક્સિડેશન અવસ્થા -1/2 હોય છે.

  • H₂O₂માં O = -1
  • KO₂ (પોટેશિયમ સુપરઑક્સાઇડ)માં O = -1/2
  • OF₂માં O = +2 (ફ્લોરિનની ઇલેક્ટ્રોનેગેટિવિટી વધુ હોવાથી)
  • મેટલ હાઇડ્રાઇડમાં H = -1 (ઉદા. NaH, CaH₂)

જૈવિક મહત્વ (Biological Significance)

જૈવરાસાયણિક પ્રક્રિયાઓમાં ઓક્સિડેશન અવસ્થા મહત્વની ભૂમિકા ભજવે છે. ઉદા. શ્વસનમાં ગ્લુકોઝનું ઓક્સિડેશન થાય છે, જ્યારે પ્રકાશસંશ્લેષણમાં CO₂નું રિડક્શન થાય છે.

  • શ્વસન: ગ્લુકોઝ (C₆H₁₂O₆)નું ઓક્સિડેશન → CO₂ + H₂O + ઉર્જા
  • પ્રકાશસંશ્લેષણ: CO₂નું રિડક્શન → કાર્બોદિત (ગ્લુકોઝ)
  • ઉત્સેચકો (એન્ઝાઇમ્સ) રેડોક્સ પ્રક્રિયાઓને ઉત્તેજિત કરે છે
  • NAD⁺/NADH, FAD/FADH₂ જેવા કોફેક્ટર્સ ઓક્સિડેશન અવસ્થામાં ફેરફાર કરે છે

⚡ Quick Revision

  • ઓક્સિડેશન અવસ્થા = પરમાણુ પરનો કાલ્પનિક વીજભાર
  • મૂળતત્વની ઓક્સિડેશન અવસ્થા શૂન્ય હોય છે
  • H = +1 (મેટલ હાઇડ્રાઇડમાં -1), O = -2 (પરોક્સાઇડમાં -1)
  • તટસ્થ યોગિકમાં ઓક્સિડેશન અવસ્થાનો સરવાળ શૂન્ય હોય છે
  • ઓક્સિડેશન = ઇલેક્ટ્રોન ગુમાવવા, રિડક્શન = ઇલેક્ટ્રોન મેળવવા
  • રેડોક્સ પ્રક્રિયા = ઓક્સિડેશન + રિડક્શન
  • H₂O₂માં O = -1, KO₂માં O = -1/2
  • શ્વસનમાં ગ્લુકોઝનું ઓક્સિડેશન, પ્રકાશસંશ્લેષણમાં CO₂નું રિડક્શન
  • NAD⁺/NADH રેડોક્સ પ્રક્રિયાઓમાં મહત્વની ભૂમિકા ભજવે છે
🎯 Exam Tip: NEETમાં ઓક્સિડેશન અવસ્થા અને રેડોક્સ પ્રક્રિયાઓ પર આધારિત પ્રશ્નો વારંવાર પૂછાય છે. ગણતરી માટે નિયમો યાદ રાખો અને અપવાદો (H₂O₂, KO₂) પર વિશેષ ધ્યાન આપો.
