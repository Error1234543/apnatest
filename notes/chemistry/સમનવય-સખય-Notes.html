સમન્વય સંખ્યા

📖 સમન્વય સંખ્યા

પરિભાષા અને મૂળભૂત સંકલ્પ

સમન્વય સંખ્યા એટલે કોષમાં રહેલા વર્ણસૂત્રોના સેટની સંખ્યા. હેપ્લોઈડ (n) કોષમાં એક સેટ અને ડિપ્લોઈડ (2n) કોષમાં બે સેટ વર્ણસૂત્રો હોય છે. મનુષ્યમાં 2n = 46, જ્યારે મધમાખીમાં n = 16 હોય છે.

  • સમન્વય સંખ્યા = વર્ણસૂત્રોના સેટની સંખ્યા
  • હેપ્લોઈડ (n) = એક સેટ, ડિપ્લોઈડ (2n) = બે સેટ
  • મનુષ્ય: 2n = 46, n = 23
  • મધમાખી: હેપ્લોઈડ નર, ડિપ્લોઈડ માદા
🧠 Trick: 2n = મનુષ્ય, n = મધમાખી નર

હેપ્લોઈડ vs ડિપ્લોઈડ કોષ

હેપ્લોઈડ કોષમાં એક જ સેટ વર્ણસૂત્રો હોય છે (જેમકે ગેમેટ્સ: શુક્રાણુ, અંડકોષ). ડિપ્લોઈડ કોષમાં બે સેટ હોય છે (જેમકે સોમેટિક કોષ). હેપ્લોઈડ કોષમાં જનીનની પ્રગટતા સીધી જોવા મળે છે.

  • હેપ્લોઈડ: ગેમેટ્સ (n), ડિપ્લોઈડ: સોમેટિક કોષ (2n)
  • હેપ્લોઈડમાં જનીનની અસર સીધી જોવા મળે
  • ડિપ્લોઈડમાં હોમોલોગસ વર્ણસૂત્રો હોય છે
  • મિયોસિસથી હેપ્લોઈડ કોષ બને છે
🧠 Trick: n = ગેમેટ, 2n = સોમા

સમન્વય સંખ્યાનો જીવવિજ્ઞાનમાં ઉપયોગ

સમન્વય સંખ્યા અનુવંશિકતાના અભ્યાસમાં મહત્વની ભૂમિકા ભજવે છે. તે જનીનના વિતરણ અને પ્રજનન પ્રક્રિયામાં મદદ કરે છે. ઉદાહરણ તરીકે, મિયોસિસ દરમિયાન વર્ણસૂત્રોની સંખ્યા અડધી થાય છે, જેથી સંતતિમાં સ્થિરતા રહે.

  • મિયોસિસમાં વર્ણસૂત્રોની સંખ્યા અડધી થાય છે
  • ફર્ટિલાઇઝેશન પછી ડિપ્લોઈડ સ્થિતિ પુનઃસ્થાપિત થાય છે
  • સમન્વય સંખ્યા અનુવંશિક લક્ષણોની આગાહી કરવામાં મદદ કરે છે
  • પોલિપ્લોઈડી (3n, 4n) કેટલાક વનસ્પતિઓમાં જોવા મળે છે
🧠 Trick: મિયોસિસ = n, ફર્ટિલાઇઝેશન = 2n

પોલિપ્લોઈડી અને તેનું મહત્વ

પોલિપ્લોઈડી એટલે ત્રણ અથવા વધુ વર્ણસૂત્ર સેટ ધરાવતા કોષ. તે વનસ્પતિઓમાં સામાન્ય છે (જેમકે બનાના: 3n, વીટ: 6n). પોલિપ્લોઈડ વનસ્પતિઓ વધુ મજબૂત અને ઉત્પાદક હોય છે.

  • પોલિપ્લોઈડી: 3n, 4n, 6n વગેરે
  • વનસ્પતિઓમાં પોલિપ્લોઈડી સામાન્ય છે
  • બનાના: 3n, વીટ: 6n
  • પોલિપ્લોઈડ વનસ્પતિઓ વધુ ઉત્પાદક હોય છે
🧠 Trick: 3n = બનાના, 6n = વીટ

સમન્વય સંખ્યા અને જનીન પ્રગટતા

ડિપ્લોઈડ કોષમાં, હોમોલોગસ વર્ણસૂત્રો પર સમાન જનીન હોઈ શકે અથવા અલગ અલગ (એલિલ્સ). હેપ્લોઈડ કોષમાં, જનીનની પ્રગટતા સીધી જોવા મળે છે કારણ કે એક જ એલિલ હોય છે.

  • ડિપ્લોઈડ: હોમોલોગસ વર્ણસૂત્રો પર એલિલ્સ
  • હેપ્લોઈડ: એક જ એલિલ, સીધી પ્રગટતા
  • મેન્ડેલના નિયમો ડિપ્લોઈડ કોષ પર લાગુ પડે છે
  • હેપ્લોઈડ કોષમાં રિસેસિવ લક્ષણો પ્રગટ થાય છે
🧠 Trick: એક એલિલ = હેપ્લોઈડ, બે એલિલ = ડિપ્લોઈડ

સમન્વય સંખ્યા અને ઉત્ક્રાંતિ

સમન્વય સંખ્યામાં ફેરફાર ઉત્ક્રાંતિના મહત્વના પગલાંમાંના એક છે. પોલિપ્લોઈડી કેટલીક વખત નવી જાતિઓના ઉદભવમાં મદદ કરે છે. ઉદાહરણ તરીકે, કેટલીક વનસ્પતિઓમાં પોલિપ્લોઈડીને લીધે નવી લક્ષણો પ્રગટ થાય છે.

  • સમન્વય સંખ્યામાં ફેરફાર = ઉત્ક્રાંતિનું કારણ
  • પોલિપ્લોઈડી નવી જાતિઓના ઉદભવમાં મદદ કરે છે
  • વનસ્પતિઓમાં પોલિપ્લોઈડી સામાન્ય છે
  • નવી લક્ષણો પ્રગટ થાય છે

  • પોલિપ્લોઈડી: 3n, 4n, 6n (વનસ્પતિઓમાં સામાન્ય)
  • બનાના: 3n, વીટ: 6n

🎯 Exam Tip: NEETમાં સમન્વય સંખ્યાના પ્રશ્નો મિયોસિસ, ફર્ટિલાઇઝેશન અને પોલિપ્લોઈડી સાથે જોડાયેલા હોય છે. મનુષ્ય અને મધમાખીની સમન્વય સંખ્યા યાદ રાખો.
