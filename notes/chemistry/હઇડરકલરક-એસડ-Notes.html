હાઇડ્રોક્લોરિક એસિડ

📖 હાઇડ્રોક્લોરિક એસિડ (Hydrochloric Acid - HCl)

રચના અને ગુણધર્મો

HCl એ હાઇડ્રોજન અને ક્લોરિનના સંયોજનથી બનતો મજબૂત એસિડ છે. તે રંગહીન, તીવ્ર ગંધવાળો, પાણીમાં સરળતાથી દ્રાવ્ય અને પ્રબળ વિદ્યુતવિભાજક છે. જલીય દ્રાવણમાં તે H⁺ અને Cl⁻ આયનોમાં વિભાજિત થાય છે.

  • HCl એ મોનોપ્રોટિક એસિડ છે (એક H⁺ આયન આપે છે).
  • અત્યંત પ્રબળ એસિડ (pH ~ 0-1, 1M દ્રાવણમાં).
  • ઉચ્ચ ક્વથનબિંદુ (108.5°C) અને ગલનબિંદુ (-27.3°C).
  • પાણી સાથે મિશ્ર થતાં ઉષ્મા મુક્ત કરે છે (એક્ઝોથર્મિક પ્રક્રિયા).
🧠 Trick: HCl = H⁺ + Cl⁻ → 'હાઇડ્રો ક્લોર' વખતે 'પ્લસ' મનાય!

લેબોરેટરી તૈયારી

લેબમાં HCl ની તૈયારી NaCl અને સાંદ્ર H₂SO₄ ની પ્રક્રિયાથી થાય છે. NaCl + H₂SO₄ → NaHSO₄ + HCl↑. આ પ્રક્રિયા 200°C થી ઓછા તાપમાને થાય છે.

  • પ્રક્રિયા માટે સાંદ્ર H₂SO₄ વાપરવામાં આવે છે.
  • HCl વાયુ રૂપે મુક્ત થાય છે, જે પાણીમાં શોષી લેવામાં આવે છે.
  • NaHSO₄ ની સાંદ્ર H₂SO₄ સાથે વધુ પ્રક્રિયાથી વધુ HCl મળે છે.
  • આ પ્રક્રિયા 'વિસ્થાપન પ્રક્રિયા'નો ઉદાહરણ છે.

ઔદ્યોગિક ઉત્પાદન (હાર્ગ્રીવ્સ પ્રક્રિયા)

ઔદ્યોગિક રીતે HCl નું ઉત્પાદન હાર્ગ્રીવ્સ પ્રક્રિયાથી થાય છે, જેમાં H₂ અને Cl₂ ને 400-500°C તાપમાને અને ઉત્પ્રેરક (પ્લેટિનમ)ની હાજરીમાં પ્રક્રિયા કરવામાં આવે છે: H₂ + Cl₂ → 2HCl.

  • ઉત્પ્રેરક તરીકે પ્લેટિનમ અથવા એસ્બેસ્ટસ વાપરવામાં આવે છે.
  • પ્રક્રિયા ઉષ્માક્ષેપી (exothermic) છે.
  • HCl વાયુને પાણીમાં શોષીને સાંદ્ર દ્રાવણ મેળવવામાં આવે છે.
  • આ પ્રક્રિયા 'સંયોજન પ્રક્રિયા'નો ઉદાહરણ છે.
🧠 Trick: H₂ + Cl₂ → 2HCl → 'હાર્ગ્રીવ્સ હાઇડ્રો ક્લોર'!

રાસાયણિક ગુણધર્મો

HCl એ પ્રબળ એસિડ હોવાથી તે ધાતુઓ સાથે પ્રક્રિયા કરીને હાઇડ્રોજન વાયુ મુક્ત કરે છે. તે કાર્બોનેટ અને બાઇકાર્બોનેટ સાથે પ્રક્રિયા કરીને CO₂ મુક્ત કરે છે. તે નાઇટ્રેટ અને સલ્ફાઇડ સાથે પણ પ્રક્રિયા કરે છે.

  • Zn + 2HCl → ZnCl₂ + H₂↑ (ધાતુ સાથે પ્રક્રિયા).
  • Na₂CO₃ + 2HCl → 2NaCl + H₂O + CO₂↑ (કાર્બોનેટ સાથે).
  • NaHCO₃ + HCl → NaCl + H₂O + CO₂↑ (બાઇકાર્બોનેટ સાથે).
  • AgNO₃ + HCl → AgCl↓ (શ્વેત અવક્ષેપ) + HNO₃ (નાઇટ્રેટ સાથે).
🧠 Trick: HCl + ધાતુ → H₂↑, HCl + CO₃²⁻ → CO₂↑

બાયોલોજીકલ મહત્વ

માનવ શરીરમાં, પેટમાં હાઇડ્રોક્લોરિક એસિડનું સ્રાવ થાય છે, જે ખોરાકના પાચનમાં મહત્વની ભૂમિકા ભજવે છે. તે પેપ્સિન ઉત્સેચકને સક્રિય કરે છે અને ખોરાકમાં રહેલા બેક્ટેરિયાનું નાશ કરે છે.

  • પેટની પેશીઓ (parietal cells) દ્વારા સ્રાવિત થાય છે.
  • pH ~ 1.5-3.5 (અત્યંત એસિડિક વાતાવરણ).
  • પેપ્સિન જેવા ઉત્સેચકોને સક્રિય કરે છે.
  • ખોરાકમાં રહેલા સૂક્ષ્મજીવોનું નાશ કરે છે.
🧠 Trick: પેટમાં HCl → પાચન + પેપ્સિન + પેથોજેન નાશ!

ઉપયોગો

HCl ના વ્યાપક ઉપયોગો છે, જેમાં રાસાયણિક ઉદ્યોગ, ખાતર ઉત્પાદન, ધાતુ શુદ્ધિકરણ, ખોરાક ઉદ્યોગ અને પાણી શુદ્ધિકરણનો સમાવેશ થાય છે.

  • રાસાયણિક ઉદ્યોગમાં વિવિધ રાસાયણોના ઉત્પાદન માટે.
  • ખાતર ઉદ્યોગમાં ફોસ્ફેટ ખાતરના ઉત્પાદન માટે.
  • ધાતુ શુદ્ધિકરણમાં ઓક્સાઇડની સફાઈ માટે.
  • ખોરાક ઉદ્યોગમાં pH નિયંત્રણ અને સ્વાદ વર્ધક તરીકે.
  • પાણી શુદ્ધિકરણમાં ક્લોરિનેશન પ્રક્રિયામાં.

સાવધાની અને અસરો

HCl એ પ્રબળ એસિડ હોવાથી તે ચામડી, આંખો અને શ્વાસનળીને નુકસાન પહોંચાડી શકે છે. તેની સાથે કામ કરતી વખતે સુરક્ષા પગલાં લેવા જરૂરી છે.

  • ચામડી સાથે સંપર્કથી બળતરા અને ઘા થઈ શકે છે.
  • આંખો સાથે સંપર્કથી અંધત્વ થઈ શકે છે.
  • શ્વાસનળીમાં જવાથી શ્વાસમાં તકલીફ થઈ શકે છે.
  • પાણી સાથે પાતળું કરતી વખતે એસિડને પાણીમાં ઉમેરવું (એસિડ ટુ વોટર).
🧠 Trick: એસિડ ટુ વોટર, નહીં વોટર ટુ એસિડ!

⚡ Quick Revision

  • HCl એ H⁺ અને Cl⁻ આયનોમાં વિભાજિત થતો મજબૂત એસિડ છે.
  • લેબમાં NaCl + H₂SO₄ થી તૈયાર થાય છે.
  • ઔદ્યોગિક રીતે H₂ + Cl₂ → 2HCl (હાર્ગ્રીવ્સ પ્રક્રિયા).
  • ધાતુઓ સાથે પ્રક્રિયા કરીને H₂ વાયુ મુક્ત કરે છે.
  • પેટમાં પાચન અને પેપ્સિન સક્રિયતામાં મહત્વની ભૂમિકા ભજવે છે.
  • રાસાયણિક ઉદ્યોગ, ખાતર અને ધાતુ શુદ્ધિકરણમાં વ્યાપક ઉપયોગ.
  • પ્રબળ એસિડ હોવાથી સાવધાની રાખવી જરૂરી છે.
🎯 Exam Tip: NEET માં HCl ના રાસાયણિક ગુણધર્મો અને પેટમાં તેની ભૂમિકા પર પ્રશ્નો વારંવાર પૂછાય છે, તેથી આ ભાગોને ખાસ ધ્યાનમાં રાખો.
