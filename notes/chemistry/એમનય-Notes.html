Pro Notes

એમોનિયા

📖 એમોનિયા (Ammonia - NH₃)

રચના અને ગુણધર્મો

એમોનિયા એક અકાર્બનિક યૌગિક છે જે નાઇટ્રોજન અને હાઇડ્રોજનના સંયોજનથી બને છે (NH₃). તે રંગહીન, તીક્ષ્ણ ગંધવાળો વાયુ છે જે પાણીમાં અત્યંત દ્રાવ્ય છે. તે ક્ષારીય સ્વભાવ ધરાવે છે અને લૂઈસ બેઇઝ તરીકે વર્તે છે.

  • રાસાયણિક સૂત્ર: NH₃
  • આકાર: પિરામિડલ (sp³ સંકરણ)
  • ગંધ: તીક્ષ્ણ, પચી ગયેલા અંડા જેવી
  • દ્રાવ્યતા: પાણીમાં અત્યંત દ્રાવ્ય (1:700 ગુણોત્તર)
  • ક્ષારીય સ્વભાવ: લિટમસ પેપરને નીલો કરે છે
🧠 Trick: N-H₃ → Nai Hamesha 3 H (નાઇટ્રોજન હંમેશા 3 હાઇડ્રોજન સાથે)

તૈયારી પધ્ધતિઓ

હેબર પ્રક્રિયા (Habit Process) દ્વારા એમોનિયાનું ઉત્પાદન થાય છે: N₂ + 3H₂ → 2NH₃ (450°C, 200 atm, Fe ઉત્પ્રેરક). લેબમાં એમોનિયમ ક્લોરાઇડ અને કૅલ્શિયમ હાઇડ્રોક્સાઇડના મિશ્રણને ગરમ કરીને પણ મેળવી શકાય છે.

  • હેબર પ્રક્રિયા: N₂ + 3H₂ ⇌ 2NH₃ (ΔH = -92 kJ/mol)
  • ઉત્પ્રેરક: લોખંડ (Fe) + Al₂O₃ + K₂O
  • લેબ પધ્ધતિ: 2NH₄Cl + Ca(OH)₂ → 2NH₃ + CaCl₂ + 2H₂O
  • ઔદ્યોગિક મહત્વ: યુરિયા અને નાઇટ્રિક એસિડ બનાવવા માટે
🧠 Trick: હેબર = હવાઈ + બેર (N₂ + H₂ → NH₃)

રાસાયણિક પ્રતિક્રિયાઓ

એમોનિયા હાઇડ્રોક્લોરિક એસિડ સાથે પ્રતિક્રિયા કરીને એમોનિયમ ક્લોરાઇડ બનાવે છે. તે ઓક્સિજન સાથે સળગીને નાઇટ્રોજન અને પાણી બનાવે છે. તે ધાતુઓ સાથે પણ પ્રતિક્રિયા કરીને નાઇટ્રાઇડ બનાવે છે.

  • NH₃ + HCl → NH₄Cl (ધૂમાડો બને છે)
  • 4NH₃ + 3O₂ → 2N₂ + 6H₂O (સળગવું)
  • 2NH₃ + 3CuO → N₂ + 3Cu + 3H₂O (CuO ને કાળો કરે છે)
  • 8NH₃ + 3Cl₂ → N₂ + 6NH₄Cl (હળવા પીળા ધૂમાડા)
  • 6NH₃ + 3Cl₂ → N₂ + 6HCl (અતિરિક્ત Cl₂ સાથે)
🧠 Trick: એમોનિયા + HCl = ધૂમાડો (NH₄Cl)

ઉપયોગો

એમોનિયાનો ઉપયોગ ખાતર (યુરિયા, એમોનિયમ સલ્ફેટ), નાઇટ્રિક એસિડ, રેફ્રિજરેટરમાં શીતક, અને કપડા ઉદ્યોગમાં થાય છે. તે રાસાયણિક ઉદ્યોગનો મૂળભૂત કાચો માલ છે.

  • ખાતર: યુરિયા (NH₂CONH₂), એમોનિયમ સલ્ફેટ ((NH₄)₂SO₄)
  • નાઇટ્રિક એસિડ: ઓસ્ટવાલ્ડ પ્રક્રિયા
  • શીતક: રેફ્રિજરેશન સાયકલમાં
  • કપડા ઉદ્યોગ: સોડા એમોનિયા (NH₄OH)
  • રાસાયણિક ઉદ્યોગ: પોલિમર, પ્લાસ્ટિક, વિસ્ફોટક
🧠 Trick: એમોનિયા = ખાતર, એસિડ, શીતક, કપડા

જૈવિક મહત્વ

એમોનિયા નાઇટ્રોજન ચક્રનો મહત્વપૂર્ણ ભાગ છે. તે પ્રોટીન અને ન્યુક્લિક એસિડના સંશ્લેષણમાં મહત્વની ભૂમિકા ભજવે છે. મનુષ્યમાં, એમોનિયા યકૃતમાં યુરિયા ચક્ર દ્વારા યુરિયામાં રૂપાંતરિત થાય છે.

  • નાઇટ્રોજન ચક્ર: એમોનિફિકેશન (નાઇટ્રેટ → એમોનિયા)
  • યુરિયા ચક્ર: એમોનિયા → યુરિયા (યકૃતમાં)
  • પ્રોટીન સંશ્લેષણ: એમિનો એસિડમાં NH₂ સમૂહ
  • ન્યુક્લિક એસિડ: પ્યુરિન અને પિરિમિડિન બેઇઝમાં નાઇટ્રોજન
  • ઝેરી અસર: ઉચ્ચ સાંદ્રતામાં મગજ પર અસર કરે છે
🧠 Trick: એમોનિયા = નાઇટ્રોજન → પ્રોટીન → યુરિયા

પર્યાવરણીય અસર

એમોનિયા વાતાવરણમાં અમોનિયા વાયુ તરીકે મુક્ત થાય છે અને એસિડ વર્ષા તરફ દોરી જાય છે. તે જમીનના pHને પણ અસર કરે છે અને પાણીના પ્રદૂષણમાં યોગદાન આપે છે.

  • એસિડ વર્ષા: NH₃ + H₂O + O₂ → HNO₃
  • જમીનનું અલ્કલાઇનીકરણ: pH વધારે છે
  • પાણીનું પ્રદૂષણ: યૂટ્રોફિકેશન
  • વાતાવરણીય પ્રદૂષણ: સ્મોગ બનાવવામાં ભૂમિકા
  • ગ્રીનહાઉસ વાયુ: પરોક્ષ રીતે CO₂ ઉત્સર્જનમાં યોગદાન
🧠 Trick: એમોનિયા = એસિડ વર્ષા + જમીન pH + યૂટ્રોફિકેશન

⚡ Quick Revision

  • એમોનિયા = NH₃, પિરામિડલ આકાર
  • હેબર પ્રક્રિયા: N₂ + 3H₂ → 2NH₃ (Fe ઉત્પ્રેરક)
  • પાણીમાં અત્યંત દ્રાવ્ય, ક્ષારીય સ્વભાવ
  • NH₃ + HCl → NH₄Cl (ધૂમાડો)
  • યુરિયા ચક્ર દ્વારા એમોનિયા યુરિયામાં રૂપાંતરિત થાય છે
  • ખાતર, નાઇટ્રિક એસિડ, શીતક તરીકે ઉપયોગી
  • નાઇટ્રોજન ચક્રનો મહત્વપૂર્ણ ભાગ
  • અતિરિક્ત એમોનિયા મગજ માટે ઝેરી
  • એસિડ વર્ષા અને યૂટ્રોફિકેશનમાં યોગદાન
  • લિટમસ પેપરને નીલો કરે છે
🎯 Exam Tip: એમોનિયાના ગુણધર્મો અને પ્રતિક્રિયાઓ પર ધ્યાન કેન્દ્રિત કરો, ખાસ કરીને હેબર પ્રક્રિયા અને યુરિયા ચક્ર. રાસાયણિક સમીકરણો યાદ રાખો અને તેના ઉપયોગો સાથે જોડો.
