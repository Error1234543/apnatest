સ્ટાર્ચ

📖 સ્ટાર્ચ

સ્ટાર્ચની રચના અને સ્વરૂપ

સ્ટાર્ચ એક જટિલ કાર્બોદિત છે, જે ગ્લૂકોઝના બહુવિધ અણુઓના બંધારણથી બને છે. તે પોલિસેકરાઇડ છે અને વનસ્પતિમાં શક્તિના સંગ્રહના રૂપમાં જોવા મળે છે. સ્ટાર્ચ બે પ્રકારના હોય છે: એમિલોઝ (20-30%) અને એમિલોપેક્ટિન (70-80%).

  • સ્ટાર્ચ = એમિલોઝ + એમિલોપેક્ટિન
  • એમિલોઝ રેખીય અને પાણીમાં દ્રાવ્ય છે
  • એમિલોપેક્ટિન શાખાયુક્ત અને પાણીમાં અદ્રાવ્ય છે
  • ગ્લૂકોઝ અણુઓ α-1,4 અને α-1,6 ગ્લાયકોસિડિક બંધથી જોડાયેલા હોય છે
🧠 Trick: એમિલોઝ = એ (રેખીય), એમિલોપેક્ટિન = પેક્ટ (શાખાયુક્ત)

સ્ટાર્ચનું જૈવિક મહત્વ

સ્ટાર્ચ વનસ્પતિમાં પ્રકાશસંશ્લેષણ દ્વારા સંગ્રહ પામે છે અને પછી શ્વસન દ્વારા શક્તિ મુક્ત કરે છે. તે વનસ્પતિના બીજ, મૂળ, અને કંદમાં સંગ્રહિત થાય છે. મનુષ્ય માટે, સ્ટાર્ચ એક મહત્વપૂર્ણ કાર્બોદિત સ્ત્રોત છે જે શરીરને શક્તિ પૂરી પાડે છે.

  • પ્રકાશસંશ્લેષણમાં સ્ટાર્ચ સંગ્રહિત થાય છે
  • શ્વસન દ્વારા સ્ટાર્ચ ટૂટીને ગ્લૂકોઝ બને છે
  • બીજ, મૂળ, કંદમાં સંગ્રહિત થાય છે
  • મનુષ્યના આહારમાં શક્તિનો મુખ્ય સ્ત્રોત

સ્ટાર્ચનો રાસાયણિક સૂત્ર અને બંધારણ

સ્ટાર્ચનો રાસાયણિક સૂત્ર (C₆H₁₀O₅)ₙ છે, જ્યાં n હજારો સુધી હોઈ શકે છે. એમિલોઝમાં ગ્લૂકોઝ અણુઓ α-1,4 ગ્લાયકોસિડિક બંધથી જોડાયેલા હોય છે, જ્યારે એમિલોપેક્ટિનમાં α-1,4 અને α-1,6 બંધ હોય છે, જે શાખાઓ બનાવે છે.

  • રાસાયણિક સૂત્ર: (C₆H₁₀O₅)ₙ
  • એમિલોઝ: α-1,4 બંધ
  • એમિલોપેક્ટિન: α-1,4 અને α-1,6 બંધ
  • શાખાઓ એમિલોપેક્ટિનની વિશિષ્ટતા છે

સ્ટાર્ચનું પાચન

મનુષ્યમાં સ્ટાર્ચનું પાચન ઉત્સેચક એમિલેઝ દ્વારા થાય છે. લાળામાં રહેલું સેલિવરી એમિલેઝ સ્ટાર્ચને મોલ્ટોઝ અને ડેક્સ્ટ્રિનમાં તોડે છે. નાનાં આંતરડામાં પેન્ક્રિયેટિક એમિલેઝ સ્ટાર્ચને મોલ્ટોઝ અને મોલ્ટોટ્રાયોઝમાં તોડે છે, જે પછી મોલ્ટેઝ દ્વારા ગ્લૂકોઝમાં રૂપાંતરિત થાય છે.

  • લાળા એમિલેઝ: સ્ટાર્ચ → મોલ્ટોઝ + ડેક્સ્ટ્રિન
  • પેન્ક્રિયેટિક એમિલેઝ: સ્ટાર્ચ → મોલ્ટોઝ + મોલ્ટોટ્રાયોઝ
  • મોલ્ટેઝ: મોલ્ટોઝ → ગ્લૂકોઝ
  • પાચન ક્રમ: મુખ → નાનાં આંતરડું
🧠 Trick: લાળા એમિલેઝ = સ્ટાર્ચ તોડે, પેન્ક્રિયેટિક એમિલેઝ = પૂરું તોડે

સ્ટાર્ચનો ઉદ્યોગિક ઉપયોગ

સ્ટાર્ચનો ઉપયોગ ખોરાક, કાગળ, ટેક્સટાઇલ, અને ફાર્માસ્યુટિકલ ઉદ્યોગોમાં થાય છે. તે કાગળને મજબૂત બનાવવા, કપડાંને સ્ટાર્ચ કરવા, અને દવાઓમાં બાઇન્ડર તરીકે વાપરવામાં આવે છે. સ્ટાર્ચથી ઇથેનોલ અને બાયોપ્લાસ્ટિક પણ બનાવવામાં આવે છે.

  • ખોરાક ઉદ્યોગ: થિકનર અને સ્ટેબિલાઇઝર
  • કાગળ ઉદ્યોગ: મજબૂતી માટે
  • ટેક્સટાઇલ ઉદ્યોગ: સ્ટાર્ચિંગ માટે
  • ફાર્માસ્યુટિકલ: બાઇન્ડર તરીકે

સ્ટાર્ચ અને આયોડિન પરીક્ષણ

સ્ટાર્ચની હાજરી આયોડિન દ્રાવણ સાથે ચકાસી શકાય છે. જ્યારે આયોડિન દ્રાવણ સ્ટાર્ચ સાથે પ્રક્રિયા કરે છે, ત્યારે નીલો રંગ ઉત્પન્ન થાય છે. આ પરીક્ષણ સ્ટાર્ચની હાજરી માટે સરળ અને અસરકારક છે.

  • સ્ટાર્ચ + આયોડિન → નીલો રંગ
  • આ પરીક્ષણ સ્ટાર્ચની હાજરી માટે વાપરવામાં આવે છે
  • રંગ પરિવર્તન સ્ટાર્ચની હાજરી દર્શાવે છે
🧠 Trick: નીલો રંગ = સ્ટાર્ચની હાજરી

સ્ટાર્ચ અને ગ્લાયકોજન વચ્ચેનો તફાવત

સ્ટાર્ચ અને ગ્લાયકોજન બંને ગ્લૂકોઝના પોલિમર છે, પરંતુ તેમની રચના અને કાર્યમાં તફાવત છે. સ્ટાર્ચ વનસ્પતિમાં જોવા મળે છે અને શક્તિના સંગ્રહ માટે વાપરવામાં આવે છે, જ્યારે ગ્લાયકોજન પ્રાણીમાં જોવા મળે છે અને શક્તિના સંગ્રહ માટે વાપરવામાં આવે છે.

  • સ્ટાર્ચ: વનસ્પતિ, શક્તિ સંગ્રહ
  • ગ્લાયકોજન: પ્રાણી, શક્તિ સંગ્રહ
  • સ્ટાર્ચ: એમિલોઝ + એમિલોપેક્ટિન
  • ગ્લાયકોજન: શાખાયુક્ત રચના
🧠 Trick: સ્ટાર્ચ = વનસ્પતિ, ગ્લાયકોજન = પ્રાણી

  • સ્ટાર્ચ = એમિલોઝ (રેખીય) + એમિલોપેક્ટિન (શાખાયુક્ત)
  • રાસાયણિક સૂત્ર: (C₆H₁₀O₅)ₙ
  • વનસ્પતિમાં શક્તિ સંગ્રહના રૂપમાં જોવા મળે છે
  • પાચન એમિલેઝ ઉત્સેચક દ્વારા થાય છે
  • લાળા એમિલેઝ: સ્ટાર્ચ → મોલ્ટોઝ + ડેક્સ્ટ્રિન
  • પેન્ક્રિયેટિક એમિલેઝ: સ્ટાર્ચ → મોલ્ટોઝ + મોલ્ટોટ્રાયોઝ
  • સ્ટાર્ચ + આયોડિન → નીલો રંગ
  • ગ્લાયકોજન પ્રાણીમાં શક્તિ સંગ્રહ માટે વાપરવામાં આવે છે
  • સ્ટાર્ચનો ઉપયોગ ખોરાક, કાગળ, ટેક્સટાઇલ ઉદ્યોગોમાં થાય છે
  • એમિલોઝમાં α-1,4 બંધ, એમિલોપેક્ટિનમાં α-1,4 અને α-1,6 બંધ
🎯 Exam Tip: સ્ટાર્ચ અને ગ્લાયકોજન વચ્ચેના તફાવતને યાદ રાખો — સ્ટાર્ચ વનસ્પતિ માટે અને ગ્લાયકોજન પ્રાણી માટે શક્તિ સંગ્રહ કરે છે. આયોડિન પરીક્ષણ સ્ટાર્ચ માટે મહત્વપૂર્ણ છે.
