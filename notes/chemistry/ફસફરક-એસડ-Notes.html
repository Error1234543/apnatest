ફોસ્ફોરિક એસિડ

📖 ફોસ્ફોરિક એસિડ (H₃PO₄)

રચના અને ગુણધર્મો

ફોસ્ફોરિક એસિડ એ ત્રિઅમ્લીય એસિડ છે, જેમાં ફોસ્ફરસ +5 ઓક્સીડેશન અવસ્થામાં હોય છે. તે રંગહીન, ગંધહીન પ્રવાહી છે અને પાણીમાં અતિશય દ્રાવ્ય છે. તેની પ્રબળતા ક્લોરાઇડ, બ્રોમાઇડ અને આયોડાઇડ આયનોને ઓળખવા ઉપયોગી છે.

  • ત્રિઅમ્લીય એસિડ (pKa₁=2.1, pKa₂=7.2, pKa₃=12.3)
  • અણુ સૂત્ર: H₃PO₄, રચના: ટેટ્રાહેડ્રલ
  • પાણીમાં દ્રાવ્ય, હાઇગ્રોસ્કોપિક
  • ઉત્કલન બિંદુ: 158°C (40% દ્રાવણ)
  • અપચયન કરનાર (ઓક્સિડાઇઝિંગ એજન્ટ તરીકે કામ કરતો નથી)
🧠 Trick: ત્રણ H+ આપે તેને 'ત્રિઅમ્લીય' કહેવાય, યાદ રાખો '3P-O-H'

બનાવટ (પ્રયોગશાળા પદ્ધતિ)

પ્રયોગશાળામાં ફોસ્ફોરિક એસિડને ફોસ્ફરસ પેન્ટોક્સાઇડ (P₄O₁₀) ને પાણીમાં દ્રાવીને બનાવવામાં આવે છે. વાણિજ્યિક રીતે, ફોસ્ફેટ ખનિજ (Ca₃(PO₄)₂) ને સલ્ફ્યુરિક એસિડ સાથે પ્રક્રિયા કરાવીને મેળવવામાં આવે છે.

  • P₄O₁₀ + 6H₂O → 4H₃PO₄ (પ્રયોગશાળા)
  • Ca₃(PO₄)₂ + 3H₂SO₄ → 2H₃PO₄ + 3CaSO₄ (વાણિજ્યિક)
  • શુદ્ધિકરણ: કોન્સન્ટ્રેશન અને ક્રિસ્ટલાઇઝેશન
  • ઉત્પાદનમાં 70-80% H₃PO₄ મળે છે

રાસાયણિક પ્રક્રિયાઓ

ફોસ્ફોરિક એસિડ બેઇઝ સાથે પ્રક્રિયા કરે છે અને ફોસ્ફેટ આયન (PO₄³⁻) બનાવે છે. તે ધાતુઓ સાથે પ્રક્રિયા કરે છે અને હાઇડ્રોજન ગેસ મુક્ત કરે છે. તે કેલ્શિયમ સલ્ફેટ સાથે મળીને 'સુપરફોસ્ફેટ' ખાતર બનાવે છે.

  • H₃PO₄ + 3NaOH → Na₃PO₄ + 3H₂O
  • 2H₃PO₄ + 3Ca → Ca₃(PO₄)₂ + 3H₂↑
  • H₃PO₄ + CaSO₄ → Ca(H₂PO₄)₂ + H₂SO₄ (ખાતર ઉત્પાદન)
  • તાપમાન વધારે થતાં પાઇરોફોસ્ફોરિક એસિડ (H₄P₂O₇) બને છે
🧠 Trick: એસિડ + બેઇઝ = લવણ + પાણી, યાદ રાખો 'H₃PO₄ → PO₄³⁻'

ઉપયોગો

ફોસ્ફોરિક એસિડનો ઉપયોગ ખાતર (સુપરફોસ્ફેટ), ખોરાક અને પીણાં (pH નિયંત્રણ), ડિટરજન્ટ, દવા અને ધાતુશાસ્ત્રમાં થાય છે. તે DNA અને RNA ના બંધારણમાં મહત્વની ભૂમિકા ભજવે છે.

  • ખાતર: Ca(H₂PO₄)₂ (સિંગલ સુપરફોસ્ફેટ)
  • ખોરાક: કોલા પીણાંમાં એસિડિટી નિયંત્રણ
  • ડિટરજન્ટ: કેલ્શિયમ આયનોને બાંધીને પાણીને નરમ કરે
  • બાયોકેમિસ્ટ્રી: ATP (એડિનોસિન ટ્રાયફોસ્ફેટ) માં હાજર
  • ધાતુશાસ્ત્ર: ધાતુઓની સફાઈ અને કોટિંગ
🧠 Trick: ખાતર, કોલા, કેફે (CaFePO₄)

જૈવિક મહત્વ

ફોસ્ફોરિક એસિડ જૈવિક પ્રણાલીઓમાં ઉર્જા વહન (ATP/ADP) અને અનુવંશિક માહિતી (DNA/RNA) માટે આવશ્યક છે. તે હાડકાના બંધારણમાં અને કોષીય ચયાપચયમાં મહત્વની ભૂમિકા ભજવે છે.

  • ATP/ADP: ઉર્જા સંગ્રહ અને વહન
  • DNA/RNA: ન્યુક્લિયોટાઇડ્સમાં ફોસ્ફેટ જૂથ
  • હાડકા અને દાંત: હાઇડ્રોક્સીએપેટાઇટ (Ca₁₀(PO₄)₆(OH)₂)
  • કોષીય શ્વસન: ગ્લાયકોલિસિસ અને ક્રેબ્સ ચક્રમાં
  • પેશી વિકાસ: ફોસ્ફોલિપિડ્સમાં હાજર
🧠 Trick: ATP = એડિનોસિન + 3P (ફોસ્ફેટ)

પર્યાવરણીય અસરો

ફોસ્ફોરિક એસિડના અતિરેકથી પાણીના સ્રોતોમાં યુટ્રોફિકેશન થાય છે, જે જળીય જીવસૃષ્ટિ માટે હાનિકારક છે. ખાતર તરીકેના અતિ ઉપયોગથી મૃદા પ્રદૂષણ થાય છે.

  • યુટ્રોફિકેશન: શેવાળનો વિકાસ વધે છે
  • મૃદા પ્રદૂષણ: pHમાં ઘટાડો થાય છે
  • જળીય જીવન: ઓક્સિજનની ઉણપ થાય છે
  • નિયંત્રણ: સંતુલિત ખાતર ઉપયોગ

⚡ Quick Revision

  • ફોસ્ફોરિક એસિડ ત્રિઅમ્લીય છે (H₃PO₄)
  • P₄O₁₀ + H₂O → H₃PO₄ (પ્રયોગશાળા પદ્ધતિ)
  • ખાતર ઉત્પાદનમાં Ca₃(PO₄)₂ + H₂SO₄ વપરાય છે
  • ATPમાં 3 ફોસ્ફેટ જૂથ હોય છે
  • DNA/RNAમાં ફોસ્ફેટ જૂથ અનુવંશિકતામાં મહત્વનો છે
  • યુટ્રોફિકેશનથી જળીય જીવસૃષ્ટિને નુકસાન થાય છે
  • કોલા પીણાંમાં pH નિયંત્રણ માટે વપરાય છે
  • હાઇડ્રોક્સીએપેટાઇટ હાડકાના બંધારણમાં હાજર છે
🎯 Exam Tip: NEETમાં ફોસ્ફોરિક એસિડના જૈવિક મહત્વ (ATP, DNA) અને ખાતર ઉત્પાદન પર પ્રશ્નો વારંવાર પૂછાય છે — આ ભાગને પ્રાથમિકતા આપો.
