જૈવ અણુઓ

📖 જૈવ અણુઓ

કાર્બોદિત (Carbohydrates)

એક અથવા વધુ શર્કરાના અણુઓથી બનેલા જૈવ અણુઓ છે. મુખ્ય એનર્જી સ્રોત તરીકે કામ કરે છે. મોનોસેકેરાઇડ (ગ્લુકોઝ, ફ્રુક્ટોઝ), ડાયસેકેરાઇડ (સુક્રોઝ, લેક્ટોઝ), અને પોલીસેકેરાઇડ (સ્ટાર્ચ, સેલ્યુલોઝ, ગ્લાયકોજન) એ ત્રણ મુખ્ય પ્રકાર છે.

  • ગ્લુકોઝ = કોષીય શ્વસનનું મુખ્ય ઇંધણ
  • સેલ્યુલોઝ = વનસ્પતિ કોષની દિવાલ બનાવે છે
  • ગ્લાયકોજન = પ્રાણીઓમાં ગ્લુકોઝનું સંગ્રહ સ્વરૂપ
  • સુક્રોઝ = વનસ્પતિમાં પરિવહન માટેનો મુખ્ય કાર્બ
  • લેક્ટોઝ = દૂધમાં હાજર ડાયસેકેરાઇડ
🧠 Trick: ગ્લુકોઝ ગ્લાયકોજન ગોળ, સેલ્યુલોઝ સખત સોળ

પ્રોટીન (Proteins)

અમીનો એસિડના પોલિમર છે જેમાં પેપ્ટાઇડ બંધો દ્વારા જોડાયેલા હોય છે. રચનાત્મક, ઉત્સેચકીય, અને પરિવહન જેવી ભૂમિકા ભજવે છે. પ્રાથમિક, દ્વિતીય, તૃતીય, અને ચતુર્થ સ્ટ્રક્ચર જોવા મળે છે.

  • અમીનો એસિડ = પ્રોટીનની મૂળ ઇટ
  • પેપ્ટાઇડ બંધ = -CO-NH- બંધ
  • હીમોગ્લોબિન = ઓક્સિજન પરિવહન
  • ઉત્સેચક = જૈવ ઉત્પ્રેરક (બાયોકેટાલિસ્ટ)
  • કોલેજન = શરીરમાં સૌથી વધુ પ્રોટીન
🧠 Trick: પ્રો ટીન પેપ્ટાઇડ, એમીનો એસિડ જોડે

ચરબી (Lipids)

ચરબી, તેલ, અને સ્ટેરોઇડ્સનો સમાવેશ થાય છે. લાંબી ચેઇન વસા એસિડ અને ગ્લિસરોલથી બનેલા હોય છે. એનર્જી સંગ્રહ, કોષીય પડદાની રચના, અને હોર્મોન સંકેતનમાં મહત્વની ભૂમિકા ભજવે છે.

  • ટ્રાયગ્લિસરાઇડ = એનર્જી સંગ્રહનું સ્વરૂપ
  • ફોસ્ફોલિપિડ = કોષીય પડદાનો મુખ્ય ઘટક
  • સ્ટેરોઇડ = હોર્મોન (ટેસ્ટોસ્ટેરોન, ઇસ્ટ્રોજન)નું બિલ્ડિંગ બ્લોક
  • સેચ્યુરેટેડ ચરબી = પ્રાણી સ્રોત
  • અનસેચ્યુરેટેડ ચરબી = વનસ્પતિ સ્રોત
🧠 Trick: ચરબી ચરબી તેલ, ગ્લિસરોલ જોડે

ન્યુક્લિક એસિડ (Nucleic Acids)

DNA અને RNA એ બે મુખ્ય પ્રકાર છે. ન્યુક્લિયોટાઇડના પોલિમર છે જેમાં ફોસ્ફેટ, શર્કરા, અને નાઇટ્રોજનયુક્ત બેઇઝ હોય છે. અનુવંશિક માહિતીનું સંગ્રહ અને વહન કરે છે.

  • DNA = ડબલ હેલિક્સ, એડિનિન-થાયમિન, ગ્વાનિન-સાયટોસિન
  • RNA = સિંગલ સ્ટ્રેન્ડ, યુરેસિલ બેઇઝ
  • એટીપી = એનર્જી કરન્સી (ન્યુક્લિયોટાઇડ વ્યુત્પન્ન)
  • જનીન = DNAનો વંશાનુગત એકમ
  • મ્યુટેશન = ન્યુક્લિક એસિડમાં ફેરફાર
🧠 Trick: ડી એન એ ડબલ, આર એન એ સિંગલ

વિટામિન (Vitamins)

જૈવ રાસાયણિક પ્રક્રિયાઓ માટે જરૂરી સૂક્ષ્મ પોષક તત્ત્વો છે. જલીય (B, C) અને ચરબીમાં દ્રાવ્ય (A, D, E, K) એમ બે પ્રકાર છે. ઉત્સેચકીય ક્રિયાઓમાં સહાયક તરીકે કામ કરે છે.

  • વિટામિન A = દ્રષ્ટિ માટે
  • વિટામિન D = કેલ્શિયમ શોષણ
  • વિટામિન C = કોલેજન સંશ્લેષણ
  • વિટામિન B12 = RBC નિર્માણ
  • વિટામિન K = રક્ત ગંઠાવા માટે
🧠 Trick: A દ્રષ્ટિ, D હાડકા, C રોગપ્રતિકારક

ઉત્સેચક (Enzymes)

જૈવ ઉત્પ્રેરક છે જે રાસાયણિક પ્રક્રિયાઓની ગતિ વધારે છે. પ્રોટીન સ્વરૂપે હોય છે અને સબસ્ટ્રેટ સાથે વિશિષ્ટ રીતે બંધાય છે. તાપમાન અને pH ના સંવેદનશીલ હોય છે.

  • સક્રિય સ્થાન = સબસ્ટ્રેટ બંધન સ્થાન
  • ઉત્સેચક-સબસ્ટ્રેટ કોમ્પ્લેક્સ = પ્રક્રિયા દરમિયાન બને છે
  • ઑપ્ટિમમ તાપમાન = 37°C (માનવ શરીર)
  • pH સંવેદનશીલતા = પેપ્સિન (pH 2), ટ્રિપ્સિન (pH 8)
  • ઇનહિબિટર = ઉત્સેચક ક્રિયા અવરોધે છે
🧠 Trick: ઉત્સેચક ઉત્પ્રેરક, સબસ્ટ્રેટ સાથે જોડે

⚡ Quick Revision

  • કાર્બોદિત = ગ્લુકોઝ, સ્ટાર્ચ, સેલ્યુલોઝ
  • પ્રોટીન = અમીનો એસિડના પોલિમર
  • ચરબી = ટ્રાયગ્લિસરાઇડ, ફોસ્ફોલિપિડ
  • DNA = ડબલ હેલિક્સ, RNA = સિંગલ સ્ટ્રેન્ડ
  • વિટામિન = સૂક્ષ્મ પોષક તત્ત્વો
  • ઉત્સેચક = જૈવ ઉત્પ્રેરક
  • ન્યુક્લિક એસિડ = અનુવંશિક માહિતી વહન
  • ગ્લાયકોજન = પ્રાણીઓમાં ગ્લુકોઝ સંગ્રહ
  • સેલ્યુલોઝ = વનસ્પતિ કોષ દિવાલ
  • હીમોગ્લોબિન = ઓક્સિજન પરિવહન
🎯 Exam Tip: NEETમાં જૈવ અણુઓના પ્રકાર, બંધારણ, અને કાર્ય પર ધ્યાન કેન્દ્રિત કરો. ખાસ કરીને DNA-RNA, પ્રોટીનની રચના, અને ઉત્સેચકની ભૂમિકા પર વધુ પ્રશ્નો પૂછવામાં આવે છે.
