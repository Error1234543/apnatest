સપાટી આવેશ ઘનતા

📖 સપાટી આવેશ ઘનતા

સપાટી આવેશ ઘનતા શું છે?

સપાટી આવેશ ઘનતા એ એકમ ક્ષેત્રફળ દીઠ સપાટી પર રહેલો વિદ્યુતભાર છે. તેને ગ્રીક અક્ષર σ (સિગ્મા) વડે દર્શાવાય છે અને તેનો એકમ કુલંબ પ્રતિ વર્ગ મીટર (C/m²) છે. આ ખ્યાલ વિદ્યુતસ્થિતિમાન, કેપેસિટર અને વિદ્યુતક્ષેત્રની ગણતરીમાં મહત્વપૂર્ણ છે.

  • σ = Q/A (કુલ વિદ્યુતભાર/ક્ષેત્રફળ)
  • સપાટી જેટલી નાની, σ તેટલું વધારે
  • સુવાહકોમાં વિદ્યુતભાર હંમેશા બાહ્ય સપાટી પર જ રહે છે
🧠 Trick: સિગ્મા સપાટી પર સંગ્રહાય: σ = Q/A યાદ રાખો

સુવાહક અને અવાહકમાં σ નું વિતરણ

સુવાહકોમાં વિદ્યુતભાર માત્ર બાહ્ય સપાટી પર જ વિતરિત થાય છે, જ્યારે અવાહકોમાં વિદ્યુતભાર સમગ્ર કદમાં વિતરિત થઈ શકે છે. સુવાહકના અંદરના ભાગમાં વિદ્યુતક્ષેત્ર શૂન્ય હોય છે, તેથી σ માત્ર બાહ્ય સપાટી માટે જ ગણવામાં આવે છે.

  • સુવાહક: વિદ્યુતભાર બાહ્ય સપાટી પર જ
  • અવાહક: વિદ્યુતભાર સમગ્ર કદમાં વિતરિત
  • સુવાહકના અંદર વિદ્યુતક્ષેત્ર શૂન્ય
🧠 Trick: સુવાહક સપાટી પર સંગ્રહે, અવાહક અંદર ભરે

ગોળાકાર સુવાહક માટે σ ની ગણતરી

ગોળાકાર સુવાહક માટે સપાટી આવેશ ઘનતા σ = Q/(4πR²) હોય છે, જ્યાં Q કુલ વિદ્યુતભાર અને R ત્રિજ્યા છે. ગોળાકારની સપાટી પર σ સમાન રીતે વિતરિત થાય છે, જે વિદ્યુતક્ષેત્રની ગણતરીને સરળ બનાવે છે.

  • σ = Q/(4πR²) (ગોળાકાર માટે)
  • સપાટી પર σ સમાન રીતે વિતરિત
  • R વધે તો σ ઘટે (Q અચળ હોય તો)
🧠 Trick: ગોળા પર સિગ્મા: ચાર પાઇ આર સ્ક્વેર

વિદ્યુતક્ષેત્ર અને σ નો સંબંધ

સપાટી આવેશ ઘનતા σ અને વિદ્યુતક્ષેત્ર E વચ્ચેનો સંબંધ ગૉસના નિયમ દ્વારા મળે છે: E = σ/ε₀, જ્યાં ε₀ શૂન્યાવકાશની પરમિટિવિટી છે. આ સંબંધ સપાટી નજીકના વિદ્યુતક્ષેત્રની તીવ્રતા જાણવા માટે ઉપયોગી છે.

  • E = σ/ε₀ (ગૉસનો નિયમ)
  • ε₀ = 8.85 × 10⁻¹² C²/N·m²
  • σ વધે તો E પણ વધે
🧠 Trick: ઇ સિગ્મા ઇપ્સિલોન: E = σ/ε₀

કેપેસિટરમાં σ નું મહત્વ

કેપેસિટરની પ્લેટો પર સપાટી આવેશ ઘનતા σ = Q/A હોય છે, જે કેપેસિટરની ક્ષમતા C = ε₀A/d સાથે સંબંધિત છે. σ ની મદદથી કેપેસિટરમાં સંગ્રહિત ઊર્જા અને વિદ્યુતક્ષેત્રની ગણતરી કરી શકાય છે.

  • કેપેસિટરમાં σ = Q/A
  • C = ε₀A/d (સમાંતર પ્લેટ કેપેસિટર)
  • ઊર્જા U = ½CV² = ½QV
🧠 Trick: કેપેસિટરમાં સિગ્મા: Q ભાગ્યા A

આવેશિત સપાટી પર વિદ્યુત દબાણ

આવેશિત સપાટી પર વિદ્યુત દબાણ P = σ²/(2ε₀) હોય છે, જે સપાટી પરના વિદ્યુતભારો વચ્ચેના અપાકર્ષણને કારણે ઉત્પન્ન થાય છે. આ ખ્યાલ કેપેસિટરની પ્લેટો વચ્ચેના આકર્ષણ-અપાકર્ષણને સમજાવવામાં મદદરૂપ છે.

  • P = σ²/(2ε₀)
  • આવેશિત સપાટી પર અપાકર્ષણ દબાણ
  • σ વધે તો P વધે (વર્ગ પ્રમાણમાં)
🧠 Trick: દબાણ સિગ્મા સ્ક્વેર: P = σ²/(2ε₀)

⚡ Quick Revision

  • σ = Q/A: એકમ ક્ષેત્રફળ દીઠ વિદ્યુતભાર
  • સુવાહકમાં વિદ્યુતભાર માત્ર બાહ્ય સપાટી પર જ
  • ગોળાકાર માટે σ = Q/(4πR²)
  • E = σ/ε₀: ગૉસના નિયમથી વિદ્યુતક્ષેત્ર
  • કેપેસિટરમાં σ = Q/A, C = ε₀A/d
  • આવેશિત સપાટી પર દબાણ P = σ²/(2ε₀)
  • σ વધે તો E અને P બંને વધે
  • અવાહકમાં વિદ્યુતભાર સમગ્ર કદમાં વિતરિત
  • ε₀ = 8.85 × 10⁻¹² C²/N·m² (શૂન્યાવકાશની પરમિટિવિટી)
  • સપાટી નાની હોય તો σ વધારે હોય
🎯 Exam Tip: NEET પરીક્ષામાં σ સંબંધિત પ્રશ્નોમાં ગૉસના નિયમ (E = σ/ε₀) અને કેપેસિટરના સૂત્રો (C = ε₀A/d) વારંવાર પૂછાય છે. ગોળાકાર સુવાહક માટે σ = Q/(4πR²) યાદ રાખો અને σ ની મદદથી વિદ્યુતક્ષેત્ર કે દબાણની ગણતરી કરતી વખતે એકમો પર ખાસ ધ્યાન આપો.
