LCR પરિપથ

📖 LCR પરિપથ

LCR પરિપથનો પરિચય

LCR પરિપથ એ એવો વિદ્યુત પરિપથ છે જેમાં ઇન્ડક્ટર (L), કેપેસિટર (C) અને રેઝિસ્ટર (R) શ્રેણીમાં અથવા સમાંતરમાં જોડાયેલા હોય છે. આ પરિપથનો ઉપયોગ AC સર્કિટમાં રેઝોનન્સ, ફિલ્ટરિંગ અને ફેઝ એંગલના અભ્યાસ માટે થાય છે. NEETમાં આ પરિપથની ઊર્જા સંગ્રહ, ઓસિલેશન અને રેઝોનન્સ આવૃત્તિ પર પ્રશ્નો પૂછાય છે.

  • LCR પરિપથ = ઇન્ડક્ટર (L) + કેપેસિટર (C) + રેઝિસ્ટર (R)
  • શ્રેણી LCRમાં કુલ અવરોધ = √(R² + (X_L - X_C)²)
  • સમાંતર LCRમાં કુલ અવરોધ = 1/√(1/R² + (1/X_L - 1/X_C)²)
  • AC સર્કિટમાં ફેઝ એંગલ (φ) = tan⁻¹((X_L - X_C)/R)
🧠 Trick: LCR ને યાદ રાખવા: 'લાલ-કાળો-રંગ' (L-લાલ, C-કાળો, R-રંગ)

રેઝોનન્સની ઘટના

જ્યારે ઇન્ડક્ટિવ રિએક્ટન્સ (X_L) અને કેપેસિટિવ રિએક્ટન્સ (X_C) સમાન થાય ત્યારે પરિપથ રેઝોનન્સમાં હોય છે. આ સ્થિતિમાં પરિપથનો અવરોધ ન્યૂનતમ (R) અને પ્રવાહ મહત્તમ હોય છે. રેઝોનન્સ આવૃત્તિ (f₀) = 1/(2π√(LC)) દ્વારા નક્કી થાય છે.

  • રેઝોનન્સ સ્થિતિ: X_L = X_C ⇒ ω₀L = 1/(ω₀C)
  • રેઝોનન્સ આવૃત્તિ (f₀) = 1/(2π√(LC))
  • રેઝોનન્સમાં કુલ અવરોધ = R (ન્યૂનતમ)
  • પ્રવાહ મહત્તમ અને ફેઝ એંગલ (φ) = 0°
🧠 Trick: રેઝોનન્સ માટે: 'L અને C નો ગુણાકાર, વર્ગમૂળમાં 2π વડે ભાગ'

શ્રેણી LCR પરિપથની વિશેષતાઓ

શ્રેણી LCR પરિપથમાં વોલ્ટેજ અને પ્રવાહ વચ્ચે ફેઝ તફાવત ઉત્પન્ન થાય છે. ઇન્ડક્ટર વોલ્ટેજને પ્રવાહથી આગળ રાખે છે, જ્યારે કેપેસિટર વોલ્ટેજને પ્રવાહથી પાછળ રાખે છે. કુલ વોલ્ટેજ (V) = √(V_R² + (V_L - V_C)²) અને ફેઝ એંગલ (φ) = tan⁻¹((V_L - V_C)/V_R).

  • V_R = IR, V_L = IX_L, V_C = IX_C
  • કુલ વોલ્ટેજ (V) = √(V_R² + (V_L - V_C)²)
  • ફેઝ એંગલ (φ) = tan⁻¹((X_L - X_C)/R)
  • X_L > X_C ⇒ ઇન્ડક્ટિવ પરિપથ, X_C > X_L ⇒ કેપેસિટિવ પરિપથ
🧠 Trick: શ્રેણી LCR માટે: 'V_R ને પાયો, (V_L - V_C) ને ઊંચાઈ'

સમાંતર LCR પરિપથની વિશેષતાઓ

સમાંતર LCR પરિપથમાં પ્રવાહ વિભાજિત થાય છે અને વોલ્ટેજ સમાન રહે છે. કુલ પ્રવાહ (I) = √(I_R² + (I_C - I_L)²) અને ફેઝ એંગલ (φ) = tan⁻¹((I_C - I_L)/I_R). રેઝોનન્સમાં કુલ પ્રવાહ ન્યૂનતમ અને અવરોધ મહત્તમ હોય છે.

  • I_R = V/R, I_L = V/X_L, I_C = V/X_C
  • કુલ પ્રવાહ (I) = √(I_R² + (I_C - I_L)²)
  • ફેઝ એંગલ (φ) = tan⁻¹((I_C - I_L)/I_R)
  • રેઝોનન્સમાં I = I_R (ન્યૂનતમ)
🧠 Trick: સમાંતર LCR માટે: 'I_R ને પાયો, (I_C - I_L) ને ઊંચાઈ'

ઊર્જા અને પાવર

LCR પરિપથમાં ઇન્ડક્ટર અને કેપેસિટર ઊર્જા સંગ્રહ કરે છે, જ્યારે રેઝિસ્ટર ઊર્જા વ્યય કરે છે. સરેરાશ પાવર (P_avg) = V_rms I_rms cosφ, જ્યાં cosφ ને પાવર ફેક્ટર કહે છે. રેઝોનન્સમાં પાવર ફેક્ટર મહત્તમ (1) હોય છે.

  • ઇન્ડક્ટર ઊર્જા = ½LI², કેપેસિટર ઊર્જા = ½CV²
  • પાવર ફેક્ટર (cosφ) = R/Z
  • સરેરાશ પાવર (P_avg) = V_rms I_rms cosφ
  • રેઝોનન્સમાં P_avg = V_rms I_rms (મહત્તમ)
🧠 Trick: પાવર ફેક્ટર માટે: 'R ને Z વડે ભાગો'

ગુણવત્તા અવયવ (Q-ફેક્ટર)

Q-ફેક્ટર એ LCR પરિપથની ગુણવત્તા માપવાનો એક પરિમાણ છે, જે રેઝોનન્સની તીક્ષ્ણતા દર્શાવે છે. Q = ω₀L/R = 1/(ω₀CR) = √(L/C)/R. ઉચ્ચ Q-ફેક્ટરનો અર્થ વધુ તીક્ષ્ણ રેઝોનન્સ અને ઓછું ઊર્જા વ્યય છે.

  • Q = ω₀L/R = 1/(ω₀CR)
  • Q = √(L/C)/R
  • ઉચ્ચ Q ⇒ તીક્ષ્ણ રેઝોનન્સ, ઓછું વ્યય
  • બેન્ડવિડ્થ (Δf) = f₀/Q
🧠 Trick: Q-ફેક્ટર માટે: 'L ને R વડે ભાગો, અથવા C ને R વડે ગુણો'

⚡ Quick Revision

  • LCR પરિપથ = ઇન્ડક્ટર (L) + કેપેસિટર (C) + રેઝિસ્ટર (R)
  • રેઝોનન્સ સ્થિતિ: X_L = X_C ⇒ ω₀L = 1/(ω₀C)
  • રેઝોનન્સ આવૃત્તિ (f₀) = 1/(2π√(LC))
  • શ્રેણી LCRમાં કુલ અવરોધ = √(R² + (X_L - X_C)²)
  • સમાંતર LCRમાં કુલ અવરોધ = 1/√(1/R² + (1/X_L - 1/X_C)²)
  • ફેઝ એંગલ (φ) = tan⁻¹((X_L - X_C)/R)
  • પાવર ફેક્ટર (cosφ) = R/Z
  • સરેરાશ પાવર (P_avg) = V_rms I_rms cosφ
  • Q-ફેક્ટર = ω₀L/R = √(L/C)/R
  • રેઝોનન્સમાં પાવર ફેક્ટર = 1 (મહત્તમ)
🎯 Exam Tip: NEETમાં LCR પરિપથના પ્રશ્નોમાં રેઝોનન્સ આવૃત્તિ અને પાવર ફેક્ટર પર વધુ ધ્યાન આપો. રેઝોનન્સ સ્થિતિ (X_L = X_C) અને પાવર ફેક્ટરનું સૂત્ર (cosφ = R/Z) યાદ રાખીને ગણતરી ઝડપી બનાવો. ગ્રાફ-આધારિત પ્રશ્નોમાં ફેઝ એંગલ અને પ્રવાહ/વોલ્ટેજના સંબંધ પર ધ્યાન આપો.
