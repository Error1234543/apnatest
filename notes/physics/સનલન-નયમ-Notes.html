સ્નેલનો નિયમ

📖 સ્નેલનો નિયમ (Snell's Law)

પરિભાષા અને મૂળભૂત સંકલ્પના

સ્નેલનો નિયમ પ્રકાશના અપવર્તન સાથે સંબંધિત છે. જ્યારે પ્રકાશ એક માધ્યમથી બીજા માધ્યમમાં પસાર થાય છે, ત્યારે તેનો માર્ગ બદલાય છે. આ નિયમ પ્રકાશના આપતન કોણ (i) અને અપવર્તન કોણ (r) વચ્ચેનો સંબંધ વ્યાખ્યાયિત કરે છે.

  • પ્રકાશનું અપવર્તન = માર્ગમાં વક્રીકરણ
  • આપતન કોણ (i) = આપતિત કિરણ અને લંબ વચ્ચેનો કોણ
  • અપવર્તન કોણ (r) = અપવર્તિત કિરણ અને લંબ વચ્ચેનો કોણ
  • સ્નેલનો નિયમ: n₁sin(i) = n₂sin(r)
  • n = માધ્યમનો વક્રીભવનાંક (RI)
🧠 Trick: SIN i = n₂/n₁ * SIN r → 'SIN' સદા યાદ રાખો!

વક્રીભવનાંક (Refractive Index)

વક્રીભવનાંક (n) એ માધ્યમની પ્રકાશની ગતિને નિર્ધારિત કરે છે. તે નિર્વાતમાં પ્રકાશની ગતિ (c) અને માધ્યમમાં પ્રકાશની ગતિ (v) ના ગુણોત્તર તરીકે વ્યાખ્યાયિત થાય છે: n = c/v.

  • n = c/v (c = 3×10⁸ m/s, નિર્વાતમાં ગતિ)
  • n > 1: પ્રકાશ મંદ પડે (ઉદા. કાચ, પાણી)
  • n < 1: થીયરેટિકલ (વાયુમાં ≈1)
  • સાપેક્ષ વક્રીભવનાંક: n₂₁ = n₂/n₁
  • ઉચ્ચ n = વધુ વક્રીકરણ

સ્નેલનો નિયમનો ગણિતી સ્વરૂપ

સ્નેલનો નિયમ અનુસાર, આપતન કોણ અને અપવર્તન કોણના સાઇનના ગુણોત્તર માધ્યમોના વક્રીભવનાંકના વ્યસ્ત ગુણોત્તર જેટલું હોય છે: n₁sin(i) = n₂sin(r). આ સૂત્ર અપવર્તનની દિશા અને પ્રમાણ નક્કી કરે છે.

  • n₁sin(i) = n₂sin(r)
  • જો n₂ > n₁ → કિરણ લંબ તરફ વળે (r < i)
  • જો n₂ < n₁ → કિરણ લંબથી દૂર વળે (r > i)
  • i = 0° → r = 0° (કોઈ અપવર્તન નહીં)
  • ક્રાંતિક કોણ (θ_c) = sin⁻¹(n₂/n₁), જ્યારે n₁ > n₂
🧠 Trick: ઊંચું n → કિરણ લંબ તરફ (HIGH n → TOWARDS NORMAL)

ક્રાંતિક કોણ અને પૂર્ણ આંતરિક પરાવર્તન

જ્યારે પ્રકાશ ઘન માધ્યમથી વિરળ માધ્યમ તરફ જાય છે અને આપતન કોણ ક્રાંતિક કોણથી વધુ હોય છે, ત્યારે પૂર્ણ આંતરિક પરાવર્તન થાય છે. ક્રાંતિક કોણ θ_c = sin⁻¹(n₂/n₁), જ્યાં n₁ > n₂.

  • θ_c = sin⁻¹(n₂/n₁) (n₁ > n₂)
  • i > θ_c → પૂર્ણ આંતરિક પરાવર્તન
  • ઉદા. હીરામાં θ_c ≈ 24.4° (n=2.42)
  • ઓપ્ટિકલ ફાઇબર અને મણકામાં ઉપયોગ
  • પૂર્ણ પરાવર્તન માટે માધ્યમોની જરૂર: ઘન → વિરળ
🧠 Trick: ક્રાંતિક કોણ = SIN⁻¹(નાનું n / મોટું n)

અપવર્તનના નિયમો

અપવર્તનના ત્રણ મૂળભૂત નિયમો છે: 1) આપતિત કિરણ, અપવર્તિત કિરણ અને લંબ એક જ તલમાં હોય છે. 2) આપતન કોણ અને અપવર્તન કોણનો સાઇનનો ગુણોત્તર સ્થિર હોય છે (સ્નેલનો નિયમ). 3) પ્રકાશની ગતિ બદલાય છે પરંતુ આવર્તન અચળ રહે છે.

  • એક જ તલમાં આપતિત, અપવર્તિત કિરણ અને લંબ
  • sin(i)/sin(r) = n₂/n₁ = સ્થિર
  • આવર્તન (frequency) અચળ રહે છે
  • તરંગલંબાઇ (λ) બદલાય: λ ∝ 1/n
  • પ્રકાશની ગતિ (v) = c/n

ઉદાહરણો અને એપ્લિકેશન્સ

સ્નેલનો નિયમ લેન્સ, દર્પણ, પ્રિઝમ અને ઓપ્ટિકલ યંત્રોમાં વ્યાપકપણે ઉપયોગમાં લેવાય છે. ઉદાહરણ તરીકે, કાચના પ્રિઝમમાં પ્રકાશના વિભાજન (વર્ણપટ) માટે અપવર્તન જવાબદાર છે.

  • લેન્સ: ઉત્તલ/અવતલ લેન્સમાં ફોકલ લંબાઇ n પર આધારિત
  • પ્રિઝમ: વર્ણપટ રચના (n વર્ણ અનુસાર બદલાય)
  • દૂરબીન/માઈક્રોસ્કોપ: પ્રકાશના અપવર્તનનો ઉપયોગ
  • મણકો: પૂર્ણ આંતરિક પરાવર્તન દ્વારા પ્રકાશનો પરિવહન
  • રેઇનબો: પાણીના ટીપામાં અપવર્તન અને પરાવર્તન

⚡ Quick Revision

  • સ્નેલનો નિયમ: n₁sin(i) = n₂sin(r)
  • વક્રીભવનાંક (n) = c/v
  • ઊંચું n → પ્રકાશ મંદ પડે
  • ક્રાંતિક કોણ θ_c = sin⁻¹(n₂/n₁) (n₁ > n₂)
  • i > θ_c → પૂર્ણ આંતરિક પરાવર્તન
  • અપવર્તનમાં આવર્તન અચળ, λ અને v બદલાય
  • લેન્સ/પ્રિઝમમાં અપવર્તનનો ઉપયોગ
  • ઓપ્ટિકલ ફાઇબરમાં પૂર્ણ પરાવર્તન
  • રેઇનબો = અપવર્તન + પરાવર્તન + વિક્ષેપન
  • sin(i)/sin(r) = n₂/n₁ = સ્થિર
🎯 Exam Tip: NEETમાં સ્નેલનો નિયમ અને ક્રાંતિક કોણ પર પ્રશ્નો વારંવાર પૂછાય છે. n₁, n₂, i, r ની કિંમતો આપીને sin(r) અથવા θ_c શોધવા માટે તૈયાર રહો. ગણિતી પ્રશ્નો માટે સૂત્ર યાદ રાખો અને યોગ્ય એકમોનો ઉપયોગ કરો.
