એમ્પિયરનો નિયમ

📖 એમ્પિયરનો નિયમ (Ampere's Law)

પરિભાષા અને મૂળભૂત સંકલ્પના

એમ્પિયરનો નિયમ વિદ્યુતપ્રવાહ દ્વારા ઉત્પન્ન થતાં ચુંબકીય ક્ષેત્રની તીવ્રતાનું વિતરણ વર્ણવે છે. આ નિયમ મુજબ, કોઈપણ બંધ લૂપમાં ચુંબકીય ક્ષેત્રની રેખા-સમાકલન (line integral) એ લૂપ દ્વારા પસાર થતાં વિદ્યુતપ્રવાહના કુલ મૂલ્યના μ₀ ગુણ્યા જેટલું હોય છે.

  • એમ્પિયરનો નિયમ: ∮ **B**·d**l** = μ₀ * I_enc
  • μ₀ = 4π × 10⁻⁷ T·m/A (નિર્વાતની ચુંબકીય શોષકતા)
  • I_enc = લૂપ દ્વારા પસાર થતો કુલ પ્રવાહ
  • લૂપની પસંદગી કોઈપણ હોઈ શકે, પરંતુ સરળતા માટે સહેજ સમમિત લૂપ પસંદ કરવામાં આવે
🧠 Trick: B·dl = μ₀I → 'બી ડી એલ મ્યૂ ઝીરો આઈ' (બધા મહત્વના પડાવ)

ગાણિતિક સ્વરૂપ અને વ્યાખ્યા

એમ્પિયરનો નિયમ ગાણિતિક રીતે ∮ **B**·d**l** = μ₀ * ΣI ના સ્વરૂપે વ્યક્ત થાય છે. અહીં **B** ચુંબકીય ક્ષેત્ર, d**l** લૂપનો અતિશય નાનો વેક્ટર તત્વ, અને ΣI એ લૂપ દ્વારા પસાર થતો કુલ પ્રવાહ છે. આ નિયમ મેક્સવેલના ચુંબકીય ક્ષેત્રના ચાર સમીકરણોમાંનું એક છે.

  • સદિશ રૂપે: ∮ **B**·d**l** = μ₀ * I_enc
  • સ્કેલર રૂપે: B * (2πr) = μ₀ * I (લંબચોરસ લૂપ માટે)
  • લૂપની દિશા અને પ્રવાહની દિશા જમણા હાથના નિયમ દ્વારા નક્કી કરવામાં આવે

લંબચોરસ લૂપ માટેનો ઉપયોગ

જ્યારે વિદ્યુતપ્રવાહ વહન કરતો તાર લંબચોરસ આકારમાં હોય, ત્યારે એમ્પિયરનો નિયમ સરળતાથી લાગુ પડે છે. અહીં, ચુંબકીય ક્ષેત્રની તીવ્રતા B = (μ₀ * I) / (2πr) થાય છે, જ્યાં r એ તારથી અંતર છે.

  • લંબચોરસ લૂપ માટે B સમાન રહે છે
  • B ∝ 1/r (તારથી અંતર વધે ત્યારે B ઘટે)
  • લૂપની લંબાઈ L હોય તો કુલ ચુંબકીય ક્ષેત્ર B * L = μ₀ * I
🧠 Trick: B = μ₀I / 2πr → 'બી મ્યૂ આઈ બાર પાઈ આર'

સોલેનોઇડ અને ટોરોઇડ માટેનો ઉપયોગ

સોલેનોઇડ (વીંટાળેલ તાર) અને ટોરોઇડ (ગોળાકાર વીંટાળેલ તાર) માટે એમ્પિયરનો નિયમ ચુંબકીય ક્ષેત્રની ગણતરી માટે ઉપયોગી છે. સોલેનોઇડમાં B = μ₀ * n * I થાય છે, જ્યાં n એ પ્રમાણમાં ફેરાની સંખ્યા છે.

  • સોલેનોઇડમાં B = μ₀ * n * I (અંદરના ભાગમાં)
  • ટોરોઇડમાં B = (μ₀ * N * I) / (2πr) (N = કુલ ફેરા)
  • બહારના ભાગમાં B ≈ 0 (ઇડિયલ કેસમાં)
🧠 Trick: સોલેનોઇડ: B = μ₀nI → 'બી મ્યૂ એન આઈ'

એમ્પિયરનો નિયમ અને મેક્સવેલના સમીકરણો

એમ્પિયરનો નિયમ મેક્સવેલના ચુંબકીય ક્ષેત્રના ચાર સમીકરણોમાંનું એક છે. આ નિયમ સ્થિર પ્રવાહ માટે લાગુ પડે છે, પરંતુ ચલ પ્રવાહ માટે મેક્સવેલે તેને સુધારીને ∮ **B**·d**l** = μ₀ * (I + ε₀ * dΦ_E/dt) કર્યું છે, જ્યાં dΦ_E/dt એ વિદ્યુત ફ્લક્સનો ફેરફાર છે.

  • સ્થિર પ્રવાહ માટે: ∮ **B**·d**l** = μ₀ * I
  • ચલ પ્રવાહ માટે: ∮ **B**·d**l** = μ₀ * (I + I_d) (I_d = વિસ્થાપન પ્રવાહ)
  • મેક્સવેલના સમીકરણો વિદ્યુતચુંબકીય તરંગોની વ્યાખ્યા આપે છે

ઉદાહરણ: લાંબો સીધો તાર

લાંબો સીધો તાર જ્યારે વિદ્યુતપ્રવાહ I વહન કરે છે, ત્યારે તારથી r અંતરે ચુંબકીય ક્ષેત્ર B = (μ₀ * I) / (2πr) થાય છે. આ એમ્પિયરના નિયમનું સૌથી સરળ ઉદાહરણ છે.

  • B ∝ I (પ્રવાહ વધે ત્યારે B વધે)
  • B ∝ 1/r (અંતર વધે ત્યારે B ઘટે)
  • દિશા જમણા હાથના નિયમ દ્વારા નક્કી કરવામાં આવે
🧠 Trick: જમણો હાથ: અંગૂઠો = પ્રવાહ, આંગળીઓ = Bની દિશા

⚡ Quick Revision

  • એમ્પિયરનો નિયમ: ∮ B·dl = μ₀ I_enc
  • μ₀ = 4π × 10⁻⁷ T·m/A (નિર્વાતની શોષકતા)
  • લંબચોરસ લૂપ: B = μ₀I / (2πr)
  • સોલેનોઇડ: B = μ₀nI (અંદર)
  • ટોરોઇડ: B = μ₀NI / (2πr)
  • લાંબો તાર: B ∝ I/r
  • દિશા: જમણા હાથનો નિયમ
  • મેક્સવેલના સમીકરણોમાં એક ભાગ
  • સ્થિર પ્રવાહ માટે લાગુ
  • ચુંબકીય ક્ષેત્રની ગણતરી માટે ઉપયોગી
🎯 Exam Tip: એમ્પિયરનો નિયમ લાગુ કરતી વખતે લૂપની પસંદગી સહેજ સમમિત રાખો (જેમ કે લંબચોરસ અથવા વર્તુળાકાર) જેથી B·dl સરળતાથી ગણી શકાય. સોલેનોઇડ અને ટોરોઇડના પ્રશ્નો માટે આ નિયમ ખૂબ જ ઉપયોગી છે.
