ન્યુક્લિયર વિખંડન

📖 ન્યુક્લિયર વિખંડન

ન્યુક્લિયર વિખંડનની મૂળભૂત સમજ

ન્યુક્લિયર વિખંડન એ ભારે અણુકોષના ન્યુક્લિયસનું ટૂટીને હલકા અણુકોષોમાં વિભાજન થવાની પ્રક્રિયા છે. આ પ્રક્રિયામાં મોટી માત્રામાં ઉર્જા મુક્ત થાય છે, જે આણ્વિક બંધનોમાં રહેલી બંધન ઉર્જાના તફાવતને લીધે થાય છે. યુરેનિયમ-235 અથવા પ્લુટોનિયમ-239 જેવા વિખંડનશીલ પદાર્થો આ પ્રક્રિયા માટે વપરાય છે.

  • ભારે ન્યુક્લાઇડ્સ (A > 200) વિખંડન પામે છે.
  • ન્યુટ્રોનના અથડામણથી પ્રારંભ થાય છે.
  • બંધન ઉર્જા પ્રતિ ન્યુક્લિયોન વધારે હોય છે.
  • ઉર્જા E=mc² અનુસાર મુક્ત થાય છે.
🧠 Trick: ભારે ન્યુક્લિયસ + ન્યુટ્રોન → હલકા ન્યુક્લિયસ + ઉર્જા + ન્યુટ્રોન (બીજા વિખંડન માટે)

વિખંડન પ્રક્રિયાનો તબક્કાવાર વર્ણન

ન્યુટ્રોન યુરેનિયમ-235 ન્યુક્લિયસમાં પ્રવેશે છે, જે અસ્થિર યુરેનિયમ-236 બનાવે છે. આ ન્યુક્લિયસ તરત જ વિભાજિત થઈને બેરિયમ-141, ક્રિપ્ટોન-92, 3 ન્યુટ્રોન અને મોટી માત્રામાં ઉર્જા મુક્ત કરે છે. મુક્ત થયેલા ન્યુટ્રોન પુનરાવર્તિત વિખંડન પ્રક્રિયા શરૂ કરે છે, જે શૃંખલા પ્રક્રિયા તરીકે ઓળખાય છે.

  • ન્યુટ્રોન અથડામણ → અસ્થિર ન્યુક્લિયસ (U-236).
  • વિભાજન → Ba-141 + Kr-92 + 3n + ઉર્જા (≈200 MeV).
  • શૃંખલા પ્રક્રિયા માટે ક્રિટિકલ દ્રવ્યમાન જરૂરી છે.
  • પરમાણુ બોમ્બ આ જ સિદ્ધાંત પર કામ કરે છે.
🧠 Trick: U-235 + n → U-236* → Ba + Kr + 3n + 200 MeV

શૃંખલા પ્રક્રિયા અને ક્રિટિકલ દ્રવ્યમાન

શૃંખલા પ્રક્રિયામાં, એક વિખંડનથી મુક્ત થયેલા ન્યુટ્રોન બીજા ન્યુક્લિયસને વિખંડિત કરે છે, જે અત્યંત ઝડપી ઉર્જા મુક્તિ તરફ દોરી જાય છે. ક્રિટિકલ દ્રવ્યમાન એ વિખંડનશીલ પદાર્થની ન્યૂનતમ માત્રા છે જે શૃંખલા પ્રક્રિયા ચાલુ રાખવા માટે જરૂરી છે. જો દ્રવ્યમાન ક્રિટિકલથી ઓછું હોય, તો પ્રક્રિયા બંધ થઈ જાય છે.

  • શૃંખલા પ્રક્રિયા = સ્વ-નિયંત્રિત વિખંડન પ્રક્રિયા.
  • ક્રિટિકલ દ્રવ્યમાન = પ્રક્રિયા ચાલુ રાખવા માટેની ન્યૂનતમ માત્રા.
  • સબ-ક્રિટિકલ દ્રવ્યમાન → પ્રક્રિયા બંધ.
  • સુપર-ક્રિટિકલ દ્રવ્યમાન → વિસ્ફોટક પ્રક્રિયા.
🧠 Trick: ક્રિટિકલ = ચાલુ, સબ-ક્રિટિકલ = બંધ, સુપર-ક્રિટિકલ = વિસ્ફોટ

ન્યુક્લિયર રિયેક્ટરમાં વિખંડનનો ઉપયોગ

ન્યુક્લિયર રિયેક્ટરમાં, વિખંડન પ્રક્રિયાનો ઉપયોગ નિયંત્રિત રીતે ઉર્જા ઉત્પન્ન કરવા માટે થાય છે. રિયેક્ટરમાં કંટ્રોલ રોડ્સ (બોરોન અથવા કેડમિયમના બનેલા) વાપરીને ન્યુટ્રોનની સંખ્યા નિયંત્રણમાં રાખવામાં આવે છે. મોડરેટર (જેમ કે ગ્રેફાઇટ અથવા ભારે પાણી) ન્યુટ્રોનની ગતિ ઘટાડીને વિખંડનની સંભાવના વધારે છે.

  • રિયેક્ટરમાં નિયંત્રિત શૃંખલા પ્રક્રિયા થાય છે.
  • કંટ્રોલ રોડ્સ ન્યુટ્રોન શોષીને પ્રક્રિયા નિયંત્રે છે.
  • મોડરેટર ન્યુટ્રોનની ગતિ ઘટાડે છે.
  • કૂલન્ટ (પાણી, સોડિયમ) ઉષ્મા શોષીને ટર્બાઇન ચલાવે છે.
🧠 Trick: રોડ્સ = નિયંત્રણ, મોડરેટર = ધીમું, કૂલન્ટ = ઠંડક

ન્યુક્લિયર વિખંડનની ઉર્જા અને ઉપયોગિતા

ન્યુક્લિયર વિખંડન દ્વારા મુક્ત થતી ઉર્જા અત્યંત મોટી હોય છે. 1 કિગ્રા યુરેનિયમ-235ના પૂર્ણ વિખંડનથી લગભગ 8 x 10^13 જૂલ ઉર્જા મુક્ત થાય છે, જે 3 મિલિયન ટન કોળસા જેટલી ઉર્જા બરોબર છે. આ ઉર્જાનો ઉપયોગ વિદ્યુત ઉત્પાદન, પરમાણુ શસ્ત્રો અને ચિકિત્સા ક્ષેત્રે થાય છે.

  • 1 કિગ્રા U-235 → 8 x 10^13 J ઉર્જા.
  • વિદ્યુત ઉત્પાદન માટે ન્યુક્લિયર પાવર પ્લાન્ટ્સ.
  • પરમાણુ બોમ્બમાં અણુ વિસ્ફોટ માટે વપરાય છે.
  • કૅન્સરના ઉપચારમાં રેડિયોથેરાપી માટે વપરાય છે.
🧠 Trick: 1 કિગ્રા U-235 = 3 મિલિયન ટન કોળસા

ન્યુક્લિયર વિખંડનના જોખમો અને સાવચેતી

ન્યુક્લિયર વિખંડન દરમિયાન રેડિયોએક્ટિવ કચરો ઉત્પન્ન થાય છે, જે પર્યાવરણ અને માનવ સ્વાસ્થ્ય માટે હાનિકારક છે. રેડિયોએક્ટિવ કચરાનું નિપટારા કરવા માટે ખાસ પદ્ધતિઓ જરૂરી છે. આ ઉપરાંત, પરમાણુ અકસ્માતો (જેમ કે ચેરનોબિલ, ફુકુશિમા) ના જોખમો પણ છે, જેના માટે સખત સુરક્ષા પગલાં લેવા પડે છે.

  • રેડિયોએક્ટિવ કચરો → પર્યાવરણ માટે હાનિકારક.
  • ચેરનોબિલ અને ફુકુશિમા જેવા અકસ્માતોના જોખમો.
  • કચરાનું નિપટારા → ગહન ભૂગર્ભ સંગ્રહ.
  • સુરક્ષા પગલાં → શીલ્ડિંગ, કંટ્રોલ સિસ્ટમ્સ.
🧠 Trick: રેડિયો કચરો = ખતરનાક, સુરક્ષા = જરૂરી

⚡ Quick Revision

  • ન્યુક્લિયર વિખંડન = ભારે ન્યુક્લિયસનું હલકા ન્યુક્લિયસમાં વિભાજન.
  • U-235 + n → Ba + Kr + 3n + 200 MeV ઉર્જા.
  • શૃંખલા પ્રક્રિયા = સ્વ-નિયંત્રિત વિખંડન.
  • ક્રિટિકલ દ્રવ્યમાન = પ્રક્રિયા ચાલુ રાખવા માટેની ન્યૂનતમ માત્રા.
  • રિયેક્ટરમાં કંટ્રોલ રોડ્સ અને મોડરેટર વપરાય છે.
  • 1 કિગ્રા U-235 → 8 x 10^13 J ઉર્જા મુક્ત થાય છે.
  • રેડિયોએક્ટિવ કચરો પર્યાવરણ માટે હાનિકારક છે.
  • પરમાણુ બોમ્બ અને વિદ્યુત ઉત્પાદનમાં ઉપયોગી છે.
🎯 Exam Tip: ન્યુક્લિયર વિખંડન અને ન્યુક્લિયર સંલયન વચ્ચેનો તફાવત સ્પષ્ટ કરો. વિખંડન ભારે ન્યુક્લિયસનું વિભાજન છે, જ્યારે સંલયન હલકા ન્યુક્લિયસનું જોડાણ છે.
