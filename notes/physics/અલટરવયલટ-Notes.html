અલ્ટ્રાવાયોલેટ

📖 અલ્ટ્રાવાયોલેટ (UV) કિરણોત્સર્ગ

પરિચય અને વર્ગીકરણ

અલ્ટ્રાવાયોલેટ (UV) કિરણોત્સર્ગ એ વિદ્યુતચુંબકીય સ્પેક્ટ્રમનો એક ભાગ છે, જેમાં તરંગલંબાઈ 10 nm થી 400 nm વચ્ચે હોય છે. NCERT મુજબ, તેને ત્રણ ઉપવિભાગોમાં વર્ગીકૃત કરવામાં આવે છે: UV-A (315–400 nm), UV-B (280–315 nm), અને UV-C (100–280 nm). UV-C સૌથી વધુ ઊર્જાવાન અને નુકસાનકારક હોય છે, પરંતુ તે વાતાવરણના ઓઝોન સ્તર દ્વારા શોષાઈ જાય છે.

  • UV કિરણોત્સર્ગની તરંગલંબાઈ: 10–400 nm
  • UV-A: લાંબી તરંગલંબાઈ, ઓછું નુકસાનકારક
  • UV-B: મધ્યમ તરંગલંબાઈ, ત્વચાને નુકસાન પહોંચાડે
  • UV-C: ટૂંકી તરંગલંબાઈ, સૌથી ઊર્જાવાન અને જીવાણુનાશક
  • ઓઝોન સ્તર UV-C ને શોષી લે છે
🧠 Trick: UV-A B C: A=આરોગ્યકર (less harmful), B=બળતરા (sunburn), C=ચેપનાશક (germicidal)

ભૌતિક ગુણધર્મો

UV કિરણો અદૃશ્ય હોય છે પરંતુ ફ્લોરેસેન્ટ પદાર્થોને પ્રકાશિત કરવા માટે જવાબદાર હોય છે. તેની ઊર્જા X-kirણો કરતાં ઓછી અને દૃશ્ય પ્રકાશ કરતાં વધુ હોય છે. UV કિરણો ફોટોઇલેક્ટ્રિક ઇફેક્ટ અને ફોટોકેમિકલ પ્રક્રિયાઓમાં મહત્વની ભૂમિકા ભજવે છે, જેમ કે ફોટોગ્રાફી અને સ્ટેરિલાઇઝેશન.

  • અદૃશ્ય કિરણો, પરંતુ ફ્લોરેસન્સ પેદા કરે
  • ઊર્જા: દૃશ્ય પ્રકાશ > UV > X-kirણો
  • ફોટોઇલેક્ટ્રિક ઇફેક્ટમાં સક્રિય
  • ફોટોકેમિકલ પ્રક્રિયાઓ માટે જવાબદાર
  • સ્ટેરિલાઇઝેશનમાં ઉપયોગી

જૈવિક અસરો (Biology - NCERT Class 11)

UV કિરણો DNA અને RNA ને નુકસાન પહોંચાડી શકે છે, જે મ્યુટેશન તરફ દોરી શકે છે. તે ત્વચામાં મેલેનિન ઉત્પાદનને ઉત્તેજિત કરે છે, જે સનટેનની જેમ કાળા પડવાનું કારણ બને છે. લાંબા સમય સુધી UV બ્લોચરને કારણે ત્વચા કેંસર અને મોતિયા જેવા રોગો થઈ શકે છે. ક્લોરોફિલ UV કિરણોને શોષી શકતું નથી, પરંતુ તે પ્રકાશસંશ્લેષણ પર પરોક્ષ અસર કરે છે.

  • DNA/RNA ને નુકસાન: થાયમિન ડાઇમર બનાવે
  • મેલેનિન ઉત્પાદન વધારે: સનટેન
  • લાંબા સમયની અસર: ત્વચા કેંસર, મોતિયા
  • ક્લોરોફિલ UV શોષતું નથી
  • પ્રકાશસંશ્લેષણ પર પરોક્ષ અસર
🧠 Trick: UV = Unwanted Visitor for DNA (મ્યુટેશન)

રસાયણશાસ્ત્રમાં ઉપયોગ (Chemistry - NCERT Class 11 & 12)

UV કિરણોત્સર્ગ રસાયણશાસ્ત્રમાં સ્પેક્ટ્રોસ્કોપી, ફોટોકેમિકલ પ્રક્રિયાઓ અને જીવાણુનાશક તરીકે ઉપયોગી છે. UV-Vis સ્પેક્ટ્રોસ્કોપી દ્વારા પદાર્થોની સાંદ્રતા અને રચના નક્કી કરવામાં આવે છે. ફોટોક્લોરિનેશન અને ફોટોઓક્સિડેશન જેવી પ્રક્રિયાઓમાં UV કિરણો કેટલીક રાસાયણિક પ્રક્રિયાઓને ઉત્તેજિત કરે છે.

  • UV-Vis સ્પેક્ટ્રોસ્કોપી: સાંદ્રતા અને રચના નક્કી કરે
  • ફોટોકેમિકલ પ્રક્રિયાઓમાં કેટલીક રાસાયણિક પ્રક્રિયાઓ ઉત્તેજિત કરે
  • જીવાણુનાશક તરીકે ઉપયોગ
  • ફોટોક્લોરિનેશન અને ફોટોઓક્સિડેશનમાં મહત્વપૂર્ણ
  • પોલિમરાઇઝેશન પ્રક્રિયાઓમાં ઉપયોગી
🧠 Trick: UV = Useful in Chemistry for Spectroscopy & Sterilization

ભૌતિકશાસ્ત્રમાં અભ્યાસ (Physics - NCERT Class 12)

UV કિરણોત્સર્ગના અભ્યાસમાં ફોટોઇલેક્ટ્રિક ઇફેક્ટ, કોમ્પટન ઇફેક્ટ અને બ્લેકબોડી રેડિયેશન જેવા વિષયો સામેલ છે. UV કિરણોની ઊર્જા (E = hν) તેની ફ્રિક્વન્સી પર આધારિત હોય છે. તે અવકાશી અભ્યાસમાં પણ મહત્વપૂર્ણ છે, જ્યાં UV ટેલિસ્કોપ દ્વારા તારાઓ અને ગેલેક્સીઓનો અભ્યાસ કરવામાં આવે છે.

  • ફોટોઇલેક્ટ્રિક ઇફેક્ટ: UV કિરણો ઇલેક્ટ્રોનને બહાર કાઢે
  • E = hν: ઊર્જા ફ્રિક્વન્સી પર આધારિત
  • કોમ્પટન ઇફેક્ટ: UV કિરણો સાથે ઇલેક્ટ્રોનનું પ્રકીર્ણન
  • બ્લેકબોડી રેડિયેશનમાં સામેલ
  • UV ટેલિસ્કોપ: અવકાશી અભ્યાસ માટે
🧠 Trick: UV in Physics: Photoelectric, Compton, Blackbody

પર્યાવરણીય અસરો

UV કિરણો ઓઝોન સ્તરના ક્ષયમાં મહત્વપૂર્ણ ભૂમિકા ભજવે છે. ઓઝોન (O₃) UV-B અને UV-C કિરણોને શોષીને પૃથ્વી પરના જીવનનું રક્ષણ કરે છે. ઓઝોન સ્તરનું ક્ષયન થવાને કારણે UV-B કિરણોની માત્રા વધી રહી છે, જે માનવ સ્વાસ્થ્ય અને પારિસ્થિતિકી પર નકારાત્મક અસર કરે છે.

  • ઓઝોન સ્તર UV-B અને UV-C શોષે
  • O₃ + UV → O₂ + O (ઓઝોન ક્ષયની પ્રક્રિયા)
  • UV-B વૃદ્ધિ: ત્વચા કેંસર, પાકને નુકસાન
  • પારિસ્થિતિકી પર અસર: પ્લવક જીવોને નુકસાન
  • માનવ સ્વાસ્થ્ય પર અસર: પ્રતિરક્ષાતંત્ર નબળું પડે
🧠 Trick: O₃ = Ozone = Our UV Shield

સુરક્ષા અને ઉપાયો

UV કિરણોના નુકસાનકારક અસરોથી બચવા માટે સનસ્ક્રીન લોશન, ટોપી, અને UV-બ્લોકિંગ કાચનો ઉપયોગ કરવો જોઈએ. સનસ્ક્રીનમાં SPF (Sun Protection Factor) જેવા પદાર્થો હોય છે, જે UV-B કિરણોને અવરોધે છે. લાંબા સમય સુધી સૂર્યપ્રકાશના સંપર્કમાં રહેવાનું ટાળવું જોઈએ, ખાસ કરીને દિવસના 10 વાગ્યાથી 4 વાગ્યા સુધી.

  • સનસ્ક્રીન: SPF UV-B ને અવરોધે
  • UV-બ્લોકિંગ કાચ: આંખોનું રક્ષણ
  • ટોપી અને કાપડ: ત્વચાનું રક્ષણ
  • દિવસના 10–4 વાગ્યા: સૂર્યપ્રકાશનો સૌથી તીવ્ર સમય
  • લાંબા સમય સંપર્ક ટાળો
🧠 Trick: SPF = Save Skin From UV

⚡ Quick Revision

  • UV કિરણોત્સર્ગની તરંગલંબાઈ: 10–400 nm
  • UV-A (315–400 nm), UV-B (280–315 nm), UV-C (100–280 nm)
  • UV-C સૌથી ઊર્જાવાન, ઓઝોન સ્તર દ્વારા શોષાઈ જાય
  • DNA/RNA ને નુકસાન: થાયમિન ડાઇમર બનાવે
  • મેલેનિન ઉત્પાદન વધારે: સનટેન
  • UV-Vis સ્પેક્ટ્રોસ્કોપી: રાસાયણિક વિશ્લેષણ માટે
  • ફોટોઇલેક્ટ્રિક ઇફેક્ટ: UV કિરણો ઇલેક્ટ્રોન બહાર કાઢે
  • ઓઝોન સ્તર UV-B અને UV-C શોષે
  • સનસ્ક્રીન (SPF): UV-B ને અવરોધે
  • UV કિરણો ફ્લોરેસન્સ પેદા કરે
🎯 Exam Tip: NEET માં UV કિરણોના જૈવિક અસરો (DNA નુકસાન, મ્યુટેશન) અને ઓઝોન સ્તરના ક્ષય પર ધ્યાન કેન્દ્રિત કરો. રસાયણશાસ્ત્રમાં UV-Vis સ્પેક્ટ્રોસ્કોપી અને ભૌતિકશાસ્ત્રમાં ફોટોઇલેક્ટ્રિક ઇફેક્ટ મહત્વપૂર્ણ ટોપિક્સ છે.
