ધારિતા

📖 ધારિતા (Capacitance)

પરિભાષા અને મૂળભૂત સંકલ્પના

ધારિતા એ કોઈ વસ્તુની વિદ્યુત આરોપ સંગ્રહવાની ક્ષમતા છે. તે સંકેત C દ્વારા દર્શાવવામાં આવે છે અને ફેરડ (F) એ તેની એકમ છે. ધારિતા એ ચાર્જ (Q) અને વોલ્ટેજ (V) ના ગુણોત્તર તરીકે વ્યાખ્યાયિત થાય છે: C = Q/V.

  • ધારિતા = ચાર્જ / વોલ્ટેજ (C = Q/V)
  • એકમ: ફેરડ (F)
  • 1 ફેરડ = 1 કુલંબ / 1 વોલ્ટ
  • ધારિતા એ સ્કેલર રાશિ છે.
🧠 Trick: C = Q/V → 'ચાર્જ વોલ્ટેજ વાગે'

સમાંતર પ્લેટ ધારિત્ર (Parallel Plate Capacitor)

સમાંતર પ્લેટ ધારિત્રમાં બે પ્લેટ્સ વચ્ચે ડાયલેક્ટ્રિક પદાર્થ હોય છે. ધારિતા પ્લેટના ક્ષેત્રફળ (A), પ્લેટ વચ્ચેના અંતર (d) અને ડાયલેક્ટ્રિક સ્થિરાંક (ε) પર આધારિત છે: C = ε₀εᵣA/d.

  • ધારિતા સૂત્ર: C = ε₀εᵣA/d
  • ε₀ = મુક્ત અવકાશની પરમિટિવિટી (8.85×10⁻¹² F/m)
  • εᵣ = સાપેક્ષ પરમિટિવિટી (ડાયલેક્ટ્રિક સ્થિરાંક)
  • A = પ્લેટનું ક્ષેત્રફળ, d = પ્લેટ વચ્ચેનું અંતર
🧠 Trick: C = ε₀εᵣA/d → 'સી એપી ડી' (C A/d)

ધારિત્રનો સંયોજન (Combination of Capacitors)

ધારિત્રો શ્રેણી અથવા સમાંતર ક્રમમાં જોડી શકાય છે. શ્રેણીમાં સમાન ચાર્જ હોય છે, જ્યારે સમાંતરમાં સમાન વોલ્ટેજ હોય છે. શ્રેણી ધારિતા: 1/C_eq = 1/C₁ + 1/C₂ + ..., સમાંતર ધારિતા: C_eq = C₁ + C₂ + ...

  • શ્રેણી ક્રમ: 1/C_eq = Σ(1/Cᵢ)
  • સમાંતર ક્રમ: C_eq = ΣCᵢ
  • શ્રેણીમાં વોલ્ટેજ વહેંચાય છે, સમાંતરમાં ચાર્જ વહેંચાય છે.
  • શ્રેણી ધારિતા ઓછી હોય છે, સમાંતર ધારિતા વધુ હોય છે.
🧠 Trick: શ્રેણી = 1/C, સમાંતર = C + C

ઊર્જા સંગ્રહ (Energy Stored in a Capacitor)

ધારિત્રમાં સંગ્રહિત ઊર્જા એ ચાર્જ અને વોલ્ટેજના વર્ગ પર આધારિત છે: U = ½CV² = ½QV = ½Q²/C. આ ઊર્જા વિદ્યુત ક્ષેત્રમાં સંગ્રહિત થાય છે.

  • ઊર્જા સૂત્ર: U = ½CV²
  • U = ½QV = ½Q²/C
  • ઊર્જા જ્યારે ધારિત્ર ડિસ્ચાર્જ થાય ત્યારે મુક્ત થાય છે.
  • ઊર્જા જથ્થા એ કેપેસિટરની ધારિતા અને વોલ્ટેજ પર આધારિત છે.
🧠 Trick: U = ½CV² → 'અર્ધ સી વી સ્ક્વેર'

ડાયલેક્ટ્રિક પદાર્થો અને ધારિતા (Dielectrics and Capacitance)

ડાયલેક્ટ્રિક પદાર્થો ધારિતા વધારે છે કારણ કે તેઓ વિદ્યુત ક્ષેત્રને નબળું પાડે છે. ડાયલેક્ટ્રિક સ્થિરાંક (εᵣ) એ પદાર્થની ક્ષમતા દર્શાવે છે કે તે વિદ્યુત ક્ષેત્રને કેટલું નબળું પાડે છે.

  • ડાયલેક્ટ્રિક સ્થિરાંક (εᵣ) > 1
  • ડાયલેક્ટ્રિક પદાર્થો ધારિતા વધારે છે.
  • પોલર અને નોન-પોલર ડાયલેક્ટ્રિક્સ
  • ડાયલેક્ટ્રિક સ્થિરાંક વગરની ધારિતા: C₀ = ε₀A/d

ધારિતા અને વર્તમાન વિદ્યુત (Capacitance and Current Electricity)

ધારિત્રમાં વર્તમાન વિદ્યુત એ ચાર્જના ફેરફારના દર તરીકે વ્યાખ્યાયિત થાય છે: I = dQ/dt. ધારિત્રમાં વોલ્ટેજ બદલાય ત્યારે વર્તમાન પેદા થાય છે.

  • I = dQ/dt = C(dV/dt)
  • ધારિત્ર AC સર્કિટમાં પ્રતિક્રિયા આપે છે.
  • ધારિતા પ્રતિક્રિયા (X_C) = 1/(2πfC)
  • ધારિત્ર DC સર્કિટમાં સ્થિર અવસ્થામાં વર્તમાન રોકે છે.
🧠 Trick: I = C(dV/dt) → 'આઈ એસી ડીવી ડીટી'

  • ધારિતા = ચાર્જ / વોલ્ટેજ (C = Q/V)
  • સમાંતર પ્લેટ ધારિત્ર: C = ε₀εᵣA/d
  • શ્રેણી ધારિતા: 1/C_eq = Σ(1/Cᵢ)
  • સમાંતર ધારિતા: C_eq = ΣCᵢ
  • ઊર્જા સંગ્રહ: U = ½CV²
  • ડાયલેક્ટ્રિક સ્થિરાંક (εᵣ) ધારિતા વધારે છે.
  • ધારિત્રમાં વર્તમાન: I = C(dV/dt)
  • ધારિતા પ્રતિક્રિયા: X_C = 1/(2πfC)
  • ધારિતા એ સ્કેલર રાશિ છે.
  • 1 ફેરડ = 1 કુલંબ / 1 વોલ્ટ
🎯 Exam Tip: ધારિતા અને ધારિત્રના સૂત્રો યાદ રાખો, ખાસ કરીને સમાંતર પ્લેટ ધારિત્ર અને સંયોજનના સૂત્રો. પરીક્ષામાં આ સૂત્રોને લાગુ પાડવાના પ્રશ્નો વારંવાર પૂછાય છે.
