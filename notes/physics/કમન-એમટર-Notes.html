કોમન એમિટર

📖 કોમન એમિટર (Common Emitter Configuration in BJT)

પરિચય અને મૂળભૂત રચના

કોમન એમિટર કૉન્ફિગરેશનમાં, એમિટર ટર્મિનલ બંને ઇનપુટ અને આઉટપુટ સર્કિટ માટે સામાન્ય (કોમન) હોય છે. બેઝ-એમિટર જંકશન ઇનપુટ તરીકે અને કલેક્ટર-એમિટર જંકશન આઉટપુટ તરીકે કામ કરે છે. આ કૉન્ફિગરેશનમાં, વોલ્ટેજ અને કરંટ ગેઇન બંને ઉચ્ચ હોય છે, જે તેને એમ્પ્લિફાયર તરીકે ઉપયોગી બનાવે છે.

  • એમિટર બંને ઇનપુટ અને આઉટપુટ માટે સામાન્ય ટર્મિનલ છે
  • બેઝ-એમિટર જંકશન: ઇનપુટ, કલેક્ટર-એમિટર જંકશન: આઉટપુટ
  • વોલ્ટેજ ગેઇન (A_v) અને કરંટ ગેઇન (A_i) ઉચ્ચ હોય છે
  • NEETમાં સૌથી વારંવાર પૂછાતું કૉન્ફિગરેશન
🧠 Trick: CE = Common Emitter = Current & Voltage Gain High (CVGH)

કરંટ ગેઇન (β અને α)

કોમન એમિટરમાં, કરંટ ગેઇન β (beta) = I_c / I_b છે, જ્યાં I_c = કલેક્ટર કરંટ અને I_b = બેઝ કરંટ. આલ્ફા (α) = I_c / I_e છે, જ્યાં I_e = એમિટર કરંટ. β અને α વચ્ચેનો સંબંધ છે: β = α / (1 - α). NEETમાં β નું મૂલ્ય સામાન્ય રીતે 50 થી 200 વચ્ચે હોય છે.

  • β (DC કરંટ ગેઇન) = I_c / I_b
  • α (કલેક્ટર-એમિટર કરંટ રેશિયો) = I_c / I_e
  • β = α / (1 - α)
  • β નું મૂલ્ય સામાન્ય રીતે 50-200 હોય છે
🧠 Trick: β = બેઝ કરંટથી કલેક્ટર કરંટનો ગુણોત્તર

વોલ્ટેજ ગેઇન (A_v)

વોલ્ટેજ ગેઇન (A_v) = -β * (R_c / R_in) છે, જ્યાં R_c = કલેક્ટર રેઝિસ્ટર અને R_in = ઇનપુટ રેઝિસ્ટર. નેગેટિવ સાઇન ફેઝ રિવર્સલ દર્શાવે છે. કોમન એમિટર એમ્પ્લિફાયરમાં, આઉટપુટ વોલ્ટેજ ઇનપુટ વોલ્ટેજ કરતાં 180° આઉટ ઓફ ફેઝ હોય છે.

  • A_v = -β * (R_c / R_in)
  • નેગેટિવ સાઇન = ફેઝ રિવર્સલ (180° આઉટ ઓફ ફેઝ)
  • R_c = કલેક્ટર રેઝિસ્ટર, R_in = ઇનપુટ રેઝિસ્ટર
  • વોલ્ટેજ ગેઇન ઉચ્ચ હોય છે
🧠 Trick: CEમાં ફેઝ રિવર્સલ: ઇનપુટ ↑ → આઉટપુટ ↓

ઇનપુટ અને આઉટપુટ રેઝિસ્ટન્સ

કોમન એમિટરમાં, ઇનપુટ રેઝિસ્ટન્સ (R_in) = R_b || r_be છે, જ્યાં R_b = બેઝ રેઝિસ્ટર અને r_be = બેઝ-એમિટર જંકશન રેઝિસ્ટન્સ. આઉટપુટ રેઝિસ્ટન્સ (R_o) ≈ R_c છે, જ્યાં R_c = કલેક્ટર રેઝિસ્ટર. ઇનપુટ રેઝિસ્ટન્સ મધ્યમ અને આઉટપુટ રેઝિસ્ટન્સ ઉચ્ચ હોય છે.

  • R_in = R_b || r_be (મધ્યમ મૂલ્ય)
  • R_o ≈ R_c (ઉચ્ચ મૂલ્ય)
  • r_be = બેઝ-એમિટર જંકશન રેઝિસ્ટન્સ
  • NEETમાં R_in અને R_o ની ગણતરી પૂછાય છે
🧠 Trick: R_in મધ્યમ, R_o ઉચ્ચ

એપ્લિકેશન્સ

કોમન એમિટર કૉન્ફિગરેશનનો ઉપયોગ મુખ્યત્વે એમ્પ્લિફાયર તરીકે થાય છે, જેમ કે ઓડિયો એમ્પ્લિફાયર, રેડિયો ફ્રીક્વન્સી એમ્પ્લિફાયર, અને સિગ્નલ એમ્પ્લિફિકેશન. તેનો ઉપયોગ સ્વિચ તરીકે પણ થાય છે, પરંતુ એમ્પ્લિફાયર તરીકે તે વધુ કાર્યક્ષમ છે.

  • મુખ્ય ઉપયોગ: એમ્પ્લિફાયર (ઓડિયો, RF, સિગ્નલ)
  • સ્વિચ તરીકે પણ ઉપયોગી
  • વોલ્ટેજ અને કરંટ ગેઇન બંને ઉચ્ચ હોવાથી એમ્પ્લિફિકેશન માટે આદર્શ
  • NEETમાં એમ્પ્લિફાયર એપ્લિકેશન પર ભાર મૂકાય છે
🧠 Trick: CE = એમ્પ્લિફાયર માટે બેસ્ટ

લાભ અને મર્યાદાઓ

કોમન એમિટરના મુખ્ય લાભો છે: ઉચ્ચ વોલ્ટેજ અને કરંટ ગેઇન, મધ્યમ ઇનપુટ રેઝિસ્ટન્સ, અને ઉચ્ચ આઉટપુટ રેઝિસ્ટન્સ. મર્યાદાઓમાં ફેઝ રિવર્સલ અને તાપમાન પર નિર્ભરતા શામેલ છે. NEETમાં લાભો પર વધુ ભાર મૂકાય છે.

  • લાભ: ઉચ્ચ A_v અને A_i, મધ્યમ R_in, ઉચ્ચ R_o
  • મર્યાદા: ફેઝ રિવર્સલ, તાપમાન પર નિર્ભરતા
  • NEETમાં લાભો પર વધુ પ્રશ્નો પૂછાય છે
  • એમ્પ્લિફાયર તરીકે સર્વોત્ત
🧠 Trick: CE: ગેઇન ઉચ્ચ, ફેઝ ઉલટ

NEET માટે મહત્વના સૂત્રો

કોમન એમિટર માટે મહત્વના સૂત્રો: β = I_c / I_b, α = I_c / I_e, β = α / (1 - α), A_v = -β * (R_c / R_in). આ સૂત્રો NEETમાં વારંવાર પૂછાય છે અને ગણતરીઓ માટે આવશ્યક છે.

  • β = I_c / I_b
  • α = I_c / I_e
  • β = α / (1 - α)
  • A_v = -β * (R_c / R_in)
🧠 Trick: β, α, A_v: ગેઇન માટે ત્રણ મહત્વના સૂત્ર

⚡ Quick Revision

  • કોમન એમિટરમાં એમિટર સામાન્ય ટર્મિનલ છે
  • β = I_c / I_b (કરંટ ગેઇન)
  • A_v = -β * (R_c / R_in) (વોલ્ટેજ ગેઇન)
  • ફેઝ રિવર્સલ થાય છે (180° આઉટ ઓફ ફેઝ)
  • R_in = R_b || r_be (મધ્યમ), R_o ≈ R_c (ઉચ્ચ)
  • મુખ્ય ઉપયોગ: એમ્પ્લિફાયર
  • β = α / (1 - α)
  • NEETમાં સૌથી વારંવાર પૂછાતું કૉન્ફિગરેશન
  • ગેઇન ઉચ્ચ હોવાથી એમ્પ્લિફિકેશન માટે બેસ્ટ
  • તાપમાન પર નિર્ભરતા મર્યાદા છે
🎯 Exam Tip: NEETમાં કોમન એમિટર પરના પ્રશ્નો મુખ્યત્વે ગેઇન (β, α, A_v) અને ફેઝ રિવર્સલ પર કેન્દ્રિત હોય છે. સૂત્રો યાદ રાખો અને ગણતરીઓની પ્રેક્ટિસ કરો.
