વિદ્યુત પ્રવાહ

📖 વિદ્યુત પ્રવાહ

વિદ્યુત પ્રવાહની વ્યાખ્યા અને એકમ

વિદ્યુત પ્રવાહ એ વાહકમાં વહેતા વિદ્યુત ભારના દરને કહે છે. તેને I વડે દર્શાવવામાં આવે છે અને એમ્પિયર (A) એ તેનો SI એકમ છે. 1 એમ્પિયર = 1 કૂલોમ/સેકન્ડ. પ્રવાહની દિશા સંમેલન અનુસાર ધન ભારની ગતિની દિશામાં લેવામાં આવે છે.

  • વિદ્યુત પ્રવાહ (I) = ΔQ/Δt
  • SI એકમ: એમ્પિયર (A)
  • 1 A = 1 C/s
  • પ્રવાહની દિશા: ધન ભારની ગતિની દિશા
🧠 Trick: I = Q/t → 'I' for Intensity, 'Q' for Quantity, 't' for time

વિદ્યુત પ્રવાહના પ્રકાર

વિદ્યુત પ્રવાહ બે પ્રકારના હોય છે: (1) દિશા બદલતો પ્રવાહ (AC) - જેમાં પ્રવાહની દિશા અને પરિમાણ સમય સાથે બદલાય છે. (2) દિશા ન બદલતો પ્રવાહ (DC) - જેમાં પ્રવાહની દિશા સ્થિર રહે છે, પરંતુ પરિમાણ બદલાઈ શકે છે.

  • AC: દિશા અને પરિમાણ બદલાય છે (જેવા કે ઘરગથ્થુ વીજળી)
  • DC: દિશા સ્થિર, પરિમાણ બદલાઈ શકે (જેવા કે બેટરી)
  • ACનો આવર્તન (f) હર્ટઝ (Hz)માં માપવામાં આવે છે
  • DCનો સ્રોત: બેટરી, સોલર સેલ
🧠 Trick: AC = Alternating Current (આગળ-પાછળ), DC = Direct Current (સીધો)

ઓહ્મનો નિયમ

ઓહ્મનો નિયમ કહે છે કે, સ્થિર તાપમાને, વાહકમાંથી વહેતા પ્રવાહ (I) અને તેના બે છેડા વચ્ચેના વોલ્ટેજ (V) વચ્ચે સીધો અનુપાત હોય છે. V = IR, જ્યાં R એ વાહકનો પ્રતિરોધ છે.

  • V ∝ I (સ્થિર T માટે)
  • V = IR
  • R = વોલ્ટેજ/પ્રવાહ (ઓહ્મ, Ω)
  • ઓહ્મનો નિયમ લાગુ પડે છે: ધાતુઓ, મિશ્ર ધાતુઓ (સ્થિર T પર)
🧠 Trick: VIR → Voltage = Current × Resistance

વિદ્યુત પ્રતિરોધ અને પ્રતિરોધકતા

પ્રતિરોધ (R) એ વાહકની વિદ્યુત પ્રવાહને અવરોધવાની ક્ષમતા છે. પ્રતિરોધકતા (ρ) એ પદાર્થની આંતરિક ગુણધર્મ છે જે પ્રતિરોધ પર આધાર રાખે છે. R = ρL/A, જ્યાં L = લંબાઈ, A = કાટખૂણો.

  • R ∝ L (લંબાઈ)
  • R ∝ 1/A (કાટખૂણો)
  • ρ = RA/L (પ્રતિરોધકતા)
  • ρનો એકમ: ઓહ્મ-મીટર (Ω·m)
  • તાપમાન વધે ત્યારે ધાતુઓનો R વધે છે
🧠 Trick: R = ρL/A → 'R' for Resistance, 'ρ' for rho (resistivity), 'L' for Length, 'A' for Area

વિદ્યુત શક્તિ અને ઉર્જા

વિદ્યુત શક્તિ (P) એ એકમ સમયમાં ખર્ચાતી વિદ્યુત ઉર્જાનો દર છે. P = VI = I²R = V²/R. વિદ્યુત ઉર્જા (E) = P × t = VIt. શક્તિનો એકમ વોટ (W) અને ઉર્જાનો એકમ જૂલ (J) છે.

  • P = VI (વોલ્ટ × એમ્પિયર)
  • P = I²R (પ્રવાહનો વર્ગ × પ્રતિરોધ)
  • P = V²/R (વોલ્ટેજનો વર્ગ / પ્રતિરોધ)
  • 1 વોટ = 1 જૂલ/સેકન્ડ
  • વિદ્યુત બિલમાં ઉર્જાનો એકમ: કિલોવોટ-કલાક (kWh)
🧠 Trick: P = VI → Power = Voltage × Current

કિરચોફના નિયમો

કિરચોફના પ્રથમ નિયમ (જંકશન નિયમ): કોઈપણ જંકશન પર આવતા પ્રવાહોનો સરવાળ = જતા પ્રવાહોનો સરવાળ. કિરચોફના દ્વિતીય નિયમ (લૂપ નિયમ): કોઈપણ બંધ લૂપમાં વોલ્ટેજનો બીજગણિતીય સરવાળ શૂન્ય હોય છે.

  • જંકશન નિયમ: ΣI_in = ΣI_out
  • લૂપ નિયમ: ΣV = 0 (બંધ લૂપ માટે)
  • લાગુ પડે છે: જટિલ પરિપથો માટે
  • વોલ્ટેજના ચિહ્નો ધ્યાનમાં લેવા
🧠 Trick: KVL: કુલ વોલ્ટેજ = 0 (લૂપમાં), KCL: કુલ પ્રવાહ = 0 (જંકશન પર)

વિદ્યુત પ્રવાહના ઉષ્મીય પ્રભાવ

જ્યારે વિદ્યુત પ્રવાહ વાહકમાંથી પસાર થાય છે, ત્યારે ઉર્જા ઉષ્મા રૂપે વ્યય પામે છે. જૂલનો ઉષ્મીય નિયમ કહે છે કે, H = I²Rt, જ્યાં H = ઉત્પન્ન થતી ઉષ્મા, I = પ્રવાહ, R = પ્રતિરોધ, t = સમય.

  • H = I²Rt (જૂલનો નિયમ)
  • ઉષ્માનો એકમ: જૂલ (J) અથવા કેલોરી (cal)
  • 1 cal = 4.2 J
  • લાગુ પડે છે: હીટર, બલ્બ, ફ્યુઝ વગેરેમાં
🧠 Trick: H = I²RT → Heat = Current² × Resistance × Time

⚡ Quick Revision

  • વિદ્યુત પ્રવાહ (I) = ΔQ/Δt, એકમ: એમ્પિયર (A)
  • AC પ્રવાહ દિશા બદલે, DC પ્રવાહ દિશા સ્થિર
  • ઓહ્મનો નિયમ: V = IR (સ્થિર T પર)
  • પ્રતિરોધ (R) = ρL/A, ρ = પ્રતિરોધકતા
  • વિદ્યુત શક્તિ (P) = VI = I²R = V²/R
  • કિરચોફના નિયમો: જંકશન (ΣI_in = ΣI_out) અને લૂપ (ΣV = 0)
  • જૂલનો નિયમ: H = I²Rt (ઉષ્મીય પ્રભાવ)
  • 1 kWh = 3.6 × 10⁶ J (વિદ્યુત ઉર્જા)
  • તાપમાન વધે ત્યારે ધાતુઓનો પ્રતિરોધ વધે છે
  • વિદ્યુત પ્રવાહની દિશા: ધન ભારની ગતિની દિશા
🎯 Exam Tip: NEETમાં વિદ્યુત પ્રવાહના પ્રશ્નો માટે ઓહ્મનો નિયમ, કિરચોફના નિયમો અને જૂલનો નિયમ યાદ રાખો. પરિપથના પ્રશ્નો માટે સીરીઝ અને પેરલલ કનેક્શનનું ગણિત તૈયાર કરો.
