પરમાણુ કેન્દ્રક

📖 પરમાણુ કેન્દ્રક

પરિભાષા અને રચના

પરમાણુ કેન્દ્રક એ પરમાણુનો કેન્દ્રીય ભાગ છે, જેમાં પ્રોટોન અને ન્યુટ્રોન હાજર હોય છે. તે પરમાણુના દ્રવ્યમાનનો 99.9% ભાગ ધરાવે છે. કેન્દ્રકનો વ્યાસ લગભગ 10^-15 m (ફેમ્ટોમીટર) હોય છે. પ્રોટોન ધન વીજભાર ધરાવે છે જ્યારે ન્યુટ્રોન વીજભાર રહિત હોય છે.

  • પરમાણુ કેન્દ્રક = પ્રોટોન + ન્યુટ્રોન
  • પરમાણુ દ્રવ્યમાનનો 99.9% ભાગ કેન્દ્રકમાં
  • પ્રોટોનનો વીજભાર = +1.6 × 10^-19 C
  • ન્યુટ્રોનનો દ્રવ્યમાન ≈ પ્રોટોનના દ્રવ્યમાન જેટલો
  • કેન્દ્રકનો વ્યાસ ≈ 10^-15 m
🧠 Trick: પ્રો + ન્યૂ = કેન્દ્રક (પ્રોટોન + ન્યુટ્રોન)

કેન્દ્રકીય બળ

કેન્દ્રકમાં પ્રોટોન વચ્ચેના વિકર્ષણ બળને સંતુલિત રાખવા માટે મજબૂત કેન્દ્રકીય બળ કાર્ય કરે છે. આ બળ પ્રોટોન અને ન્યુટ્રોન વચ્ચે કાર્ય કરે છે અને અત્યંત ટૂંકા અંતર (≈1 fm) સુધી મર્યાદિત હોય છે. તે ચાર જેટલું મજબૂત બળ છે.

  • મજબૂત કેન્દ્રકીય બળ = પ્રોટોન-પ્રોટોન, પ્રોટોન-ન્યુટ્રોન, ન્યુટ્રોન-ન્યુટ્રોન વચ્ચે
  • વિકર્ષણ બળને સંતુલિત કરે છે
  • અત્યંત ટૂંકા અંતર (1 fm) સુધી કાર્ય કરે
  • ચાર જેટલું મજબૂત બળ
  • કેન્દ્રકીય બળ વીજભાર પર આધારિત નથી

પરમાણુ ક્રમાંક અને દ્રવ્યમાન ક્રમાંક

પરમાણુ ક્રમાંક (Z) એ પ્રોટોનની સંખ્યા દર્શાવે છે, જ્યારે દ્રવ્યમાન ક્રમાંક (A) એ પ્રોટોન અને ન્યુટ્રોનની કુલ સંખ્યા દર્શાવે છે. આઇસોટોપ્સમાં Z સમાન હોય છે પરંતુ A ભિન્ન હોય છે. આઇસોબારમાં A સમાન હોય છે પરંતુ Z ભિન્ન હોય છે.

  • Z = પ્રોટોનની સંખ્યા
  • A = પ્રોટોન + ન્યુટ્રોનની સંખ્યા
  • આઇસોટોપ્સ: સમાન Z, ભિન્ન A (ઉદા. Cl-35, Cl-37)
  • આઇસોબાર: સમાન A, ભિન્ન Z (ઉદા. Ar-40, Ca-40)
  • ન્યુટ્રોનની સંખ્યા = A - Z
🧠 Trick: Z = પી (પ્રોટોન), A = પી + ન્યૂ (પ્રોટોન + ન્યુટ્રોન)

કેન્દ્રકીય પ્રતિક્રિયાઓ

કેન્દ્રકીય પ્રતિક્રિયાઓમાં કેન્દ્રકનું વિભાજન અથવા સંલયન થાય છે, જેમાં મોટી માત્રામાં ઊર્જા મુક્ત થાય છે. વિખંડનમાં ભારે કેન્દ્રકનો તૂટીને હલકા કેન્દ્રકમાં વિભાજન થાય છે (ઉદા. યુરેનિયમ-235). સંલયનમાં હલકા કેન્દ્રક એકબીજા સાથે જોડાઈને ભારે કેન્દ્રક બનાવે છે (ઉદા. સૂર્યમાં હાઇડ્રોજનનું હિલિયમમાં રૂપાંતર).

  • વિખંડન: ભારે કેન્દ્રક → હલકા કેન્દ્રક + ઊર્જા
  • સંલયન: હલકા કેન્દ્રક → ભારે કેન્દ્રક + ઊર્જા
  • E = mc² (આઇન્સ્ટાઇનનો સમીકરણ)
  • વિખંડનમાં ન્યુટ્રોનનો ઉપયોગ થાય છે
  • સંલયન માટે ઉચ્ચ તાપમાન જરૂરી છે
🧠 Trick: વિખંડન = તૂટવું, સંલયન = જોડાવું

કેન્દ્રકીય સ્થિરતા

કેન્દ્રકની સ્થિરતા ન્યુટ્રોન-પ્રોટોન ગુણોત્તર (n/p) પર આધારિત છે. હલકા પરમાણુઓ માટે n/p ≈ 1 હોય છે, જ્યારે ભારે પરમાણુઓ માટે n/p > 1 હોય છે. જો n/p ગુણોત્તર અસંતુલિત હોય તો કેન્દ્રક રેડિયોએક્ટિવ હોય છે અને ક્ષય પામે છે.

  • હલકા પરમાણુઓ: n/p ≈ 1 (ઉદા. He-4: 2n, 2p)
  • ભારે પરમાણુઓ: n/p > 1 (ઉદા. U-238: 146n, 92p)
  • રેડિયોએક્ટિવ ક્ષય: α, β, γ કિરણોત્સર્ગ
  • α-ક્ષય: He-4 કેન્દ્રક ઉત્સર્જન
  • β-ક્ષય: ઇલેક્ટ્રોન અથવા પોઝીટ્રોન ઉત્સર્જન
🧠 Trick: n/p > 1 → ભારે, n/p ≈ 1 → હલકા

કેન્દ્રકીય ઊર્જા

કેન્દ્રકીય બંધન ઊર્જા એ કેન્દ્રકને તોડવા માટે જરૂરી ઊર્જા છે. તે પ્રોટોન અને ન્યુટ્રોનને એકબીજા સાથે બાંધી રાખે છે. બંધન ઊર્જા પ્રતિ ન્યુક્લિઓન જેટલી વધુ હોય, તે કેન્દ્રક વધુ સ્થિર હોય છે. લોખંડ (Fe-56) સૌથી વધુ બંધન ઊર્જા ધરાવે છે.

  • બંધન ઊર્જા = કેન્દ્રકને તોડવા જરૂરી ઊર્જા
  • પ્રતિ ન્યુક્લિઓન બંધન ઊર્જા જેટલી વધુ, તે કેન્દ્રક વધુ સ્થિર
  • Fe-56 સૌથી સ્થિર કેન્દ્રક
  • બંધન ઊર્જા = Δmc² (દ્રવ્યમાનનો તફાવત)
  • હલકા અને મધ્યમ વજનના કેન્દ્રકો સંલયન દ્વારા ઊર્જા મુક્ત કરે છે
🧠 Trick: Fe-56 = સ્થિરતાનું શિખર

રેડિયોએક્ટિવતા અને અર્ધ-આયુષ્ય

રેડિયોએક્ટિવતા એ સ્વયંસ્ફૂરિત રીતે ક્ષય પામવાની પ્રક્રિયા છે. અર્ધ-આયુષ્ય એ સમયગાળો છે જેમાં રેડિયોએક્ટિવ પદાર્થની માત્રા અડધી થઈ જાય છે. તે પદાર્થની પ્રકાર અને પર્યાવરણ પર આધારિત નથી. રેડિયોએક્ટિવ ક્ષય ના નિયંત્રણ થઈ શકે.

  • રેડિયોએક્ટિવતા = સ્વયંસ્ફૂરિત ક્ષય
  • અર્ધ-આયુષ્ય (t₁/₂) = માત્રા અડધી થવા માટેનો સમય
  • t₁/₂ પદાર્થની પ્રકાર પર આધારિત
  • રેડિયોએક્ટિવ ક્ષય ના નિયંત્રણ થઈ શકે
  • ઉદા. C-14નું t₁/₂ = 5730 વર્ષ (રેડિયોકાર્બન ડેટિંગ)
🧠 Trick: અર્ધ-આયુષ્ય = અડધું થવાનો સમય

⚡ Quick Revision

  • પરમાણુ કેન્દ્રક = પ્રોટોન + ન્યુટ્રોન
  • પ્રોટોન: +1.6 × 10^-19 C, ન્યુટ્રોન: વીજભાર રહિત
  • Z = પ્રોટોનની સંખ્યા, A = પ્રોટોન + ન્યુટ્રોન
  • મજબૂત કેન્દ્રકીય બળ વિકર્ષણ બળને સંતુલિત કરે
  • વિખંડન: ભારે → હલકા + ઊર્જા, સંલયન: હલકા → ભારે + ઊર્જા
  • n/p ગુણોત્તર સ્થિરતા નક્કી કરે
  • Fe-56 સૌથી સ્થિર કેન્દ્રક
  • રેડિયોએક્ટિવ ક્ષય: α, β, γ કિરણોત્સર્ગ
  • અર્ધ-આયુષ્ય = માત્રા અડધી થવા માટેનો સમય
  • E = mc² (કેન્દ્રકીય ઊર્જા સમીકરણ)
🎯 Exam Tip: NEETમાં પરમાણુ કેન્દ્રકના પ્રશ્નો મુખ્યત્વે Z, A, આઇસોટોપ્સ, આઇસોબાર, રેડિયોએક્ટિવ ક્ષય અને બંધન ઊર્જા પર આધારિત હોય છે. સૂત્રો અને પરિભાષાઓ યાદ રાખો અને ઉદાહરણો સાથે પ્રેક્ટિસ કરો.
