બાયોટ-સાવર્ટ નિયમ

📖 બાયોટ-સાવર્ટ નિયમ

નિયમનો પરિચય

બાયોટ-સાવર્ટ નિયમ વીજળીના પ્રવાહ દ્વારા ઉત્પન્ન થતાં ચુંબકીય ક્ષેત્રની તીવ્રતાનું મૂલ્યાંકન કરે છે. આ નિયમ અનુસાર, વીજપ્રવાહ ધરાવતા તારના અતિ સૂક્ષ્મ લંબાઈના ટુકડા દ્વારા કોઈ એક બિંદુ પર ઉત્પન્ન થતું ચુંબકીય ક્ષેત્ર, પ્રવાહના પરિમાણ, લંબાઈના ટુકડા અને દિશા પર આધાર રાખે છે.

  • વીજપ્રવાહ દ્વારા ચુંબકીય ક્ષેત્ર ઉત્પન્ન થાય છે
  • નિયમ ફ્રેન્ચ વૈજ્ઞાનિક જીન-બાપ્ટિસ્ટ બાયોટ અને ફેલિક્સ સાવર્ટના નામ પરથી પડ્યો છે
  • આ નિયમ એમ્પિયરના નિયમનો પાયો છે

ગાણિતિક સ્વરૂપ

બાયોટ-સાવર્ટ નિયમનું ગાણિતિક સ્વરૂપ છે: dB = (μ₀/4π) * (I * dl * sinθ) / r². અહીં, dB એ ચુંબકીય ક્ષેત્રનો અતિ સૂક્ષ્મ ભાગ છે, I એ વીજપ્રવાહ, dl એ તારનો લંબાઈનો ટુકડો, θ એ dl અને r વચ્ચેનો ખૂણો, r એ બિંદુથી તારના ટુકડા સુધીનું અંતર છે, અને μ₀ એ મુક્ત અવકાશની ચુંબકીય પારગમ્યતા છે.

  • dB = (μ₀/4π) * (I * dl * sinθ) / r²
  • μ₀ = 4π × 10⁻⁷ T·m/A (મુક્ત અવકાશની ચુંબકીય પારગમ્યતા)
  • θ એ dl અને r વચ્ચેનો ખૂણો છે
🧠 Trick: બાયોટ-સાવર્ટ = બી એસ (B-S) → dB = (μ₀/4π) * (I * dl * sinθ) / r²

નિયમના અર્થઘટન

નિયમ અનુસાર, ચુંબકીય ક્ષેત્રની દિશા વીજપ્રવાહની દિશા અને બિંદુની સ્થિતિ પર આધાર રાખે છે. ચુંબકીય ક્ષેત્રની દિશા જમણા હાથના નિયમ (Right-Hand Rule) દ્વારા નક્કી કરી શકાય છે, જેમાં અંગૂઠો વીજપ્રવાહની દિશા તરફ અને આંગળીઓ ચુંબકીય ક્ષેત્રની દિશા તરફ રાખવામાં આવે છે.

  • ચુંબકીય ક્ષેત્રની દિશા જમણા હાથના નિયમથી નક્કી કરો
  • વીજપ્રવાહની દિશા અને બિંદુની સ્થિતિ ચુંબકીય ક્ષેત્રની દિશા નક્કી કરે છે
  • જમણા હાથના નિયમમાં અંગૂઠો Iની દિશા તરફ અને આંગળીઓ Bની દિશા તરફ
🧠 Trick: જમણો હાથ → અંગૂઠો I, આંગળીઓ B

અનુપ્રયોગો

બાયોટ-સાવર્ટ નિયમનો ઉપયોગ વીજળીના પ્રવાહ દ્વારા ઉત્પન્ન થતાં ચુંબકીય ક્ષેત્રની ગણતરી કરવા માટે થાય છે. આ નિયમનો ઉપયોગ સોલેનોઇડ, ટોરોઇડ અને વીજળીના મોટરમાં ચુંબકીય ક્ષેત્રની ગણતરી કરવા માટે થાય છે.

  • સોલેનોઇડમાં ચુંબકીય ક્ષેત્રની ગણતરી
  • ટોરોઇડમાં ચુંબકીય ક્ષેત્રની ગણતરી
  • વીજળીના મોટરમાં ચુંબકીય ક્ષેત્રની ગણતરી

એમ્પિયરના નિયમ સાથે સંબંધ

બાયોટ-સાવર્ટ નિયમ એ એમ્પિયરના નિયમનો પાયો છે. એમ્પિયરનો નિયમ કહે છે કે, બંધ લૂપમાં વીજપ્રવાહ દ્વારા ઉત્પન્ન થતું ચુંબકીય ક્ષેત્રનો રેખા સંકલન, μ₀ ગુણ્યા લૂપ દ્વારા પસાર થતાં વીજપ્રવાહના કુલ પરિમાણ જેટલો હોય છે.

  • એમ્પિયરનો નિયમ: ∮B·dl = μ₀ * I_enc
  • બાયોટ-સાવર્ટ નિયમ એ એમ્પિયરના નિયમનો પાયો છે
  • I_enc એ લૂપ દ્વારા પસાર થતો કુલ વીજપ્રવાહ છે

મહત્વના ઉદાહરણો

બાયોટ-સાવર્ટ નિયમનો ઉપયોગ કરીને, અનંત લંબાઈના તાર દ્વારા ઉત્પન્ન થતાં ચુંબકીય ક્ષેત્રની તીવ્રતા B = (μ₀ * I) / (2π * r) કાઢી શકાય છે. આ ઉપરાંત, વર્તુળાકાર લૂપના કેન્દ્ર પર ચુંબકીય ક્ષેત્રની તીવ્રતા B = (μ₀ * I) / (2 * R) કાઢી શકાય છે.

  • અનંત તાર માટે: B = (μ₀ * I) / (2π * r)
  • વર્તુળાકાર લૂપના કેન્દ્ર પર: B = (μ₀ * I) / (2 * R)
  • આ સૂત્રો બાયોટ-સાવર્ટ નિયમ પર આધારિત છે
