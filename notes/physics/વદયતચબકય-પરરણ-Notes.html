વિદ્યુતચુંબકીય પ્રેરણ

📖 વિદ્યુતચુંબકીય પ્રેરણ

ફેરેડેનો પ્રેરણાનો નિયમ

જ્યારે કોઈ વાહક લૂપ અથવા કોઈલમાંથી પસાર થતો ચુંબકીય ફ્લક્સ બદલાય છે, ત્યારે તે લૂપમાં વિદ્યુત વાહક બળ (emf) પ્રેરિત થાય છે. આ ઘટનાને વિદ્યુતચુંબકીય પ્રેરણ કહેવાય છે. પ્રેરિત emfની દિશા લેન્ઝના નિયમ દ્વારા નક્કી કરવામાં આવે છે.

  • ફેરેડેનો બીજો નિયમ: પ્રેરિત emf = -dΦ/dt (Φ = ચુંબકીય ફ્લક્સ)
  • લૂપમાં પ્રેરિત emf તે ફ્લક્સના બદલાવના દરના સમપ્રમાણમાં હોય છે.
  • નેગેટિવ ચિહ્ન લેન્ઝના નિયમને દર્શાવે છે.
🧠 Trick: ફેરેડે = ફ્લક્સ બદલાય તો emf ઉત્પન્ન થાય

લેન્ઝનો નિયમ

પ્રેરિત વીજપ્રવાહની દિશા એવી હોય છે કે જે તેના કારણ ભૂત ચુંબકીય ફ્લક્સના બદલાવનો વિરોધ કરે છે. આ નિયમ ઉર્જાના સંરક્ષણના સિદ્ધાંત પર આધારિત છે.

  • પ્રેરિત પ્રવાહની દિશા મૂળ ફ્લક્સના બદલાવનો વિરોધ કરે છે.
  • ઉદાહરણ: જ્યારે ચુંબક કોઈલ તરફ ધકેલવામાં આવે ત્યારે કોઈલમાં પ્રેરિત પ્રવાહ એવી દિશામાં વહે છે કે જે ચુંબકને પુશ કરે.
  • લેન્ઝનો નિયમ ફેરેડેના નિયમનો પૂરક છે.
🧠 Trick: લેન્ઝ = લડો, વિરોધ કરો

સ્વ-પ્રેરણ (Self Induction)

જ્યારે કોઈ કોઈલમાં વીજપ્રવાહ બદલાય છે, ત્યારે તે જ કોઈલમાં ચુંબકીય ફ્લક્સ બદલાય છે, જેથી તે કોઈલમાં જ પ્રેરિત emf ઉત્પન્ન થાય છે. આ ઘટનાને સ્વ-પ્રેરણ કહેવાય છે.

  • સ્વ-પ્રેરિત emf = -L * (dI/dt), જ્યાં L = સ્વ-પ્રેરણાંક (Self Inductance)
  • Lનું એકમ હેનરી (H) છે.
  • સ્વ-પ્રેરણ કોઈલની ચુંબકીય ઉર્જા સંગ્રહિત કરવાની ક્ષમતા દર્શાવે છે.

પરસ્પર પ્રેરણ (Mutual Induction)

જ્યારે બે કોઈલો એકબીજા નજીક હોય છે અને એક કોઈલમાં વીજપ્રવાહ બદલાય છે, ત્યારે બીજી કોઈલમાં ચુંબકીય ફ્લક્સ બદલાય છે, જેથી બીજી કોઈલમાં પ્રેરિત emf ઉત્પન્ન થાય છે. આ ઘટનાને પરસ્પર પ્રેરણ કહેવાય છે.

  • પરસ્પર પ્રેરિત emf = -M * (dI/dt), જ્યાં M = પરસ્પર પ્રેરણાંક (Mutual Inductance)
  • Mનું એકમ હેનરી (H) છે.
  • ટ્રાન્સફોર્મર પરસ્પર પ્રેરણના સિદ્ધાંત પર કામ કરે છે.
🧠 Trick: પરસ્પર = એક બદલે તો બીજામાં પ્રેરણ

ટ્રાન્સફોર્મર

ટ્રાન્સફોર્મર એ ઉચ્ચ આઉટપુટ વોલ્ટેજ અથવા નીચા આઉટપુટ વોલ્ટેજ મેળવવા માટે AC વોલ્ટેજને બદલવા માટે વપરાતું ઉપકરણ છે. તે પરસ્પર પ્રેરણના સિદ્ધાંત પર કામ કરે છે.

  • ટ્રાન્સફોર્મરમાં પ્રાથમિક અને દ્વિતીય કોઈલ હોય છે.
  • V2/V1 = N2/N1 (જ્યાં V = વોલ્ટેજ, N = કોઈલના ફેરા)
  • આદર્શ ટ્રાન્સફોર્મરમાં 100% કુશળતા હોય છે, જ્યાં P1 = P2 (પાવર સંરક્ષિત રહે છે).
🧠 Trick: ટ્રાન્સફોર્મર = વોલ્ટેજ બદલો, ફેરા ગણો

ઊર્જા અને વિદ્યુતચુંબકીય પ્રેરણ

વિદ્યુતચુંબકીય પ્રેરણ દ્વારા ઉર્જાનું રૂપાંતર થાય છે. ઉદાહરણ તરીકે, જનરેટરમાં યાંત્રિક ઉર્જા વિદ્યુત ઉર્જામાં રૂપાંતરિત થાય છે.

  • જનરેટર ફેરેડેના નિયમ પર કામ કરે છે.
  • વિદ્યુત ઉર્જા = ½ L I² (સ્વ-પ્રેરણમાં સંગ્રહિત ઉર્જા)
  • ટ્રાન્સફોર્મરમાં ઉર્જા સંરક્ષિત રહે છે.

એડી પ્રવાહ (Eddy Currents)

જ્યારે ચુંબકીય ક્ષેત્રમાં ધાતુના ટુકડાને હલાવવામાં આવે છે, ત્યારે તેના અંદર પ્રેરિત emf ઉત્પન્ન થાય છે, જે એડી પ્રવાહ તરીકે ઓળખાય છે. આ પ્રવાહ ધાતુમાં ઉષ્મા ઉત્પન્ન કરે છે.

  • એડી પ્રવાહ લેન્ઝના નિયમનો પાલન કરે છે.
  • એડી પ્રવાહના કારણે ઉર્જાનું નુકશાન થાય છે, જે ઇલેક્ટ્રિક મોટર્સ અને ટ્રાન્સફોર્મરમાં અવગણના યોગ્ય છે.
  • એડી પ્રવાહને ઘટાડવા માટે લેમિનેટેડ કોર વપરાય છે.
🧠 Trick: એડી = ધાતુમાં ગરમી ઉત્પન્ન કરો

⚡ Quick Revision

  • ફેરેડેનો નિયમ: ફ્લક્સ બદલાય તો emf પ્રેરિત થાય.
  • લેન્ઝનો નિયમ: પ્રેરિત પ્રવાહ ફ્લક્સ બદલાવનો વિરોધ કરે.
  • સ્વ-પ્રેરણ: કોઈલમાં જ પ્રેરિત emf, L = સ્વ-પ્રેરણાંક.
  • પરસ્પર પ્રેરણ: એક કોઈલનો પ્રવાહ બદલાય તો બીજી કોઈલમાં emf.
  • ટ્રાન્સફોર્મર: V2/V1 = N2/N1, પાવર સંરક્ષિત.
  • જનરેટર: યાંત્રિક ઉર્જા → વિદ્યુત ઉર્જા (ફેરેડેનો નિયમ).
  • એડી પ્રવાહ: ધાતુમાં પ્રેરિત પ્રવાહ, ઉષ્મા ઉત્પન્ન કરે.
  • ઊર્જા સંગ્રહ: ½ L I² (સ્વ-પ્રેરણમાં).
🎯 Exam Tip: લેન્ઝના નિયમ અને ફેરેડેના નિયમને સંકળીને સમજો, કારણ કે NEETમાં આ બંને નિયમો એકસાથે પૂછવામાં આવે છે. ટ્રાન્સફોર્મરના પ્રશ્નો માટે V2/V1 = N2/N1 અને P1 = P2 યાદ રાખો.
