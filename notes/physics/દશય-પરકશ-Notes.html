દૃશ્ય પ્રકાશ

📖 દૃશ્ય પ્રકાશ

પરિભાષા અને મૂળભૂત સંકલ્પના

દૃશ્ય પ્રકાશ એ વર્ણપટના તે ભાગને કહેવાય છે જે માનવીની આંખ દ્વારા દેખાય છે. તેની તરંગલંબાઈ 400 nm (જાંબલી) થી 700 nm (લાલ) સુધીની હોય છે. પ્રકાશ એ વિદ્યુતચુંબકીય તરંગ છે જે ઉર્જાનું વહન કરે છે અને કણિકા (ફોટોન) સ્વરૂપે પણ વર્તે છે.

  • દૃશ્ય પ્રકાશની તરંગલંબાઈ: 400-700 nm
  • માનવ આંખ માટે સંવેદનશીલ રેન્જ
  • વિદ્યુતચુંબકીય સ્પેક્ટ્રમનો એક ભાગ
  • ફોટોન ઉર્જા (E = hν) દ્વારા વહન પામે છે
🧠 Trick: ROYGBIV (લાલ, ઓરેન્જ, પીળો, હરિયો, નીલો, આકાશી, જાંબલી)

પ્રકાશનો કણિકા સ્વભાવ (ફોટોન)

પ્રકાશ કણિકા (ફોટોન) સ્વરૂપે પણ વર્તે છે, જેની ઉર્જા E = hν (h = પ્લાંકનો અચળાંક, ν = આવૃત્તિ) દ્વારા ગણવામાં આવે છે. આ સંકલ્પ ફોટોવિદ્યુત પ્રભાવ અને પ્રકાશસંશ્લેષણમાં મહત્વપૂર્ણ છે.

  • ફોટોન ઉર્જા: E = hν
  • h = 6.626 × 10⁻³⁴ Js (પ્લાંકનો અચળાંક)
  • ફોટોવિદ્યુત પ્રભાવમાં ફોટોનની ભૂમિકા
  • પ્રકાશસંશ્લેષણમાં ફોટોન શોષણ

પ્રકાશનો તરંગ સ્વભાવ

પ્રકાશ તરંગ સ્વરૂપે પણ વર્તે છે, જે વિવર્તન, વક્રીભવન અને હસ્તક્ષેપ જેવી ઘટનાઓ દર્શાવે છે. તરંગલંબાઈ (λ) અને આવૃત્તિ (ν) વચ્ચેનો સંબંધ c = λν (c = પ્રકાશનો વેગ, 3 × 10⁸ m/s) છે.

  • તરંગલંબાઈ (λ) અને આવૃત્તિ (ν) વચ્ચે c = λν
  • પ્રકાશનો વેગ: 3 × 10⁸ m/s (નિર્વાતમાં)
  • વિવર્તન, વક્રીભવન, હસ્તક્ષેપ જેવી ઘટનાઓ
  • પ્રકાશ એ અનુદૈર્ઘ્ય વિદ્યુતચુંબકીય તરંગ છે

દૃશ્ય પ્રકાશ અને પ્રકાશસંશ્લેષણ

પ્રકાશસંશ્લેષણમાં, વનસ્પતિઓ દૃશ્ય પ્રકાશની ઉર્જાનો ઉપયોગ કાર્બન ડાયોક્સાઇડ અને પાણીને કાર્બોદિત (ગ્લુકોઝ)માં રૂપાંતરિત કરવા માટે કરે છે. ક્લોરોફિલ એ પ્રકાશ શોષણ માટે જવાબદાર વર્ણક છે, જે મુખ્યત્વે લાલ અને નીલા પ્રકાશને શોષે છે.

  • પ્રકાશસંશ્લેષણમાં દૃશ્ય પ્રકાશની ભૂમિકા
  • ક્લોરોફિલ લાલ અને નીલા પ્રકાશને શોષે છે
  • ગ્લુકોઝ સ્વરૂપે ઉર્જા સંગ્રહ
  • 6CO₂ + 6H₂O + પ્રકાશ → C₆H₁₂O₆ + 6O₂
🧠 Trick: લાલ-નીલો શોષે, હરિયો પરાવર્તે (ક્લોરોફિલ)

માનવ આંખ અને દૃશ્ય પ્રકાશ

માનવ આંખમાં, દૃશ્ય પ્રકાશ રેટિના પર પડે છે, જ્યાં રોડ અને કોન કોષો પ્રકાશને વિદ્યુત સંકેતોમાં રૂપાંતરિત કરે છે. રોડ કોષો નીચી પ્રકાશ તીવ્રતામાં કામ કરે છે, જ્યારે કોન કોષો રંગ દર્શન માટે જવાબદાર છે.

  • રેટિના પર પ્રકાશનું પતન
  • રોડ કોષો: નીચી પ્રકાશ તીવ્રતા
  • કોન કોષો: રંગ દર્શન
  • રોડોપ્સિન (રોડ કોષોનો વર્ણક)
🧠 Trick: રોડ = રાત, કોન = કલર

પ્રકાશનો વક્રીભવન અને લેન્સ

પ્રકાશનો વક્રીભવન એટલે એક માધ્યમથી બીજા માધ્યમમાં જતી વખતે તેના માર્ગમાં થતો વળાંક. લેન્સ અને કાચના પ્રિઝમમાં પ્રકાશનો વક્રીભવન થાય છે, જેના કારણે પ્રકાશના રંગો અલગ થાય છે (વર્ણવિભાજન).

  • વક્રીભવન: માધ્યમ બદલાતા પ્રકાશનો વળાંક
  • સ્નેલનો નિયમ: n₁sinθ₁ = n₂sinθ₂
  • લેન્સમાં ફોકલ લંબાઈ અને વક્રીભવન
  • વર્ણવિભાજન: પ્રિઝમ દ્વારા રંગોનું અલગીકરણ

દૃશ્ય પ્રકાશના ઉપયોગો

દૃશ્ય પ્રકાશનો ઉપયોગ પ્રકાશસંશ્લેષણ, દર્શન, ફોટોગ્રાફી, અને ઓપ્ટિકલ સાધનો જેવા કે માઇક્રોસ્કોપ અને ટેલિસ્કોપમાં થાય છે. તે સૌર ઉર્જાનો પણ મુખ્ય સ્રોત છે.

  • પ્રકાશસંશ્લેષણમાં ઉર્જા સ્રોત
  • માનવ દર્શન માટે આવશ્યક
  • ફોટોગ્રાફી અને ઓપ્ટિકલ સાધનો
  • સૌર ઉર્જાનો મુખ્ય સ્રોત
