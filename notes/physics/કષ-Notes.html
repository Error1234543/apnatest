Pro Notes

કોષ

📖 કોષ (Cell) - જીવનની મૂળભૂત એકમ

કોષની વ્યાખ્યા અને પ્રકાર

કોષ જીવનની સંરચનાત્મક અને ક્રિયાત્મક મૂળભૂત એકમ છે. તે બધા જીવિત પ્રાણીઓ અને વનસ્પતિઓનો બનેલો છે. કોષ બે પ્રકારના હોય છે: પ્રોકેરિયોટિક (બેક્ટેરિયા, સાયનોબેક્ટેરિયા) અને યુકેરિયોટિક (વનસ્પતિ, પ્રાણી, ફંગી, પ્રોટિસ્ટા).

  • કોષ = જીવનની મૂળભૂત એકમ
  • પ્રોકેરિયોટિક કોષમાં કોષકેન્દ્ર અને સંગઠિત અંગિકાઓ નથી
  • યુકેરિયોટિક કોષમાં કોષકેન્દ્ર અને સંગઠિત અંગિકાઓ હોય છે
  • પ્રોકેરિયોટિક કોષનું કદ 0.1–5 µm, યુકેરિયોટિક 5–100 µm
🧠 Trick: પ્રો = નાનો અને સરળ, યુ = મોટો અને જટિલ

કોષની સંરચના (યુકેરિયોટિક કોષ)

યુકેરિયોટિક કોષ ત્રણ મુખ્ય ભાગોમાં વહેંચાય છે: કોષપટલ (Plasma Membrane), કોષદ્રવ્ય (Cytoplasm), અને કોષકેન્દ્ર (Nucleus). કોષપટલ ફોસ્ફોલિપિડ બાયલેયરનું બનેલું હોય છે અને પદાર્થોના પરિવહનને નિયંત્રિત કરે છે.

  • કોષપટલ = ફોસ્ફોલિપિડ બાયલેયર + પ્રોટીન (Fluid Mosaic Model)
  • કોષદ્રવ્યમાં અંગિકાઓ (Organelles) અને સાયટોસોલ હોય છે
  • કોષકેન્દ્રમાં DNA, વર્ણસૂત્ર, અને કોષકેન્દ્રીય દ્રવ્ય હોય છે
  • કોષપટલ પાસીવ (Diffusion, Osmosis) અને એક્ટિવ પરિવહન કરે છે
🧠 Trick: પટલ-દ્રવ્ય-કેન્દ્ર = કોષના ત્રણ મુખ્ય ભાગ

કોષ અંગિકાઓ અને તેમના કાર્યો

કોષ અંગિકાઓ કોષના વિવિધ ક્રિયાઓને સંકલિત કરે છે. મિટોકોન્ડ્રિયા શ્વસન માટે ઉર્જા (ATP) ઉત્પન્ન કરે છે, ક્લોરોપ્લાસ્ટ પ્રકાશસંશ્લેષણ દ્વારા ગ્લુકોઝ બનાવે છે, અને એન્ડોપ્લાઝમિક રેટિક્યુલમ (ER) પ્રોટીન અને લિપિડના સંશ્લેષણમાં મદદ કરે છે.

  • મિટોકોન્ડ્રિયા = કોષનું ઉર્જા ઘર (ATP સંશ્લેષણ)
  • ક્લોરોપ્લાસ્ટ = પ્રકાશસંશ્લેષણ (વનસ્પતિ કોષમાં)
  • રાઇબોસોમ = પ્રોટીન સંશ્લેષણ (mRNA → પ્રોટીન)
  • લાઇસોસોમ = કોષીય પાચન (Hydrolase ઉત્સેચકો ધરાવે છે)
  • ગોલ્ગી ઉપકરણ = પ્રોટીનની પેકેજિંગ અને સેક્રેશન
🧠 Trick: મી-ક્લો-રા-લા-ગો = મિટો, ક્લોરો, રાઇબો, લાઇસો, ગોલ્ગી

કોષકેન્દ્ર અને અનુવંશિકતા

કોષકેન્દ્ર કોષનું નિયંત્રણ કેન્દ્ર છે અને તે DNAને ધરાવે છે. DNA વર્ણસૂત્રોના રૂપમાં ગોઠવાયેલું હોય છે, જે જનીન (Gene) ધરાવે છે. જનીન પ્રોટીનના સંશ્લેષણ માટે જવાબદાર હોય છે, જે કોષના કાર્યોને નિયંત્રિત કરે છે.

  • કોષકેન્દ્ર = DNA + વર્ણસૂત્ર + કોષકેન્દ્રીય દ્રવ્ય
  • વર્ણસૂત્ર = DNA + હિસ્ટોન પ્રોટીન (Nucleosome)
  • જનીન = DNAનો ભાગ જે પ્રોટીન માટે કોડ કરે છે
  • કોષકેન્દ્રીય પટલ દ્વિસ્તરી હોય છે અને કોષકેન્દ્રીય છિદ્રો (Nuclear Pores) ધરાવે છે
🧠 Trick: કેન્દ્ર = DNAનું ઘર, વર્ણસૂત્ર = DNAની માળા

કોષ ચક્ર અને કોષ વિભાજન

કોષ ચક્ર કોષના જીવનચક્રને દર્શાવે છે, જે ઇન્ટરફેઝ (G1, S, G2) અને મિટોટિક ફેઝ (M Phase)માં વહેંચાય છે. મિટોસિસ દ્વારા કોષની સંખ્યા વધે છે, જ્યારે મિયોસિસ જનન કોષોના નિર્માણ માટે જવાબદાર છે.

  • ઇન્ટરફેઝ = G1 (વૃદ્ધિ), S (DNA પ્રતિકૃતિ), G2 (તૈયારી)
  • મિટોસિસ = પ્રોફેઝ, મેટાફેઝ, એનાફેઝ, ટેલોફેઝ
  • મિયોસિસ = 2 વિભાજન, 4 હેપ્લોઇડ કોષો ઉત્પન્ન કરે છે
  • મિટોસિસ = સોમેટિક કોષો, મિયોસિસ = જનન કોષો
🧠 Trick: G1-S-G2-M = વૃદ્ધિ-પ્રતિકૃતિ-તૈયારી-વિભાજન

કોષ અને ઉર્જા પરિવર્તન

કોષમાં ઉર્જા પરિવર્તન મુખ્યત્વે મિટોકોન્ડ્રિયા અને ક્લોરોપ્લાસ્ટમાં થાય છે. શ્વસન (Respiration) દ્વારા ગ્લુકોઝનું ઓક્સિડેશન થઈ ATP બને છે, જ્યારે પ્રકાશસંશ્લેષણ (Photosynthesis) દ્વારા સૂર્યપ્રકાશની ઉર્જા રાસાયણિક ઉર્જામાં રૂપાંતરિત થાય છે.

  • શ્વસન = ગ્લુકોઝ + O2 → CO2 + H2O + ATP (38 ATP)
  • પ્રકાશસંશ્લેષણ = 6CO2 + 6H2O → C6H12O6 + 6O2 (ક્લોરોફિલની હાજરીમાં)
  • મિટોકોન્ડ્રિયા = કોષનું પાવરહાઉસ
  • ક્લોરોપ્લાસ્ટ = પ્રકાશસંશ્લેષણનું કેન્દ્ર
🧠 Trick: શ્વસન = ATP, પ્રકાશસંશ્લેષણ = ગ્લુકોઝ

કોષ અને પેશી સંરચના

કોષો એકત્રિત થઈ પેશી (Tissue) બનાવે છે, જે અંગો (Organ) અને અંગતંત્રોનો ભાગ બને છે. વનસ્પતિ કોષો કોષભિત્તિ (Cell Wall) ધરાવે છે, જે સેલ્યુલોઝનું બનેલું હોય છે અને કોષને આકાર આપે છે.

  • પેશી = કોષોનો સમૂહ જે સમાન કાર્ય કરે છે
  • વનસ્પતિ કોષ = કોષભિત્તિ (સેલ્યુલોઝ) + કોષપટલ + ક્લોરોપ્લાસ્ટ
  • પ્રાણી કોષ = કોષપટલ, કોષકેન્દ્ર, સેન્ટ્રિઓલ્સ (કોષભિત્તિ નથી)
  • કોષભિત્તિ = કોષને સુરક્ષા અને આકાર આપે છે
🧠 Trick: વનસ્પતિ = ભિત્તિ + ક્લોરો, પ્રાણી = પટલ + કેન્દ્ર

⚡ Quick Revision

  • કોષ = જીવનની મૂળભૂત એકમ
  • પ્રોકેરિયોટિક કોષ = કોષકેન્દ્ર વગરનો, યુકેરિયોટિક = કોષકેન્દ્ર સાથે
  • કોષપટલ = ફોસ્ફોલિપિડ બાયલેયર + પ્રોટીન (Fluid Mosaic Model)
  • મિટોકોન્ડ્રિયા = ATP ઉત્પાદન, ક્લોરોપ્લાસ્ટ = પ્રકાશસંશ્લેષણ
  • કોષકેન્દ્ર = DNA + વર્ણસૂત્ર + કોષકેન્દ્રીય દ્રવ્ય
  • મિટોસિસ = સોમેટિક કોષો, મિયોસિસ = જનન કોષો
  • શ્વસન = ગ્લુકોઝ → ATP, પ્રકાશસંશ્લેષણ = CO2 + H2O → ગ્લુકોઝ
  • વનસ્પતિ કોષ = કોષભિત્તિ + ક્લોરોપ્લાસ્ટ, પ્રાણી કોષ = સેન્ટ્રિઓલ્સ
🎯 Exam Tip: NEETમાં કોષની સંરચના અને કાર્યો પર આધારિત પ્રશ્નો વારંવાર પૂછાય છે. મિટોકોન્ડ્રિયા, ક્લોરોપ્લાસ્ટ, કોષકેન્દ્ર, અને કોષ ચક્ર પર ખાસ ધ્યાન આપો.
