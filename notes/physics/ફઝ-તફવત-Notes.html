ફેઝ તફાવત

📖 ફેઝ તફાવત (Phase Difference)

પરિભાષા અને મૂળભૂત સંકલ્પના

ફેઝ તફાવત એ ટ્રાન્સવર્સ તરંગો અથવા વૈદ્યુતચુંબકીય તરંગો વચ્ચેના કોણીય અંતરનું માપન છે. જ્યારે બે તરંગો એકસરખી આવૃત્તિ ધરાવે છે, ત્યારે તેમની શિખર અને ખીણ વચ્ચેનો સમય અથવા કોણીય અંતર એ ફેઝ તફાવત કહેવાય છે.

  • ફેઝ તફાવત એ કોણીય અંતર (φ) છે, જે રેડિયન અથવા ડિગ્રીમાં માપવામાં આવે છે.
  • જો બે તરંગો સંપૂર્ણપણે સુસંગત હોય, તો φ = 0° અથવા 2π રેડિયન.
  • જો તરંગો વિપરીત ફેઝમાં હોય, તો φ = 180° અથવા π રેડિયન.
  • ફેઝ તફાવત = (2π × પાથ તફાવત) / તરંગલંબાઈ (λ).
🧠 Trick: સુસંગત = સરખું, વિપરીત = 180°

ટ્રાન્સવર્સ તરંગોમાં ફેઝ તફાવત

ટ્રાન્સવર્સ તરંગોમાં, ફેઝ તફાવતને કોષ્ટક અથવા ગ્રાફ દ્વારા દર્શાવી શકાય છે. જ્યારે બે કણો એકસરખી આવૃત્તિથી દોલન કરે છે, પરંતુ વિવિધ ક્ષણે શરૂ થાય છે, ત્યારે તેમની વચ્ચે ફેઝ તફાવત હોય છે.

  • ટ્રાન્સવર્સ તરંગમાં, ફેઝ તફાવત = (2π × x) / λ, જ્યાં x એ કણો વચ્ચેનું અંતર છે.
  • જો x = λ/2, તો φ = π (180°).
  • જો x = λ, તો φ = 2π (360° અથવા 0°).
  • ફેઝ તફાવત 0° થી 360° વચ્ચે હોય શકે છે.

વૈદ્યુતચુંબકીય તરંગોમાં ફેઝ તફાવત

વૈદ્યુતચુંબકીય તરંગોમાં, વિદ્યુત ક્ષેત્ર (E) અને ચુંબકીય ક્ષેત્ર (B) વચ્ચે ફેઝ તફાવત શૂન્ય હોય છે. પરંતુ જ્યારે બે વિવિધ સ્રોતોમાંથી ઉત્પન્ન થતા તરંગોની તુલના કરવામાં આવે, ત્યારે ફેઝ તફાવત મહત્વપૂર્ણ બને છે.

  • વિદ્યુત ક્ષેત્ર અને ચુંબકીય ક્ષેત્ર વચ્ચે ફેઝ તફાવત 0° હોય છે.
  • પોલરાઇઝ્ડ પ્રકાશમાં, વિવિધ દિશાઓના દોલનો વચ્ચે ફેઝ તફાવત હોઈ શકે છે.
  • ફેઝ તફાવતનો ઉપયોગ હસ્તક્ષેપ અને વિક્ષેપ ઘટનાઓમાં થાય છે.
🧠 Trick: E અને B એ સદા સાથે, ફેઝમાં એકસરખા

હસ્તક્ષેપ અને ફેઝ તફાવત

હસ્તક્ષેપ એટલે બે અથવા વધુ તરંગોનું સુપરપોઝિશન. જ્યારે બે તરંગો સંપૂર્ણપણે સુસંગત હોય (φ = 0°), ત્યારે બંધારણીય હસ્તક્ષેપ થાય છે. જ્યારે φ = 180°, ત્યારે વિનાશક હસ્તક્ષેપ થાય છે.

  • બંધારણીય હસ્તક્ષેપ: φ = 0°, 2π, 4π... (સમીકરણ: I = (a1 + a2)²)
  • વિનાશક હસ્તક્ષેપ: φ = π, 3π... (સમીકરણ: I = (a1 - a2)²)
  • હસ્તક્ષેપના પટ્ટાઓનો ઉપયોગ પ્રકાશની તરંગલંબાઈ માપવા માટે થાય છે.
🧠 Trick: સમ = બંધારણ, વિપરીત = વિનાશ

ફેઝ તફાવતનું પ્રાયોગિક મહત્વ

ફેઝ તફાવતનો ઉપયોગ રેડિયો, ટેલિવિઝન, અને ટેલિકોમ્યુનિકેશનમાં થાય છે. તેનો ઉપયોગ એન્ટીના ડિઝાઇન, સિગ્નલ પ્રોસેસિંગ, અને ઓપ્ટિકલ ફાઇબર કમ્યુનિકેશનમાં પણ થાય છે.

  • રેડિયો તરંગોના પ્રસારણમાં ફેઝ તફાવતનો ઉપયોગ થાય છે.
  • એફએમ (FM) અને એએમ (AM) મોડ્યુલેશનમાં ફેઝ તફાવત મહત્વપૂર્ણ છે.
  • ઓપ્ટિકલ ફાઇબરમાં, ફેઝ તફાવતનો ઉપયોગ ડેટા ટ્રાન્સમિશનમાં થાય છે.

ફેઝ તફાવત અને તરંગ સમીકરણ

તરંગ સમીકરણ y = A sin(ωt + φ) માં, φ એ પ્રારંભિક ફેઝ તફાવત છે. જ્યારે બે તરંગોની તુલના કરવામાં આવે, ત્યારે તેમની વચ્ચેનો ફેઝ તફાવત Δφ = φ2 - φ1 હોય છે.

  • તરંગ સમીકરણ: y = A sin(ωt + kx + φ).
  • ω = 2πf (કોણીય આવૃત્તિ), k = 2π/λ (તરંગ સંખ્યા).
  • ફેઝ તફાવત Δφ = kΔx, જ્યાં Δx એ પાથ તફાવત છે.
🧠 Trick: y = A sin(ωt + kx + φ) → φ એ ફેઝ

⚡ Quick Revision

  • ફેઝ તફાવત એ કોણીય અંતર છે, જે રેડિયન અથવા ડિગ્રીમાં માપવામાં આવે છે.
  • સંપૂર્ણ સુસંગત તરંગો માટે φ = 0° અથવા 2π રેડિયન.
  • વિપરીત ફેઝમાં φ = 180° અથવા π રેડિયન.
  • ફેઝ તફાવત = (2π × પાથ તફાવત) / તરંગલંબાઈ (λ).
  • બંધારણીય હસ્તક્ષેપ: φ = 0°, વિનાશક હસ્તક્ષેપ: φ = 180°.
  • ટ્રાન્સવર્સ તરંગમાં, φ = (2π × x) / λ, જ્યાં x એ કણો વચ્ચેનું અંતર છે.
  • વિદ્યુત ક્ષેત્ર અને ચુંબકીય ક્ષેત્ર વચ્ચે φ = 0°.
  • તરંગ સમીકરણ: y = A sin(ωt + kx + φ).
  • હસ્તક્ષેપમાં, φ = 0° → બંધારણીય, φ = 180° → વિનાશક.
  • ફેઝ તફાવતનો ઉપયોગ રેડિયો, ટેલિવિઝન, અને ઓપ્ટિકલ કમ્યુનિકેશનમાં થાય છે.
🎯 Exam Tip: ફેઝ તફાવત અને હસ્તક્ષેપના પ્રશ્નોમાં, સદા પાથ તફાવત (Δx) અને તરંગલંબાઈ (λ) વચ્ચેનો સંબંધ યાદ રાખો: Δφ = (2π × Δx) / λ. આ સૂત્રનો ઉપયોગ કરીને સરળતાથી ફેઝ તફાવત શોધી શકાશે.
