સોલેનોઇડ

📖 સોલેનોઇડ

પરિભાષા અને બાંધકામ

સોલેનોઇડ એ વીજળીના વાહક તારનું કોઇલ જેવું ગોળાકાર અથવા લંબગોળ આકારમાં વીંટળાયેલું રચનાત્મક સ્વરૂપ છે. તેના દરેક ફેરામાં વીજપ્રવાહ પસાર થવાથી ચુંબકીય ક્ષેત્ર ઉત્પન્ન થાય છે. સોલેનોઇડની લંબાઈ અને ફેરાની સંખ્યાના આધારે ચુંબકીય ક્ષેત્રની તીવ્રતા નક્કી થાય છે.

  • સોલેનોઇડ = વીજળીના તારનું કોઇલ-જેવું રચના
  • વીજપ્રવાહ પસાર થવાથી ચુંબકીય ક્ષેત્ર ઉત્પન્ન થાય
  • ચુંબકીય ક્ષેત્રની દિશા જમણા હાથના નિયમથી નક્કી થાય
  • લંબાઈ, ફેરાની સંખ્યા અને પ્રવાહની તીવ્રતા પર ચુંબકીય ક્ષેત્ર આધારિત
🧠 Trick: સોલેનોઇડ = સોળ ફેરા + નિયમિત કોઇલ + ડીસી પ્રવાહ

ચુંબકીય ક્ષેત્રની ગણતરી

સોલેનોઇડના અંદરના બિંદુ પર ચુંબકીય ક્ષેત્ર B = μ₀ * n * I છે, જ્યાં μ₀ = મુક્ત અવકાશની પારગમ્યતા, n = એકમ લંબાઈ દીઠ ફેરાની સંખ્યા, I = વીજપ્રવાહ. સોલેનોઇડના અંતે B = (μ₀ * N * I) / (2L) થાય છે, જ્યાં N = કુલ ફેરા, L = સોલેનોઇડની લંબાઈ.

  • B = μ₀nI (અંદરના બિંદુ માટે)
  • B = (μ₀NI)/(2L) (અંતે)
  • μ₀ = 4π × 10⁻⁷ T m/A
  • n = N/L (એકમ લંબાઈ દીઠ ફેરા)
🧠 Trick: B = μ₀nI → બોલો મીટર નિયમિત ઇનપુટ

ચુંબકીય ક્ષેત્રની દિશા

સોલેનોઇડમાં વીજપ્રવાહના પ્રવાહની દિશા અનુસાર ચુંબકીય ક્ષેત્રની દિશા જમણા હાથના નિયમ (Right-Hand Thumb Rule) વડે નક્કી કરવામાં આવે છે. જ્યારે જમણા હાથના અંગૂઠાને પ્રવાહની દિશામાં રાખવામાં આવે, ત્યારે આંગળીઓ ચુંબકીય ક્ષેત્રની દિશા દર્શાવે છે.

  • જમણા હાથનો અંગૂઠો = પ્રવાહની દિશા
  • આંગળીઓ = ચુંબકીય ક્ષેત્રની દિશા
  • સોલેનોઇડના અંદર ક્ષેત્ર એકસમાન અને સમાંતર
  • બહાર ક્ષેત્ર નબળું અને વક્રીય
🧠 Trick: અંગૂઠો પ્રવાહ, આંગળી ક્ષેત્ર

સોલેનોઇડ અને ચુંબકીય ડાયપોલ

લંબાઈમાં લાંબો સોલેનોઇડ ચુંબકીય ડાયપોલ જેવું વર્તન દર્શાવે છે. તેના ઉત્તર અને દક્ષિણ ધ્રુવ હોય છે, અને તે ચુંબકીય ક્ષેત્રમાં ટોર્ક અનુભવે છે. સોલેનોઇડનું ચુંબકીય ડાયપોલ મોમેન્ટ m = N * I * A છે, જ્યાં A = ક્રોસ-સેક્શનલ એરિયા.

  • લાંબો સોલેનોઇડ = ચુંબકીય ડાયપોલ
  • m = NIA (ચુંબકીય મોમેન્ટ)
  • ટોર્ક τ = m × B (વેક્ટર ક્રોસ પ્રોડક્ટ)
  • ધ્રુવો ઉત્તર અને દક્ષિણ
🧠 Trick: m = NIA → મેં નહીં ઇચ્છા એરિયા

સોલેનોઇડના અનુપ્રયોગો

સોલેનોઇડનો ઉપયોગ ઇલેક્ટ્રોમેગ્નેટ, રિલે, વીજળીના ઘંટ, મોટર્સ અને જનરેટરમાં થાય છે. ઇલેક્ટ્રોમેગ્નેટમાં સોલેનોઇડમાં લોખંડનું કોર મૂકવાથી ચુંબકીય ક્ષેત્રની તીવ્રતા વધે છે.

  • ઇલેક્ટ્રોમેગ્નેટમાં લોખંડનું કોર વાપરવામાં આવે
  • રિલે અને ઘંટમાં ઉપયોગી
  • મોટર્સ અને જનરેટરમાં મહત્વપૂર્ણ ભૂમિકા
  • ચુંબકીય ક્ષેત્રની તીવ્રતા વધારવા લોખંડનું કોર

સોલેનોઇડ અને ફેરાડેનો નિયમ

જ્યારે સોલેનોઇડમાં વીજપ્રવાહ બદલાય છે, ત્યારે ફેરાડેના નિયમ અનુસાર પ્રેરિત વીજવાહક બળ ઉત્પન્ન થાય છે. આ પ્રેરિત EMF = -L * (dI/dt) છે, જ્યાં L = સ્વ-પ્રેરકત્વ છે.

  • પ્રેરિત EMF = -L(dI/dt)
  • L = μ₀N²A/l (સ્વ-પ્રેરકત્વ)
  • ફેરાડેનો નિયમ: પ્રેરિત EMF = -dΦ/dt
  • લેન્ઝનો નિયમ: પ્રેરિત પ્રવાહ મૂળ ક્ષેત્રના વિરોધમાં
🧠 Trick: EMF = -L di/dt → એમ એલ આઈ ડી ટી

🎯 Exam Tip: સોલેનોઇડના ચુંબકીય ક્ષેત્રની ગણતરી અને દિશા નક્કી કરવા માટે જમણા હાથના નિયમ અને સૂત્ર B = μ₀nI યાદ રાખો. પરીક્ષામાં આ સૂત્ર અને નિયમનો ઉપયોગ કરવા માટે પ્રશ્નો આવે છે.
