લેન્સ નિર્માતા સમીકરણ

📖 લેન્સ નિર્માતા સમીકરણ

લેન્સ નિર્માતા સમીકરણનો પરિચય

લેન્સ નિર્માતા સમીકરણ એ પાતળા લેન્સની કેન્દ્રલંબાઈ (f), વક્રતાત્રિજ્યા (R₁ અને R₂) અને લેન્સના દ્રવ્યના વક્રીભવનાંક (n) વચ્ચેનો સંબંધ દર્શાવે છે. આ સમીકરણનો ઉપયોગ લેન્સની કેન્દ્રલંબાઈ ગણવા માટે થાય છે, જે પ્રકાશશાસ્ત્રમાં મહત્વપૂર્ણ છે. NEETમાં આ સમીકરણ પર આધારિત પ્રશ્નો વારંવાર પૂછાય છે.

  • સૂત્ર: 1/f = (n-1)(1/R₁ - 1/R₂)
  • f = લેન્સની કેન્દ્રલંબાઈ
  • n = લેન્સના દ્રવ્યનો વક્રીભવનાંક
  • R₁, R₂ = લેન્સની બંને સપાટીઓની વક્રતાત્રિજ્યા (સંકેત પ્રણાલી મહત્વની)
🧠 Trick: ફ્રેન્ડ (f) ને નમસ્કાર (n-1) કરો, રાજા (R₁) અને રાણી (R₂) ની વચ્ચે

સંકેત પ્રણાલી (Sign Convention)

લેન્સ નિર્માતા સમીકરણમાં વક્રતાત્રિજ્યાનો સંકેત મહત્વપૂર્ણ છે. લેન્સની સપાટી ઉતલ હોય તો R ધન અને અંતર્ગોળ હોય તો R ઋણ ગણવામાં આવે છે. પ્રકાશના કિરણોની દિશા અનુસાર સંકેત નક્કી થાય છે.

  • ઉતલ સપાટી માટે R ધન (+)
  • અંતર્ગોળ સપાટી માટે R ઋણ (-)
  • પાતળા લેન્સ માટે જ લાગુ પડે
  • NCERT મુજબ કાર્ટેશિયન સંકેત પ્રણાલી અનુસરો
🧠 Trick: ઉતલ = ઉપર (+), અંતર્ગોળ = અંદર (-)

પાતળા લેન્સ માટેની ધારણાઓ

લેન્સ નિર્માતા સમીકરણ પાતળા લેન્સ માટે જ માન્ય છે, જ્યાં લેન્સની જાડાઈ તેની વક્રતાત્રિજ્યા કરતાં નગણ્ય હોય. જાડા લેન્સ માટે આ સમીકરણમાં સુધારાની જરૂર પડે છે. NEETમાં પાતળા લેન્સ પર જ પ્રશ્નો પૂછાય છે.

  • લેન્સની જાડાઈ << R₁, R₂
  • પ્રકાશનું કિરણ લેન્સના પ્રિન્સિપલ અક્ષને સમાંતર ગણવામાં આવે
  • અપવર્તન માત્ર બે સપાટીઓ પર થાય છે
  • મુખ્ય કેન્દ્રો વચ્ચેનું અંતર નગણ્ય
🧠 Trick: પાતળો લેન્સ = પાતળી સમસ્યા (સરળ સમીકરણ)

સમતલ સપાટીનો કિસ્સો

જો લેન્સની એક સપાટી સમતલ હોય (R = ∞), તો તે સપાટી માટે 1/R = 0 થાય. આ કિસ્સામાં લેન્સ નિર્માતા સમીકરણ સરળ બને છે. પ્લાનો-કોન્વેક્સ અથવા પ્લાનો-કોન્કેવ લેન્સ માટે આ સૂત્રનો ઉપયોગ થાય છે.

  • R = ∞ ⇒ 1/R = 0
  • સમતલ-ઉતલ લેન્સ: 1/f = (n-1)(1/R)
  • સમતલ-અંતર્ગોળ લેન્સ: 1/f = -(n-1)(1/R)
  • NEETમાં વારંવાર પૂછાતો કિસ્સો
🧠 Trick: સમતલ = શૂન્ય (1/R = 0)

લેન્સની શક્તિ સાથેનો સંબંધ

લેન્સની શક્તિ (P) એ તેની કેન્દ્રલંબાઈના વ્યસ્ત પ્રમાણમાં હોય છે. લેન્સ નિર્માતા સમીકરણનો ઉપયોગ કરીને શક્તિ ગણી શકાય છે. ડાયોપ્ટર (D) એ લેન્સની શક્તિનું એકમ છે, જે m⁻¹ માં માપવામાં આવે છે.

  • P = 1/f (f મીટરમાં)
  • P = (n-1)(1/R₁ - 1/R₂)
  • 1 D = 1 m⁻¹
  • ઉતલ લેન્સ માટે P ધન, અંતર્ગોળ માટે P ઋણ
🧠 Trick: શક્તિ (P) = પ્રકાશને પકડવાની તાકાત (1/f)

વ્યવહારિક ઉદાહરણો અને પ્રશ્નો

NEETમાં લેન્સ નિર્માતા સમીકરણ પર આધારિત ગણતરીઓ વારંવાર પૂછાય છે. ઉદાહરણ તરીકે, જો કાચના લેન્સની વક્રતાત્રિજ્યા અને વક્રીભવનાંક આપેલ હોય, તો તેની કેન્દ્રલંબાઈ ગણવી. આ પ્રકારના પ્રશ્નોમાં સંકેત પ્રણાલીનું ધ્યાન રાખવું ખૂબ જરૂરી છે.

  • ઉદાહરણ: n=1.5, R₁=20 cm, R₂=-20 cm ⇒ f=20 cm
  • સંકેત ભૂલ ન થાય તેનું ધ્યાન રાખો
  • એકમો સુસંગત રાખો (સેમી અથવા મીટર)
  • NCERTના ઉદાહરણો અને પ્રશ્નોનો અભ્યાસ કરો
🧠 Trick: R₁ અને R₂ ના સંકેત યાદ રાખો: ઉતલ (+), અંતર્ગોળ (-)

⚡ Quick Revision

  • લેન્સ નિર્માતા સમીકરણ: 1/f = (n-1)(1/R₁ - 1/R₂)
  • ઉતલ સપાટી માટે R ધન, અંતર્ગોળ માટે R ઋણ
  • સમતલ સપાટી માટે 1/R = 0
  • પાતળા લેન્સ માટે જ માન્ય
  • લેન્સની શક્તિ P = 1/f (Dમાં)
  • n = લેન્સના દ્રવ્યનો વક્રીભવનાંક
  • કાર્ટેશિયન સંકેત પ્રણાલી અનુસરો

  • NEETમાં ગણતરી આધારિત પ્રશ્નો વધુ પૂછાય છે
  • સંકેત ભૂલ ન થાય તેનું ખાસ ધ્યાન રાખો
🎯 Exam Tip: લેન્સ નિર્માતા સમીકરણના પ્રશ્નોમાં સૌથી પહેલા સંકેત પ્રણાલી ચકાસી લો. R₁ અને R₂ ના મૂલ્યો ધન કે ઋણ છે તે જોઈને જ ગણતરી શરૂ કરો. NCERTના ઉદાહરણો અને પ્રશ્નો વારંવાર ઉકેલો, જેથી સંકેત ભૂલ ન થાય.
